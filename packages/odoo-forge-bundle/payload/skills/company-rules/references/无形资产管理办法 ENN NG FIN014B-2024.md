# 新奥天然气股份有限公司无形资产管理办法

# 第一章 总 则

第一条 为规范无形资产的管理行为，避免因违反国家法律法规而遭受的财产损失风险，防范无形资产管理中的差错和舞弊，明确职责权限，降低经营决策、资产管理风险，特制定本办法。

第二条 本办法所称无形资产是指企业拥有或者控制的没有实物形态的可辨认非货币性资产,主要包括土地使用权、专利权、非专利技术权、采矿权、用水权、软件、商标权和著作权等。

第三条 本办法适用于新奥天然气股份有限公司（以下简称新奥股份）本部及所属业务组织。

# 第二章 分 则

第四条 无形资产管理主要涉及财运群、技术群、政联群等，具体如下：

（一）财运群是新奥股份无形资产的综合管理群，其主要职责包括：

1.负责编制无形资产管理制度，并组织宣贯落实；  
2.负责建立全面的无形资产管理台账，对无形资产进行会计核算；  
3.负责定期对无形资产进行全面盘点，并参与无形资产的检查、验收和处置工作。

（二）数智安全委员会是软件类无形资产的专业管理群，主要负责该类无形资产的申购、验收、日常维护、使用费缴纳、清查盘点，以及台账的建立与维护等工作。  
（三）技术群是专利权和非专利技术权类无形资产的专业管理群，主要负责该类无形资产的申购、验收、年费缴纳、清查盘点，以及台账的建立与维护等工作。  
（四）政联群是土地使用权、商标权、著作权类无形资产的专业管理群，主要负责该类无形资产的申购、验收、年费缴纳、证照管理、清

查盘点、以及台账的建立与维护等工作。

第五条 无形资产管理过程中需遵循不相容岗位分离原则，具体如下：

（一）无形资产投资预算的编制与审批分离。  
（二）无形资产的采购与验收，验收与款项支付分离。  
（三）无形资产处置的申请与审批、审批与执行分离。  
（四）无形资产业务的审批、执行与相关会计记录分离。

# 第六条 无形资产的取得与验收

# （一）无形资产投资预算管理

1.新奥股份无形资产投资预算的编制、调整、审批、执行等按照《新奥天然气股份有限公司预算管理办法》执行。  
2.新奥股份根据发展战略和生产经营实际需要，并综合考虑无形资产投资方向、规模、资金占用成本、预计盈利水平和风险程度等因素编 制预算。  
3.新奥股份对无形资产拟投资项目进行可行性研究和分析论证的基础上合理安排投资进度和资金投放。

# （二）无形资产取得

# 1.外购取得无形资产

（1）采购申请。采购无形资产由相应专业管理群根据年度预算提出采购申请，采购前需对拟采项目或资产进行可行性论证，并对无形资产的性能、技术参数进行明确要求，然后填制《无形资产购置申请表》（详见附件1）。  
（2）采购实施。由无形资产专业管理群依据购置申请联系供应商，组织采购。

# 2.自制取得无形资产

自制无形资产按新奥股份年度综合计划及年度预算批准的立项执行，需临时立项的自制无形资产，根据项目预算按外购无形资产审批权限执行。

# （三）无形资产验收

1.外购取得无形资产的验收

外购无形资产由专业管理群组织，按照合同、技术交底文件规定的验收标准进行验收；在办理无形资产验收手续的同时，专业管理群应完整地取得产品说明书及其他相关说明资料，并持发票和相关资料到财运群办理无形资产入账手续。

2.自制无形资产的验收

自制无形资产制作完成后，由业务群项目负责人向专业管理群提出验收申请专业管理群负责组织验收；财运群依据业务群提供的项目验收报告及相关验收单据进行相应的账务处理。

# 第七条 无形资产的日常管理

# （一）台账与盘点

1.专业管理群负责根据无形资产的使用状况，及时维护本专业群无形资产台帐。台帐登记的主要内容包括：资产名称、规格型号、业务组等。  
2.专业管理群、财运群定期（至少每年一次）核对相关账簿、记录和文件，对于发现问题及时向主管领导和上级业务组织报告和处理，以确保无形资产账务处理和资产价值的真实性。

# （二）财务摊销

财运群根据企业会计准则规定确定无形资产的摊销年限，所有权归属公司的无形资产均需纳入摊销范围。

# 第八条 无形资产的处置

# （一）处置申请与鉴定

1.处置申请。无形资产不能继续使用时，视资产情况由使用群或专业管理群填写《无形资产处置申请表》（详见附件2）。  
2.处置鉴定。由提出申请的业务群组织有关群的技术专业人员对处置的无形资产进行技术鉴定，确保无形资产处置的合理性。

# （二）处置审批

根据《新奥天然气股份有限公司经营授权表》规定对业务群上报的

《无形资产处置申请表》（详见附件2）进行审查，并签署意见。

# （三）处置审计与账务处理

1.示险群负责对无形资产的处置情况进行抽查审计，重点对无形资产的处置依据、处置方式、处置价格等进行关注，检查处置依据是否充分，方式是否适当，价格是否合理。  
2.财运群在无形资产处置后根据审批后的处置手续，认真审核无形资产处置凭证，检查批准手续是否齐全，批准权限是否适当等，审核无误后据以编制记账凭证，进行账务处理。

# 第九条 无形资产的特许使用

新奥股份特许其他公司使用本组织的无形资产，由专业管理群会同财运群拟定方案，经授权人员批准后办理相关手续，签订合同。合同应当对无形资产特许使用期间的权利义务进行明确。

# 第三章 附 则

第十条 本办法由财运群负责解释、修订。

第十一条 本办法自2021年11月29日起开始试行，试行期为3个月。3个月内若无修订，则自动转入实施。
