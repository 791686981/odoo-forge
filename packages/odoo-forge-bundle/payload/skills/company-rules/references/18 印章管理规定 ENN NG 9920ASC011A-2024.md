# 新地能源工程技术有限公司印章管理规定

# 第一章 总 则

# 第一条 目的

为确保印章管理和使用的规范性、安全性、有效性及严肃性，更好地防范企业行政风险，特制订本规定。

# 第二条 印章的分类

（一）实体印章分为法人印章、专用印章、法定代表人印章。

1.法人印章：具有独立法人资格的公司公章。  
2.专用印章：

（1）冠名专用印章：具有独立法人资格的公司为履行某种特定的职责和权力，经国家机关备案，刻制的具有法律效力的冠以法定名称的专用印章，包括：合同专用章、财务专用章、发票专用章、技术工程类专用章等。  
（2）文书备案章：是企业对重要文书进行管理的内部专用印章。由集团统一刻制、编号、发放和收回，冠 ENN 标志，文书备案负责人保管和利用。  
3.法定代表人印章：公司法定代表人用来行使职权的个人印章。

# （二）电子印章

将印章通过加密技术，以数字认证存储介质方式，在电子文件中应用的电子版的印章。

# 第三条 管理原则

1.独立保管、牵制使用。各级印章管理员独立保管本单位的印章，不受他人影响。除文书备案章、标记专用印章外，原则上其他印章的使用申请、决策、保管三个角色需要分离。

2.流程严密、手续严格。各级单位在印章的刻制、保管、加盖、外借、停用和销毁等环节必须严格执行规定的程序，以保障印章管理的严肃、高效和安全。

# 第四条 适用范围

本规定适用于新地能源工程技术有限公司（以下简称“公司”）及所属各单位。

# 第二章 组织与职责

第五条 各单位一号位为本单位印章规范管理的第一责任人，并原则上作为本单位印章使用最终决策人（党工青团类印章除外），如有其他人员为最终决策人，则需由一号位签字确认进行相关授权。

# 第六条 印章管理组织职责

各单位政联组织，对本单位所有印章保管组织和保管人员负有规范指导的职责：

（一）负责本单位印章管理制度的编制与修订、解读宣贯、推动落实与执行监督；  
（二）负责对本单位所有印章进行全生命周期规范管理，牵头组织、指导、监督印章刻制、保管、移交、停用等各环节的规范操作，并对下级单位指导、监督、检查、纠偏；  
（三）负责选拔机要秘书。

# 第七条 印章保管组织职责

各级印章保管组织根据自身职责，对本组织保管印章的规范管理负有主要责任：

（一）对本组织保管的印章进行全生命周期规范管理；  
（二）细化本组织保管印章使用规范；  
（三）接受本级和上级印章管理组织的监督、检查。

# 第八条 印章管理人员职责

# （一）机要秘书

1.机要秘书承接本单位机要管理职责，指导、培训本单位印章管理员实施印章规范化管理；同时负责本单位的印章刻制的专业把关;负责本单位法人印章、合同专用章及其他需要统管、代管印章的保管、加盖和外借。  
2.实体公司必须设置一名机要秘书，一名机要秘书可统管、代管多家实体公司印章。不具备条件的单位由行政负责人兼任机要秘书。  
3.机要秘书聘任时，需填写《机要秘书审批表》（附件1）在新奥股份完成审批，并在集团备案。  
4.机要秘书任职资格

（1）专业要求：原则上需法律、经济、管理类本科及以上学历；  
（2）从业要求：2 年以上新奥司龄，且无违纪情况；  
（3）考核要求：近 2 年考核不得出现 C；  
（4）品质要求：正直、诚信、品行端正。

# （二）印章管理员

公司其他印章保管人员均为印章管理员，负责所保管印章的保管、加盖、台帐维护等。所有印章管理员均需填写《印章管理人员备案表》（附件2）备案。

# 第三章 印章制发

第九条 所有印章刻制必须按照要求履行审批、报备手续，且所有印章只能刻制一枚，不得重复刻制。

# 第十条 新设立公司印章制发

（一）新设境内公司印章制发由设立发起单位在领取企业营业执照后，严格按照当地工商行政部门的要求进行刻制，一般包含法人章、合同专用章、法定代表人章和财务专用章等，刻制完毕后交本单位印章管理组织。

（二）新设境外企业刻制印章，须由业务需求单位提出刻制申请，经公司及新奥股份财务、法务、行政、一号位会签，报集团审社会联系智能能力群。

# 第十一条 更名印章制发

（一）因公司名称变更重新刻制印章时，由企业变更发起单位在办理企业名称变更登记相关手续后，按照当地工商行政管理部门的要求，将变更前的印章移交工商行政管理部门，并刻制更名后的企业印章。刻制完毕后交本单位印章管理组织保管。  
（二）因法定代表人更换重新刻制印章时，由财运群发起法定代表人章刻制申请，在当地公安部门指定地点进行印章刻制并进行系统备案；变更前的印章移交本单位印章管理组织，新刻制印章印模信息在本单位印章管理组织备案。

# 第十二条 更新印章制发

因磨损、损坏或丢失，印章需要更新时，须由印章保管组织提交印章刻制流程，并在当地公安部门指定地点进行印章刻制并进行系统备案；刻制完毕后及时通知财运等理正组织及时完成变更备案手续。磨损的印章移交本单位印章管理组织，新刻制印章印模信息在本单位印章管理组织备案。

# 第十三条 电子印章制发

电子印章刻制应遵循“实际业务一事一议”的原则，经印章管理组织召集专业评审通过后，由使用组织提出刻制申请，仅限在议定的业务场景、操作流程中使用。

第十四条 各单位新刻印章后，须在《印章信息统计表》（附件3）上拓具印模，标明印章刻制时间、保管人及接收时间，并于三日内上报公司政联群备案。

# 第五章 印章的保管

第十五条 印章的保管

（一）法人章、合同章须由机要秘书保管，没有设立机要秘书的企业由上级单位机要秘书保管。  
（二）其他专用章由专业组织指定印章管理员专人保管。财务专用章和法定代表人印章必须分人保管。  
（三）印章保管人员发生变化时，需走线上印章移交流程，经审批后方可进行移交，并填写印章移交表（附件4）。  
（四）各组织须选择安全条件良好的处所及装置存放印章，由印章保管人员妥善保管钥匙，不得随意放置。政联群、财运群须使用双锁保险柜。  
（五）公章的操作区域安装监控，用于实时录制往来人员现场用章的图像，要求安装后录像范围覆盖保险柜存放处、印章加盖操作台。

第十六条 一旦发生印章丢失、损坏等异常情况，须立即启动风险处理预案，并由印章管理组织与企业一号位协同各相关赋能群实施。

（一）QHSE群、政联群须查明具体情况和具体原因，如涉及违规违纪，人发群拟定处罚意见；  
（二）如印章丢失，财运群须及时通知银行等相关机构，暂停丢失印章的使用；法务须及时通知工商行政管理部门，注销丢失印章并公告声明作废；政联群联合印章保管组织，进行印章更新刻制。  
（三）如印章损坏，政联群联合印章保管组织，进行印章更新刻制，并通知财运等理正组织及时完成变更备案手续。

# 第五章 印章的申请和使用

第十七条 审批授权的梳理

各单位印章管理组织应结合本单位业务，梳理《印章审批授权表》附件5），经一号位签署后发布生效，并及时更新。

第十八条 一般业务的用章申请

（一）申请人填写并提交印章申请工作流，完成审批后方可用印。同时，申请人须在《用章登记表》（附件 6）上进行登记。  
（二）因文件变更导致需要重新加盖印章的，均需重新履行印章使用申请。

# 第十九条 专业流程关联的用章

# （一）合同用章

1.合同评审完成后，定稿文本须加盖文书备案专用章，经法定代表人或授权委托人签署后，提交原件，按照合同约定的份数进行加盖。  
2.下列不规范合同不得用印：

（1）没有合同编号或合同编号不符合编制规则的合同;  
（2）未经审核的合同;  
（3）未经法定代表人或其授权的委托代理人签署的合同。

# （二）证照用章

证照申请涉及到复印件盖章的，机要秘书依据已经审批通过的证照申请，打印加有水印的证照复印件并加盖印章，申请人签字领用。严禁个人持复印件直接申请用章。

# （三）公文用章

在发文流程审批完结后，由公文管理员持发文审批单和定稿文本，办理用章。

# （四）其他专业流程关联的用章

其他专业流程关联专用章使用的，由专业章管理组织制定管理细则。

第二十条 印章加盖须由印章管理员严格按照印章使用的操作要求规范办理，不得对禁止文件、禁止事项违规操作用章。

（一）必须由印章管理员亲手加盖。  
（二）空白纸张及文件空白处不得用章;无正文的盖章页须在此页适当处注明文件名称。

（三）对外具有法律效力的多页文件如（合同、协议、证明）等，须加盖骑缝章。  
（四）需要加盖印章的股东会决议、公司章程、授权委托书、业务函件、诉讼文书等法律文件，须先由法定代表人或其他相关人员签字后，方可用章。  
（五）外文文件须由印章所属单位指定人员进行审核后，方可用章。  
（六）加盖印章须端正，盖在单位名称和日期上，骑年压月。印章颜色须按规定使用红色或蓝色。

# 第二十一条 财务专用章、法定代表人印章的加盖

（一）财务日常交易业务必须基于发生的真实、合法、手续完备的财务会计业务。加盖财务印章时，须加盖于规定位置。  
（二）日常交易以外的财务业务，需要加盖财务印章和法定代印，同时进行印章使用登记。  
（三）非财务事项如需加盖法定代表人印章，须经本单位印章管理组织负责人、本单位运营一号位批准后，方可用印。禁止非财务事项加盖财务专用章。  
（四）严禁在合同、协议、承诺书、确认函等法律文件上使用法定代表人印章。

第二十二条 印章外借时，印章管理员和业务申请人须严格按照印章外借的操作要求办理印章的借取、保管、使用和归还手续。财务专用章不得外借。

（一）外借印章时，借章单位须安排两名业务相关人员共同办理印章的借取、保管、使用与归还手续。借出时，应签署《借章保证书》 。借出后，两名经办人要随身携带和妥善保管所借印章。使用印章时，两名经办人要共同拆启与加盖印章，不得委托他人。用印完毕后，须将印章放在保险盒中。

（二）所办事项完成后，借章单位须在规定日期内归还，如需延长借用时间，须经借用审批决策人同意后，方可延长，印章管理员不得自行应允延长借用时间。  
（三）归还印章时，经办人须向印章管理组织提交其签名的《外借用印文件登记表》备案（附件8），并在借章流程中上传用印文件电子版。  
（四）印章管理员须注意检查是否为原借出印章。

# 第二十三条 印章使用风险备案

在印章使用流程审批完毕，印章管理员仍认为存在风险时，发起风险备案，由业务负责人复核重审。

# 第六章 印章的停用、报废、销毁

第二十四条 如遇到公司注销、印章损坏等情况，印章管理组织须发布印章停用的通知，并履行印章报废审批后，按工商行政部门要求上缴或向上级印章管理组织移交。

第二十五条 任何企业不得私自销毁印章，工商行政部门不予回收的报废印章统一上交上级印章管理组织。

# 第七章 罚则

第二十六条 在印章管理及使用过程中存在以下违规行为的，对涉事人员依照相关制度按责论处，如情节严重构成犯罪的，依法移交司法机关处理：

（一）未履行机要秘书选聘程序，或印章未按规则要求分类合规保管，视情节轻重，给予相关责任人警告或通报批评处分，扣除1-3个月的月度业绩分享；  
（二）未经批准，私自刻制和使用印章，或利用职务之便用章，或强令指使机要秘书用章，将视情节轻重，给予相关责任人记过、记大过、降职降级、撤职或解除劳动关系处分，扣除4-6个月的月度业绩分享，涉及触犯法律法规的，依法追究其法律责任；

（三）未按规定程序刻制或销毁印章、违规用章，视情节轻6月的月度业绩分享；  
（四）印章外借过程中，借章人即为印章管理员，如未按制度用印的，借章人应承担相应的责任，并根据违规用印情节的严重程度，给与前三条中相同处罚。  
（五）以上奖惩由印章归口管理组织出具调查报告及奖惩建议，报公司一号位审定后执行。

# 第八章 附则

第二十七条 本规定由新地能源工程技术有限公司行政与社会联系智能能力群负责解释、修订。

第二十八条 本规定自 2024 年 7 月 1 日起试运行，试行期为 6个月。6 个月内若无修订，则自动转入实施。
