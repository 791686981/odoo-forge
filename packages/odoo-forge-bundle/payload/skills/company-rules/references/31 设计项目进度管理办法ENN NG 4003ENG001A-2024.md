# 新地能源工程技术有限公司设计分公司

# 设计项目进度管理办法

# 第一章 总则

# 第一条 目的

在设计管理中，项目进度是一个至关重要的方面。合理的项目进度管理可以确保项目按时完成，提高工作效率，保证项目质量。为提高项目设计执行效率，满足客户需求、提高能源工程项目整体效益，特制定本办法。

# 第二条 管理原则

（一）全员参与:项目进度管理是全公司范围内的工作，需要全体员工共同参与和努力。每个员工都应根据自身职责和分工，积极参与进度管理，共同维护项目进度的顺利进行。  
（二）科学合理:项目进度计划应科学、合理，能够充分考虑项目的技术难度、资源投入和工期限制等因素，并能够适应项目变化和不确定性。  
（三）实时监控:建立有效的项目进度管控机制，实时了解项目进展情况，及时发现并解决可能影响进度的问题，确保项目按时完成。  
（四）信息共享:项目进度信息应及时共享给相关项目组成员和相应协同组织，实现信息的透明化和高效传递，提高沟通效率。  
（五）持续改进:根据实际工作中的经验和反馈，不断完善和优化项目进度管理制度，提高管理水平和执行效果。

# 第三条 适用范围

本办法适用于适用于新地能源工程技术有限公司设计分公司（以下简称“燃气设计中心”）承接的综合项目设计进度管理。

# 第二章 分则

# 第四条 项目立项

燃气设计中心所承接的综合设计项目在接收到设计委托书或设计合同后，由业务群项目管理人员根据工程类型、工程规模在网络设计平台发布设计任务

# 第五条 人员安排

（一） 设计经理由燃气设计中心业务理正根据工程类型、工程规模进行任命，设计经理须具备所担任相应工程类型、规模的设计经理任职资格，合同中要求具备注册师执业资格的，要按注册师的执业规定执行。  
(二)各专业设、校、审人员由专业群召集人根据工程类型、规模进行安排，设计人员需具备相应的设计资格。

# 第六条 项目工期

设计项目工期根据客户建设项目进度需求与客户沟通后确定；EPC项目根据项目总进度计划确定设计项目工期。

# 第七条 项目进度计划制定

（一）项目启动阶段，由设计经理牵头召开项目进度计划会议，根据与客户或EPC项目经理部确定的设计工期，在网络设计平台制定项目进度计划，并经业务群项目管理人员审批，EPC项目还需经项目经理部审批。项目进度计划应包括项目工期、关键节点、关键事项等内容。  
（二）项目进度计划应根据实际情况进行合理调整和更新，确保计划的有效性和可行性。项目进度计划调整时需要出具与客户的沟通单。

# 第八条 关键节点

开工报告编制、审批时间；

方案计划完成时间、设计资料收集时间、项目计划开工时间；

项目设计实施过程中各专业计划完成时间；

项目会审计划时间；

项目会签计划时间；

输出评审计划时间;

项目评定计划时间；

文件发放计划时间；

注：大型项目设计增加项目策划文件编制工作，可按单体或增加部分关键节点实行计划管控。

# 第九条 关键事项

# （一）方案设计

在接到任务委托书，设计经理应在第一时间组织项目方案设计工作，如果有委托内容不明或有争议的问题，设计经理或专业负责人要编制方案沟通记录单发客户沟通确认，最终方案应经客户签字确认。

# （二）资料收集

1．资料收集时设计经理要以函件的形式向客户提供所需资料清单，所需资料清单的函件上要注明需客户完成所有资料的收集具体时间，并注明资料齐全后的设计周期计划，所有沟通内容应经客户确认。  
2．EPC项目所需资料清单及要求提供资料时间以函件的形式发至项目经理部，由项目经理部发相关方。

# （三）设计开工

1.基础资料或主要节点资料齐全后，设计经理根据项目要求编制项目开工报告，并经过审批，组织召开项目开工会。  
2.如有些资料在项目开工后可以随着项目进行情况由客户陆续提供时，设计经理必须组织项目开工，但需在给客户的沟通记录单中列明“所缺资料要在什么时间内提供，设计周期为多少天，如果资料不能提供设计周期顺延”等字样，需经客户确认，并在设计进度计划表中备注此情况。

# （四）设计过程

1．在项目设计过程中，未遇到条件变更或其它特殊情况时，应按计划执行。  
2．如果遇到客户提出新要求、条件变化或其他特殊情况造成项目无法按原计划进行设计时，设计经理必须以沟通单的形式与客户重

新确认完成时间（EPC 项目还需与项目经理部沟通确认），调整进度计划，并将调整后的进度计划上报业务群。

# （五）设计文件出版

1．各版次设计文件出版时必须按照燃气设计中心工作手册和技术档案管理办法进行归档后方可晒图。  
2．设计文件加晒、发放电子版图纸时，需线下签批《设计文件出版申请及发放记录》、《设计文件外发、借阅审批单》。

注：特殊紧急情况需要在区域设计部出版的设计文件，必须先进行归档，并在业务群备案后方可出版。

# 第十条 项目设计进度控制

（一） 设计经理负责每日监控项目进度，及时发现偏差和问题，并采取相应的措施加以解决。  
（二）重点项目实行周报,自项目开工后设计经理定期向上级及方案等信息。  
（三）业务群项目管理人员根据项目设计进度计划对项目执行情况进行监督和检查，如进度出现较大偏差及时示警。  
（四）燃气设计中心每周召开项目生产经营例会，重点解决项目设计过程中存在的问题。

# 第三章罚则

第十一条 本办法检查结果及最终项目出版情况将在项目创值分享中体现；对于缺少资料收集沟通记录单、开工报告未按程序审批、审批滞后、未按设计进度计划完成且没有沟通单的项目，将调低设计

# 第四章 附则

第十二条 本办法由燃气设计中心燃气业务群负责解释、修订，如有未尽事宜按公司相关制度执行；

第十三条 本办法自发布之日起实施。
