# 新地能源工程技术有限公司石家庄能源化工技术分公司图书管理办法

# 第一章 总则

# 第一条 目的

为了妥善保护燃气化工设计中心的图书资产免受损毁、流失，保证图书资源的正常流通，满足员工的工作需要，提高图书利用率，特制定本办法。

# 第二条 适用范围

本管理办法适用于燃气化工设计中心所采购的内部使用标准、规范、图集、手册等图书的管理。

# 第二章 分则

# 第三条 组织与职责

（一）各组织负责图书需求的收集、整理、申请；  
（二）技术质量智能能力群负责图书申请的确认、订购、登记、入库、发放、借阅，以及报销手续；  
（三）技术质量智能能力群召集人负责图书购买申请的审批，当涉及国外标准采购时候，应报请专业理正批准。

# 第四条 购买

（一）各组织需订购图书时，由申请人填写《标准图书订购申请单》，提出订购图书的名称、标准号、数量、价格和用途，由所在组织负责人签字同意后，交技术质量智能能力群；  
（二）技术质量智能能力群图书管理人员核对与现有图书不重复，且为最新版本后确认，提交专业理正/技术质量智能能力群召集人审批。  
（三）技术质量智能能力群根据已完成审批的《标准图书订购申请单》信息采购图书。

# 第五条 登记、入库、报销

（一）无论新购或其他形式获得的图书均应进行编号登记，编号规则为首字母表示采购地 L-廊坊，S-石家庄；GF-专业标准规范类，SC-专业手册类，TJ-专业图集类，QT-其他图书类；

示例：L-SC36-03 表示廊坊采购第 36 号专业手册的第 3 本；

（二）无论新购或其他形式获得的图书均需加盖图书专用章；  
（三）图书购回后，由图书管理人员办理入库登记手续；  
（四）图书入库后，有效版本目录及员工借阅情况在网上共享，实现图书动态管理，并随时保持最新版本；  
（五）图书办理登记入库后即可办理报销手续；

# 第六条 管理

（一）图书管理人员对书库图书进行统一管理，建立书库总体图书台帐和借阅登记台帐，并进行动态管理。  
（二）各组织借书人自行管理所借图书；  
（三）图书的持有人和管理人应妥善使用和保管图书，不得转借、损坏；  
（四）标准规范应随国家有关规定，适时更新，并加盖受控标识；  
（五）对已作废的图书，应及时归还图书管理人员，由图书管理人员统一加盖作废章，对保留的作废图书，应予以标识、另存，防止误用；  
（六）不再保留的作废图书，由图书管理人员提出申请，图书管理组织负责人审批后，予以销毁，并将所得的钱财递交财务管理部门；  
（七）设计中心内任何人员发现书库图书有作废或未更新版本时，均应及时反馈图书管理人员，由图书管理人员复核信息的准确性，并及时完成图书信息的动态维护和标准图书更新、购买等管理。

# 第七条 借阅、归还

（一）借阅人借阅图书时，廊坊入库图书需在燃气化工设计中

心办公平台标准图书借阅系统上登记后，到图书管理人员处办理签字借阅手续；石家庄入库图书，可直接到图书管理人员处办理签字借阅手续。

（二）当本人可实现签字借阅图书时，不得让他人代借。本人无法实现签字借阅图书时，应由代借人在借阅登记表上签署借阅人及代理人姓名。  
（三）借阅人借阅图书后，应及时归还，以便其他人员借阅；  
（四）借阅人归还图书时，经图书管理人员核对图书后借阅人在借阅表签字确认归还，图书管理人员在图书网站上进行归还确认；  
（五）调(离)职员工应将个人借阅所有图书归还给图书管理人员，图书管理人员通过工作流系统确认归还，方可继续办理调(离)职手续。

# 第八条 丢失、赔付

（一）借阅人不得在借阅的图书上批改、圈点、画线、折角、涂写，如有破损或遗失等情况，求按图书的定价进行赔付，损坏情节严重时，可要求加倍赔付。  
（二）中心内部员工之间私自转借图书，并在转借期间发生的图书损坏或丢失，由原借阅人承担赔偿责任；  
（三）赔付时，借阅人应提出赔付申请，经图书管理人员确认、技术质量智能能力群召集人审批后，借阅人凭审批后的申请单到财务管理部门办理缴款手续。

# 第三章 附则

第九条 本规定由技术质量智能能力群负责解释、修订。

第十条 本规定自发布之日起执行。
