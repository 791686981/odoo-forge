# 能源工程技术有限公司设计文件及技术文件编号规定

# 1 总则

# 1.1 目的

规范公司设计文件及技术文件的编号原则。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 各设计项目组人员负责按此规定对设计文件进行编号。  
2.2 技术赋能群负责综合设计项目工程代号的编排下发，各区域召集人负责本部门设计项目工程代号的编排下发。

# 3.1.3 年号和项目序号

（1） 年号：任务下达时年份。  
（2） 序号：下达任务流水号，由 00001～99999

# 6 文件修改和补充的编号

6.1 设计文件必须精心编制，严格校审，一旦完成入库后，一般不得轻易作修改或增减内容，如因各种原因需要修改原设计时，必须按照本章下列各条规定。  
6.2 前期工作和初步设计文件的修改  
6.2.1 在设计文件归档验收后到文件发送建设单位之前这段时间内，如果需要对设计文件进行修改或补充时，由设计经理提出修改原因，总工程师同意，可修改相关文件，重新出版，但不修改编号（原文件编号不变）。  
6.2.2 当设计文件发送建设单位后，如果需要对设计文件进行修改或补充时，若原设计文件不作废，仅作局部修改和补充，则修改补充件的工程代号同原文件，仅在文件或图纸编号后加修改号，如：改01、改 02……；如原设计文件作废，重新修改设计，工程代号不变，用版次以资区别，首次出版的版次为0 版。  
6.3 施工图设计图纸的修改  
6.3.1 设计部门在完成设计图并归档验收后到设计底图交付晒图前这段时间内，如果需要对设计图纸进行修改或补充时，由设计专业提出修改原因，总工程师同意后，可修改相关文件，但不修改编号和作修改标记。  
6.3.2 设计底图、电子文件入库后且已晒出蓝图，在蓝图发送建设单位之前这段时间内，如果需要对设计图纸进行修改或补充时，应重新办理入库手续，进行修改，重晒蓝图，原蓝图相应作废，不修改编号和作修改标记（原图号不变）。

6.3.3 当设计蓝图发送建设单位后，如果需要对设计图纸进行修改或补充时，应重新办理入库手续，进行修改，重晒蓝图，并通知业主调换蓝图，原蓝图作废。原文件编号视不同情况作如下修改。  
6.3.3.1 整套图纸变更，即原套图纸作废，重新设计出图，需要对其版次进行修改，首次出版的版次为0 版。若原图有保存参考价值的，经，总工程师同意，可保留不予作废。  
6.3.3.2 个别图纸修改，如原底图不作废，就在原底图上作修改，将需修改的尺寸、文字或图形用细实线划掉，换注新尺寸、文字或图1 2 形，并标注修改标记，第一次用△、第二次用△……。

例如：
