# 新地能源工程技术有限公司固定资产管理制度

# 第一章 总 则

第一条 为有效规范固定资产管理，保障资产安全，提高资产使用效率，确保资产的保值增值，依据《新奥天然气股份有限公司固定资产管理办法》，并结合公司实际情况，特制定本办法。

第二条 本办法所指固定资产是指同时具有下列特征的有形资产：

（一）为生产商品、提供劳务、出租或经营管理而持有；  
（二）单位价值在 1000 元（含）以上；  
（三）使用寿命（即业务组织使用固定资产的预计期间或者该固定资产所能生产产品或提供劳务的数量）超过一个会计年度。

第三条 按照新地能源工程技术有限公司（以下简称“新地能源”）下属业务组织资产的不同情况，固定资产可分为房屋、构筑物、办公设备、生产设备、储存设备、消防安全设备、分析实验设备、运输设备、电气设备、仪器仪表检验设备、电子产品、通讯设备、家具用具共十三大类。各类别资产内容及使用年限、折旧率和折旧方法，具体详见附件1。

第四条 本办法适用于新地能源本部及所属分、子公司等业务组织。

# 第二章 组织与职责

第五条 公司的固定资产实行三级管理，其中财运智能能力群为一级管理组织，行政与社会联系智能能力群和招采中心为二级管理组织，使用部门为三级管理组织。三个级别资产管理组织需依照管理口径按要求对在不同账套挂账的固定资产进行全面管理。

第六条 财运智能能力群为固定资产一级管理组织，主要职责如下：

1.负责建立固定资产分级管理体系，明确不同层级组织职责，并监督体系运行情况；  
2.负责固定资产管理制度的制定与传达；  
3.负责组织固定资产的清查、盘点、检查，评估固定资产的管理效果，按要求编制固定资产盘点/分析报告；  
4.负责审核归口管理部门提交的固定资产盘盈、盘亏、毁损、调拨、出售、报废等处理事项，提出处理意见；  
5.负责参与固定资产的采购、招标、验收及重大固定资产投资方案的立项和论证。

第七条 行政与社会联系智能能力群是固定资产二级管理组织，主要职责如下：

1.负责土地、房屋建筑物、运输设备、文体设备、图书文物及陈列品、办公设备及家用家具等资产的采购、验收、变更、出售、报废、维修维护；  
2.负责上述固定资产证照的保管，台账的建立与更新，明确资产所在组织/使用人，按要求组织完成盘点工作；  
3.负责统计收集各使用部门上述固定资产投资预算，并组织实施；  
4.负责组织对自建固定资产建造过程进行管理；  
5.负责组织报废固定资产的技术鉴定；  
6.负责处理盘盈、盘亏、毁损的固定资产；  
7.负责固定资产的评估、接收、移交等事项；  
8.负责对资产使用组织此类型资产的管理与使用情况进行监督指导；  
9.负责确定办公资产配备标准，并按照年度预算作出调整。

第八条 招采中心是固定资产二级管理组织，主要职责如下：

1.负责机器生产设备、仪器仪表设备、存储设备、电器设备、分析实验设备等机器设备和电子设备资产的采购、验收、报废、技术改造等工作的审核审批；

2.负责上述资产台账的建立与更新，明确资产所在组织/使用人，按要求组织完成盘点工作；  
3.负责对业务组织此类型资产管理和使用情况进行监督指导。

第九条 使用部门是固定资产的三级管理组织，主要职责如下：

1.负责按要求使用资产，确保资产安全、完整；  
2.负责设立专/兼职资产管理员，编制和管理固定资产实物台账及资产变动情况表，实时掌握增减变动情况；  
3.负责保障和维护闲置实物资产和零附件的完整无缺，未经批准不得随意将附件和零部件拆卸，挪做它用；  
4.负责对所使用资产的完整性、完好性和安全性负责，并按要求做好资产的清查盘点工作；  
5.负责向归口管理部门提交固定资产需求计划和调出、报废、出售申请；  
6.负责协助对毁损、报废固定资产的责任认定和技术鉴定。

第十条 技术智能能力群、客品智能能力群、QHSE智能能力群等相关部门需在职责范围内协同赋能完成资产管理相关工作。

# 第三章 固定资产管理内容

第十一条 固定资产管理主要包括资产增加、资产使用、资产调拨与借用、资产出租、资产盘点、资产封存、资产处置、资产信息与文档管理、资产的监督与检查等九项内容。

# 第十二条 固定资产增加管理

固定资产增加分为购置增加，自建或扩建增加，业务组织并购增加，接受投资增加，其他等不同增加方式。

# （一）购置固定资产

1.购置固定资产属于项目投资类型的，需按规定进行审批，根据设计文件编制采购清单，并依清单进行购置。购置资产属于非项目投资类型的，需按规定编制采购预算并经线上审批实施购置；未上工作流的业务组织采购需由使用组织编制《固定资产购置申请单》（详见

附件 2），并按照经营授权的规定程序审批。

对于限额以上固定资产的购置，需执行公司采购招标管理相关规定和要求。

2.固定资产购置到货后，采购部门需及时通知并联合相关部门进行验收，填写《固定资产增加验收单》（详见附件3），对资产价格、发票、设备附件、设备质量和数量等进行认定，资产的三级管理组织需及时登记和更新固定资产台账。  
3.财运智能能力群在录入新增固定资产信息时，需查验新增资产是否属于旧资产更换。符合此种情况时，需由资产使用部门或资产管理部门出具有关旧资产的报废清理说明，并明确是否执行固定资产清理程序。如未执行财运智能能力群不得录入新增固定资产信息，以避免账面资产虚增。  
4.资产采购部门需及时收集、整理固定资产文档资料，在购置的固定资产验收完毕后，及时移交资产归口管理智能能力群审核后存档。

# （二）自建、扩建增加固定资产

1.自建、扩建固定资产需根据生产经营计划和技术要求，由业务组织提出申请并参照公司投资管理相关规定，经分析论证后列入年度预算，经审批后实施。  
2.自建、扩建固定资产须按规定通过招投标方式确定施工单位，并签订施工合同实施固定资产建设。资产归口管理部门对项目实施全过程管理，技改项目由技术智能能力群牵头对项目实施全过程管理，资产归口智能能力群协同。  
3.建设完工交付使用的固定资产，由资产归口管理部门牵头，会同建设组织、财运智能能力群及其他相关部门/组织，共同填制《固定资产增加验收单》（详见附件 3），改/扩建工程填制《固定资产改良验收单》（详见附件4），作为办理资产增加业务的依据。  
4.资产竣工交付验收后，固定资产归口管理部门需协调办理结算审计工作；由外部中介机构实施结算审计的，固定资产建设相关部门

配合完成。财运智能能力群根据结算审计结论做好项目费用的归集分配，合理确定固定资产入账价值。

5.资产归口管理部门需及时归集和整理自建、扩建固定资产文档，主要包括立项、概预算、工程监理、结算单、施工图、竣工图、竣工验收资料等，并按照公司文档管理相关规定和要求进行归档。

# （三）业务组织整体并购增加固定资产

1.因业务组织整体并购增加的固定资产，并购项目组需制定完整的资产接收方案，该方案必须与并购协议等并购文件严密衔接。  
2.并购前，并购项目组需向被并购方要求提供照固定资产清单信息、准备并购协议，沟通资产评估方案；资产接收时，并购项目组和接收组必须共同做好固定资产清查，对其质量和数量作出认定，清查发现重大差错时，需提出调整并购价格等意见；并购后业务组织固定资产归口管理部门和使用部门需及时登记固定资产台账，财运智能能力群需根据资产接收清查结论、并购协议、资产评估结论和政府核准、备案文件、资产交接文件和交接清单等做好固定资产会计记录和调整。  
3.并购项目组及并购后业务组织的固定资产归口部门和使用部门须及时整理如下资料并进行归档：

（1）资产接收清查结论；  
（2）并购协议；  
（3）资产评估结论；  
（4）政府核准、备案文件；  
（5）资产交接文件和交接清单。

# （四）接受投资增加的固定资产

业务组织接受固定资产投资，需首先核实接受投资的资产与业务组织未来产业是否匹配，并在合作协议中予以明确。固定资产归口管理部门、财运部门以及投资方有关人员对用于投资的固定资产进行清查交接，接受投资的业务组织依据交接单、投资协议、评估结论及政府的核准/备案文件等相关资料对接收的资产进行账务处理，固定资

产归口和使用部门登记资产台账。

# （五）其他方式增加的固定资产

对于非货币性资产交换、债务重组、融资租赁、接受捐赠等方式增加的固定资产，必须签署相应的法律文件，项目承办人员会同接收业务组织财运和归口管理部门认真清点，认定固定资产数量和质量，其认定结论与有关协议、资产评估报告、政府核准、备案文件、资产交接文件和交接单一起作为资产入账依据，资产管理部门做好台账登记。

# 第十三条 固定资产使用管理

# （一）固定资产使用原则

1.合理利用，充分发挥资产使用效率。各智能能力群和业务组织对固定资产做好统筹安排，使资产达到满负荷运转状态，做好资产运作，适时更新，或采用出租、置换、投资等运作手段，发挥资产的最佳效用；同时对关键设备的功能性贬值和经济性贬值实施检测，定期  
2.加强资产运营维护，保障资产安全完整。各智能能力群和业务组织需根据固定资产习性，做好资产保养、日常维修和大修理工作，使其达到正常使用状态，延长使用寿命；同时保证资产的安全使用，安全管理，避免其损毁或丢失。  
3.强化信息管理，保证资产信息真实完整。各业务组织和相关智能能力群需加强固定资产的信息管理，建立健全固定资产台账，定期盘点核对，确保其信息真实可靠。

# （二）固定资产的维修维护

1.各业务组织需明确固定资产维修管理的具体要求和流程并严格执行，对固定资产保养、日常维修和大修理实施预算管理；通过维修确保资产处于良好状态，防止其实体性贬值。固定资产维修保养需形成台账。  
2.固定资产大修理后，资产归口管理部门和使用部门，以及财运智能能力群需根据结算资料及时更新资产台账。

3.固定资产归口管理部门每年须对资产使用、维修情况进行检查、监督，使用部门对检查发现的问题进行整改。  
4.资产归口管理部门应实时掌握资产使用情况，组织做好日常维护工作，并负责公共资产的维护；个人使用资产的维护由使用者本人负责。

# 第十四条 固定资产调拨与借用

# （一）固定资产调拨

固定资产调拨是指固定资产在新地能源所属独立核算的业务组织内部或业务组织之间的调出和拨入行为，调出组织不再拥有资产所有权。拨出业务组织要向拨入业务组织提供调出固定资产的必要资料和信息，并将其原始价值、已提折旧额等信息通知拨入业务组织。

1.固定资产在资产所有组织内部智能能力群之间变更使用或新地能源所属业务组织之间、新地能源公司本部与业务组织之间均实行无偿调拨，由资产调入智能能力群填写《固定资产调拨/借用审批单》（详见附件 5），经使用部门领导、资产归口管理部门和财运智能能力群审核批准后执行。

其中业务组织之间调拨、公司本部与业务组织之间调拨均需资产管理组织及时维护SAP成本中心，由资产调出组织成本中心变更为调入组织成本中心。

2.向公司以外的新奥成员企业调拨原则上实行有偿调拨，参照出售程序执行，其转让价格原则上按转让时现行市场价值确定。对于因特殊情况未能实行有偿调拨的，需履行特殊审批程序，相关组织做好台账登记与核对工作。  
3.人员离职或人员调动至公司以外的新奥成员企业，对所使用资产均需办理资产归还手续，如未按要求归还则由离职/调动前所在组织负责。

# （二）固定资产借用

1.固定资产借用是指在资产所有业务组织内部的不同部门之间

或新地能源所属不同业务组织之间实施的行为，资产借出单位仍拥有资产所有权。资产借用期限原则为3个月，3个月以内为无偿借用，3个月以上按资产出租处理。

2.固定资产借用需由借用组织经办人填写《固定资产调拨/借用审批单》（详见附件5），按规定审批后执行；借用资产需按时归还，由借出组织进行验收，并在《固定资产调拨/借用审批单》中填写归还信息；如需续借应重新填写《固定资产调拨/借用审批单》，办理借用手续。  
3.人员调动需临时使用原固定资产的，按资产借用条款执行，资产借出期间由借用组织负责维护保养。

# 第十五条 固定资产出租

固定资产出租需按规定履行审批程序。

# （一）固定资产出租给内部业务组织

固定资产出租给内部业务组织是指新地能源工程技术有限公司所属业务组织之间的租赁行为。

固定资产在内部业务组织之间进行出租实行有偿使用，租金收取标准由双方协商确定，其他处理流程参照出租给关联业务组织。

# （二）固定资产出租给关联业务组织

固定资产出租给关联业务组织是指以下情形：

1. 新地能源工程技术有限公司及所属业务组织将资产出租给新奥能源（02688）下属企业及新奥股份（600803）下属企业的行为，包括但不限于将资产出租给新奥新能源工程技术有限公司、新地数安（河北）管道检测有限公司、廊坊华卓建设工程有限公司、上海国际化建工程咨询有限公司。  
2. 新地能源工程技术有限公司及所属业务组织将资产出租给与新奥有关的合营或联营企业的行为。

关联业务组织之间租赁固定资产实行有偿使用，双方签订租赁合同，对资产租赁期限、维护保养责任、租金收取标准、结算方式等进

行明确。合同签订后，报财运智能能力群进行关联业务账务处理，由出租方向租入方开具租赁业务完税发票。资产移交当月出租方有关智能能力群登记资产明细台账，租入方在租赁期间履行资产日常管理职责。

出租给关联业务组织的固定资产，由资产归口管理部门负责组织移交和到期后资产的回收验收工作。

# （三）固定资产出租给外部组织

固定资产出租给外部组织是指新地能源及所属业务组织将资产出租给非新奥成员企业的行为。

1.固定资产出租给外部组织需由资产归口管理部门填写《固定资产出租审批单》（详见附件6），按规定审批后实施。未经批准，任何业务组织和个人不得向外部组织出租固定资产。如出租方在一段时间内经常反复地从事对外租赁业务，可一次性对该类业务办理出租审批。  
2.租赁双方必须签订租赁合同，对租赁期限、维护保养责任、租金收取标准、结算方式等进行明确，合同签订后报业务组织财运智能能力群和上级资产管理部门备案。出租固定资产必须向对方收取一定数量的租赁保证金，具体标准由租赁双方协商确定；实际收取的月租金收入，原则上不得少于当月计提折旧额和应负担的税金之和，特殊情况需出具书面申请并经出租方运营一号位批准。  
3.租赁期结束返还资产时，资产归口管理部门应填写《固定资产出租回收单》（详见附件7），并组织验收，详细检查和记录资产状况。资产完好且附属设施齐全的，退回保证金；发现问题必须出具详细说赁保证金或追加赔偿。

# 第十六条 固定资产盘点

（一）每半年末，财运智能能力群制定固定资产盘点方案，各业务组织成立以分管领导为组长的盘点小组，按方案要求对本业务组织全部固定资产进行盘点。盘点时需将实盘结果与固定资产财务账和固

定资产台账进行逐一核对，对盘点结果形成盘点报告，并对资产管理情况进行分析，经公司分管领导或一号位审批后存档，并报上级组织财运智能能力群备案

1.经盘点确认盘盈、盘亏的资产，固定资产归口管理部门和使用部门须及时查明原因，提出处理方案，并按照《新地能源工程技术有限公司经营授权表》规定权限审批后进行账务处理。属于固定资产丢失的必须落实责任人，由使用部门提交书面说明报资产归口管理部门和财运智能能力群进行责任认定，由责任人按照资产丢失日净值进行赔偿。资产使用部门同时办理资产报废和保险索赔等事宜。  
2.对于盘点过程中发现的固定资产损毁，资产归口管理部门和资产使用部门及财运智能能力群必须进行损失鉴定，填制《固定资产损失责任鉴定表》（详见附件 9），明确责任，并按以下方式处理：

（1）属于人为因素（故意或操作不当等）造成的资产损毁，资产使用部门需落实责任人并提交书面说明，财运智能能力群协同归口管理部门进行损失鉴定，由直接责任人按照鉴定的损失承担相应的赔偿责任。  
（2）非人为因素（自然或地质灾害等）造成的资产损毁需先报保险，由资产使用部门写明具体情况，报资产归口管理部门和财运智能能力群进行损失鉴定，办理非正常报废手续，同时实施保险索赔。  
3.对于盘亏和损毁的资产，财运智能能力群必须及时向主管税务机关办理税前列支审批手续。资产丢失和人为因素造成固定资产损毁而不能落实具体责任人的，由资产使用部门领导和资产管理人员承担赔偿责任。

（二）业务组织进行资产重组、兼并、收购时，须对重组和并购的资产进行盘点。

第十七条 固定资产封存。对于暂不使用的固定资产做封存处理，由资产使用部门提出申请，报固定资产归口管理部门核实后，填写《固定资产停（启）用通知单》（详见附件 8）办理封存手续。重新启用该资产时，由固定资产使用部门填写《固定资产停（启）用通知单》，

实施资产启用。

# 第十八条 固定资产处置管理

# （一）固定资产出售

1.对闲置和使用不经济的固定资产进行出售时，由使用业务组织填制《固定资产处置审批单》（详见附件 10），按照《新地能源工程技术有限公司经营授权表》规定的权限履行审批手续。  
2.出售净值在 50 万元（含）以上的固定资产，必须实行招投标净价转让，并与出售的相关凭证（包括招投标等有关文件资料）一起作为会计处理的依据。

# （二）固定资产报废

1 .正常报废的固定资产。对于达到使用年限且无继续使用价值，或不能正常发挥使用价值且无其他处理方式的固定资产予以正常报废，由资产使用业务组织填制《固定资产处置审批单》（详见附件10），并按照《新地能源工程技术有限公司经营授权表》规定的权限执行审批后进行会计核算。  
2.非正常报废的固定资产。非正常报废的固定资产必须及时查明原因，由资产使用部门提交情况说明，然后按照《新地能源工程技术有限公司经营授权表》规定的权限履行审批手续。未达到使用年限的资产按照财务账面净值进行损失确认。  
3.经批准报废的固定资产，资产使用部门、归口管理部门及财运智能能力群要及时进行清理，更新资产台账和记录。

# （三）固定资产对外投资和捐赠

1.业务组织对外投资转让的固定资产，应依据签订的投资协议、评估报告、接收清单等有关资料，办理资产转移手续。  
2.业务组织对外捐赠的固定资产，必须按规定进行逐级审批，且捐赠业务组织与受赠业务组织需及时办理资产交接手续。

# 第十九条 固定资产信息与文档管理

# （一）固定资产信息管理

1.各资产所有业务组织须建立健全固定资产管理台账，资产使用

业务组织和资产所有业务组织不一致时，双方均需及时维护固定资产管理台账，注明资产性质（调入、调出、借入、借出等），以匹配核对。针对组织内资产在不同账套挂账的情况，业务组织需明确资产管理层级和职责，建立和维护全部的资产信息，并按照核算账套等不同维度进行归集，实时掌握资产现状和动态。

2.各业务组织资产归口管理部门和财运智能能力群须指定专人登记固定资产台账，包括资产名称、类别、编码、取得年限、取得方式、使用单位、现行状态及变动情况等信息，各层级管理部门间至少每半年核对一次。资产归口管理部门和财运智能能力群统一编制确定固定资产编码，做到每项资产和每个编码一一对应。固定资产购置时应配置资产编码牌，并对资产拍照作为附件存档，资产台账中应备注资产购置厂家及保修期限。  
3.各业务组织资产管理员须保持稳定，因工作需要或其他情况变动的，必须填写工作移交清单，注明所负责台账期间，移交的各项资料及业务组织需说明情况等，工作移交清单必须经交接双方签字确认，主管领导监交签字。  
4.各业务组织固定资产信息化系统建设必须能够满足固定资产管理要求，已经建立固定资产信息化系统的业务组织必须遵循固定资产主数据标准，对固定资产进行分类管理，并及时维护固定资产数据，不断提高固定资产管理水平。

# （二）固定资产文档管理

1.固定资产文档包括固定资产购置申请、招投标资料、购置批准说明书、相关技术资料、翻译资料、报废处置资料及记载固定资产本身性质状态发生变化的资料，如改良、大修理等。  
2.固定资产文档管理要求执行新地能源文档管理相关规定。

# 第二十条 固定资产管理监督检查

# （一）固定资产监督检查的主要内容：

1.固定资产管理及岗位人员设置情况：重点检查是否存在不相容

岗位混岗情况；

2.固定资产授权审批执行情况：重点检查采购、验收、付款、处置、调拨、出租等是否有按要求进行了申请审批，是否存在越权审批情况；  
3.固定资产投资预算管理情况：重点检查购建固定资产是否纳入预算管理，预算的编制、调整与审批程序是否正确合规；  
4.固定资产日常保管保养情况：重点检查固定资产的归口和分级管理，岗位责任制是否有效落实，维修保养记录是否齐全且合规；  
5.固定资产的处置情况：重点检查资产处置是否有效履行审批手续，作价是否合理，旧设备更换新增资产卡片是否有相应的旧设备资产清理情况说明；  
6.其他方面：重点检查台账建立、对账机制及制度建设是否健全，制度是否符合实际情况且有效执行。

（二）固定资产检查人员在检查过程中发现固定资产管理的薄弱环节及问题时，需告知责任部门，并按照管理权限向上级部门报告。责任部门必须及时查明原因，采取措施加以纠正和完善，并将整改落实情况反馈给检查部门和上级资产管理部门。

# 第四章 奖 惩

第二十一条 对因使用或管理不善造成固定资产损毁的，按照本办法第十六条相关规定由责任人赔偿，恶意造成固定资产损毁的，从重处罚。对于其他资产管理违规行为按照《新奥股份员工违规违纪行为处罚办法》进行处罚。

# 第五章 附 则

第二十二条 本办法由新奥能源财运智能能力群负责解释、修订，各业务组织可根据自身固定资产管理需要和实际情况，在此制度基础上进行细化。

第二十三条 本办法自2024年1月1日起实施。
