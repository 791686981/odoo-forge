# 能源工程技术有限公司员工培训管理办法

# 1 总则

# 1.1 目的

为规范员工培训管理，提升培训效果，完善培训体系，构建能力提升机制，促进员工能力提升，特制定本办法。

# 1.2 管理原则

(1)人人受培训，人人培训人；  
(2)以开发员工核心专长、核心技能为主。

# 1.3 适用范围

本制度适用于能源工程技术有限公司所有单位。

# 2 组织与职责

2.1 各级人力资源管理部门须根据本制度，负责所属单位的所有培训管理工作。  
2.2 公司人力资源部统筹管理公司所有培训工作，主要负责：

(1)制定公司培训管理制度和相关政策；  
(2)建立公司培训管理体系；  
(3)制定公司年度培训计划并跟踪落实；  
(4)对所属各单位的培训管理工作提供指导。

2.3 各分公司培训管理部门负责本企业培训的归口管理，主要负责：

(1)制定本企业年度培训计划并跟踪实施情况；  
(2)对企业个部门的培训工作提供指导；

2.4 公司各部门主管领导是其主管部门培训工作的第一责任人，负责指导、督促其直接下属提高素质和技能。

# 3 培训类型和要求

# 3.1 新员工入司培训

新员工入司后必须接受入司培训，入司培训计划由培训管理部门制定。因个人原因未参加入司培训或在入司培训过程中表现较差

的员工属于不符合公司录用条件，公司有权解除与该员工的劳动合同。

3.2 在职培训  
3.2.1员工在职期间须参加培训和学习，培训以内部培训为主，当内部培训无法满足需要时，允许参加外出培训。  
3.2.2 在职学历教育培训具体要求见《能源工程技术有限公司员工在职学历教育管理办法》。  
3.2.3 非在职学历教育的外出培训须填写《外出培训申请表》，并附上培训内容的相关资料，经部门负责人和分管领导审核后，报人力资源管理部门审核。人力资源管理部门根据实际情况确定是否需要在培训前与参训员工签订《培训与服务期协议》。  
3.2.4非在职学历教育的外出培训借款或保险手续，具体按《能源工程技术有限公司员工借款与财务报销管理规定》执行。住宿和交通费用标准按照《能源工程技术有限公司员工出差管理规定》执行。  
3.2.5 外出培训的员工有义务将学习知识进行传承，学习资料交由本单位培训管理部门存档，否则不予报销外出培训相关费用。

# 4 培训组织

4.1 培训计划与预算

4.1.1 各单位培训管理部门依据业务对人员能力要求和员工队伍能力要素现状，制定本单位培训规划并落实到三一计划和计划预算中。  
4.1.2 各级经营单位预算管理部门须结合实际情况在年度计划预算中确定人均年度培训预算标准，报上级单位预算管理部门批准后执行，且年度培训预算总和须控制在本单位员工工资总额的 2.5%以内。  
4.1.3每年 12 月份，各单位培训管理部门组织制定本单位下一年度《年度培训计划》，明确重点培训主题等，经过分管领导审核后，上报公司人力资源部备案。  
4.2 培训实施

各单位培训管理部门须准确记录培训相关信息，主要包括：开始时间和结束时间、培训人数、培训地点、授课讲师和授课学时数等，并于每季度末建立培训管理台账，定期上报公司人力资源部备

案。

# 4.3 培训评估

各单位组织的集中面授培训必须进行培训评估，培训组织部门必须在培训结束后 2 日内完成本次培训的评估汇总，并将评估结果反馈给授课件事和培训组织领导。

# 5 培训师资管理

5.1 培训师资以培育内部师资为主，外部培训资源为辅。公司兼职讲师管理的政策和培养方式由公司培训管理部门统一规划，各级单位培训管理部门负责培训师资的日常管理。鼓励各业务领域的专家、骨干担任兼职讲师，促进知识共享，提升员工能力；同时对兼职讲师给予相应激励。兼职讲师的选聘标准和管理按照《能源工程技术有限公司兼职讲师管理办法》执行。

# 6 培训档案

# 6.1 培训档案的建立

各单位开展的各类培训须建立相应的培训档案，作为各类培训活动评价和考核的重要依据。

# 6.2 培训档案的内容

培训档案的内容包括培训计划实施情况、培训教材或课件、参训人员名单及出勤记录、培训评估结果等。

# 7 附则

7.1 本办法由能源工程技术有限公司人力资源部负责解释、修订。  
7.2 本制度自 2013 年 9 月 25 日发布起执行。

# 7.3 相关管理办法

(1)《能源工程技术有限公司外出培训管理办法》  
(2)《能源工程技术有限公司兼职讲师管理办法》  
(3)《能源工程技术有限公司员工在职学历教育管理办法》  
(4)《能源工程技术有限公司员工借款与财务报销管理办法》  
(5)《能源工程技术有限公司员工出差管理办法》
