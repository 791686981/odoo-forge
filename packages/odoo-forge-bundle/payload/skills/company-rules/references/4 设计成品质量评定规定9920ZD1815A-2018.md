# 能源工程技术有限公司设计成品质量评定规定

# 1 总则

# 1.1 目的

为统一工程设计成品质量评定标准，对项目申请报告、可行性研究报告、初步设计、技术方案、技术附件、施工图或用户有特殊要求的设计文件在交付前或交付后的内部质量评定做出规定，以提高工程设计质量管理水平。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 技术赋能群召集人负责评定结果的审核，总工程师负责评定结果的核准，总经理负责评定结果的批准。  
2.2 专家团队、常规技术指导团队负责组织项目申请报告、可行性研究报告、初步设计文件、技术方案、技术附件、施工图的质量评定工作。  
2.3 总工程师、综合项目群各专业带头人、常规设计群各区域召集人、专家团队、常规技术指导团队和有经验的工程技术人员负责质量评定的实施。  
2.4 技术赋能群负责每季度质量评定的组织、协调、复核、总结工作。

# 3 质量评定的时机

3.1 单项工程出图前，区域召集人每月抽查设计项目进行质量评定应不低于抽查月所出设计项目总数量的15%；技术赋能群组织每季度每个设计部门至少抽取 1 个项目进行质量评定。  
3.2 项目申请报告、可行性研究报告、初步设计、规划项目、技术方案、技术附件等文本类文件，出图前由专家团队、常规技术指导团队

组织进行 100%质量评定。

3.3 综合工程施工图设计由技术赋能群每季度负责组织质量评定并制定质量评定方案。

# 4 质量评定标准

4.1 可行性研究报告、初步设计、技术方案、技术附件等文本  
4.1.1设计文件质量评定采用记分法，分为基本标准评分和设计技术进步加分两部分，基本标准满分为 100 分，当记入加分后满分为 110分。  
4.1.2 设计技术进步加分标准

（1）采用先进技术或有重大革新措施，与同类工程相比，使用功能有显著改进和提高,降低造价和费用。各项技术经济指标达到国内先进水平；  
（2）采用先进的、完善的计算方法、新的结构措施，使计算更符合实际，构造更加合理，并取得明显的实际效果；  
（3）项目内容复杂、技术难度高，但精心设计，取得良好效果；  
（4）凡按基本标准评为良级的项目，符合（1）～（3）项 1 条以上的设计技术进步加分标准者，由设计人提出申请，由专家团队、常规技术指导团队讨论复核其评分可增加 1-10 分。

4.1.3 可行性研究报告、可行性研究报告代初步设计、初步设计、技术附件、技术方案的质量基本标准规定分别见附件 1、附件 2、附件  
3、附件 4、附件 5。项目申请报告的质量评定标准可参照附表执行。

4.2 施工图

4.2.1 施工图设计错误按其影响施工、投资成本、生产的程度分为技术性错误、一般性错误和图面错误三类。技术性错误又分为重大和较轻两种。  
4.2.2施工图设计按专业根据错误性质和数量，采用扣分的办法进行评分，满分为 100 分，扣分标准如下：

（1）技术性错误扣分按严重程度执行。对重大技术性错误每个扣 40分，较轻的扣 20 分；重大和较轻技术性错误的区分应视其错误

程度和对工程影响大小而定；

（2）一般性错误每个扣 5 分；  
（3）图面错误每个扣 1 分。

4.2.3技术性错误是指由于设计中的错误将会造成制造、备料、施工安装、投资成本较大浪费及生产检修方面较大的困难，将会造成设计较大的返工，造成安全生产事故或重大经济损失，例如：

（1） 不符合已批准的初步设计及其批文内容或设计委托书内容；  
（2） 不符合国家行业现行的有关标准和技术规定，或标准引用有误；  
（3） 采用的计算公式错误或计算结果错误；  
（4） 选用技术参数、安全系数错误；  
（5） 设计内容不完整，有缺项；  
（6） 重要的技术条件提错或漏提；  
（7） 设备布置错误；  
（8） 设备选型、选材或结构错误；  
（9） 土建结构不完全；  
（10） 重要的尺寸、标高、坐标、数字和小数点写错；  
（11）缺重要的施工质量要求、施工及验收规范和技术条件说明或说明不清，甚至错误；  
（12）所提设计条件、资料（含周边环境资料）不完全或不落实，遗留问题，造成隐患；  
（13）重复利用设计文件选用错误；  
（14） 选用淘汰产品；  
（15） 可能产生技术性错误后果的一般性错误和图面错误。

4.2.4 一般性错误是指在设计的局部细节上存在错误，但对制造、备料、施工安装、生产、安全和检修无较大影响。如：

（1）设计不符合公司设计文件编制规定的要求；  
（2）计算公式或系数选用不当；  
（3）图纸中设备材料的规格、数量与设备、材料表中不符；

（4）图纸目录、图号写错将造成图纸发错或漏发，或图签栏内有关内容不按规定填写；  
（5）次要的尺寸、计算单位写错，标高、坐标漏写或各视图中有矛盾；  
（6）次要的尺寸的设计条件提错或漏提；  
（7）图面表达不清，缺乏必要的剖（视）图，或漏画原有的阀门或其它管件等；  
（8）总尺寸与分尺寸有矛盾；  
（9）设计不符合工程（专业）设计技术统一规定。

4.2.5 图面错误是指制图技术、文件编制质量较差，但不影响施工，如：

（1）说明（说明书）词不达意，难以理解，但不会发生理解相反的意思；  
（2）错别字或标点符号使用不当；  
（3）图纸比例、图纸密度不当，文字资料编写程序不符合有关规定；  
（4）图例符号不符合公司标准的规定；  
（5）设计文件中材料表、计算书等未使用公司规定的表格；  
（6）同一文件中常用符号、计量单位不统一。

# 5 设计文件的质量等级

5.1 设计文件的质量等级根据评分多少，按如下标准划分：

<table><tr><td>等级</td><td>分数</td></tr><tr><td>优级</td><td>≥95</td></tr><tr><td>良级</td><td>86≤分数&lt;95</td></tr><tr><td>合格级</td><td>70≤分数&lt;86</td></tr><tr><td>不合格级</td><td>&lt;70</td></tr></table>

5.2 对初步设计、可行性研究报告、技术附件、技术方案等文本类文件，根据质量标准进行设计文件质量评定时，表中各项评定内容，除特别说明者外，有一项评分为零，无论总分多少，均为不合格文件。  
5.3 对施工图设计文件在进行质量评定时，如发现技术性错误，该产品不得评优；如果技术性错误较轻，视具体情况可评为良或合格，对出现重大技术性错误的设计产品，无论评分多少，一律作为不合格产品处理。  
5.4 评为质量不合格的设计文件，应按《质量手册》执行，修改后的设计文件只能评为合格级。

# 6 设计成品质量评定办法

6.1 可行性研究报告、初步设计、技术附件、技术方案等文本类文件

可行性研究报告、初步设计、技术附件、技术方案等文本类文件，各专业编制完成后进行专业级校审，填写《设计文本校审记录》。各专业负责人将设计文件连同校审记录交于设计经理组稿。设计经理依据质量评定标准，进行初评分值和质量等级的评定,并依据设计阶段选取相应的质量评定表进行评定意见填写。

专家团队、常规技术指导团队对设计文件进行最终评定，完成质量评定表中评定意见、最终分值和质量等级的评定。

各阶段质量评定表详见附件。

6.2 施工图设计文件  
6.2.1施工图设计文件质量评定分专业进行，在此基础上评定本项目的综合质量，并以综合质量评定结果确定本项目的设计质量等级。  
6.2.2 错误统计方法

（1）不应把几个错误合并为一个错误，或少统计错误；  
（2）同一张图上几处出现同一错别字或同一图例符号不符合规定只统计为一个错误；  
（3）同一项目由于一个错误引起相关错误时，只在错误始发处统计一个；

（4）凡属设计方法及观点不同的意见，可不作错误统计。

# 6.2.3 设计成品最终评分计算方法

（1）各专业设计成品评分/张

＝〔查图总张数（折 A1 图）×100分－查图时总扣分〕/查图总张数（折 A1 图）

（2）设计成品最终评分根据所含各专业的设计质量评分，按加权平均的方法来计算，即：

设计文件综合质量评分= 各专业设计成品评分  各专业权重系数各专业权重系数之和

权重系数：主体专业 0.3，土建（含总图）专业为 0.20，其它专业为 0.1。

6.2.4 设计成品优良率计算公式

设计成品优良率＝设计成品优良项目数/检查总项目数

6.2.5 设计成品合格率＝设计成品合格（优良）项目数/检查总项目数。  
6.3 评定结果及评定中发现的问题填入《可行性研究报告质量评定表》、《初步设计质量评定表》、《可行性研究报告代初步设计质量评定表》、《技术附件质量评定表》、《技术方案质量评定表》、《施工图质量评定表》。  
6.4 对存档前质量评定中发现的错误由设计经理组织有关人员修改后交相关部门晒图、复印和发放；对存档后质量评定中发现的错误由技术赋能群组织评审决定是否采取纠正措施。  
6.5 对于设计会审、外审、施工中反馈的质量问题，由技术赋能群根据其错误的严重程度给其责任专业部门相应的扣分。  
6.6 技术赋能群负责对抽查项目质量评定等级及分数的汇总，并填写施工图质量评定汇总表。
