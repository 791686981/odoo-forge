# 能源工程技术有限公司设计文件签署规定

# 1 总则

# 1.1 目的

为了实现设计验证过程，通过设计文件的逐级签署，明确质量职责，保证产品设计质量和做好文件有效性的标识，特制定本规定。对公司向顾客提供加盖设计资质出图印章的正式的工程设计文件做出规定。设计前期工作中的有关文件和图纸也可参照执行。

# 1.2 适用范围

本规定适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 综合项目群各专业带头人及常规设计群各区域召集人负责监督管理设计文件逐级校审和签署。  
2.2 设计经理负责组织必要的设计审定。

# 3 设计人员的签署

3.1《设计开工报告》（TB/ED-9920-201A-2009）中明确各专业的设计、校核、审核、审定等人员，以便在设计过程中密切合作进行技术问题的磋商，及时讨论和制定有关设计方案。各级人员应根据公司发布的设计人员任职资格聘任情况进行确定。  
3.2 各级设计人员应按各级设计人员岗位职责和各专业设计质量控制规定开展校审并对各自的责任范围负责,设计文件经校核、审核、审定后并不能免除前一级人员的责任。  
3.3 校审时，设计人有责任提供校审人员所需的全部设计依据。如原始设计条件、公式和数据来源资料、有关样本和图纸等。  
3.4 校审人员在校审时在校审版设计文件上标注正确与修改标记，如需修改时，可将意见在校审版文件上进行修改和提出意见，并记录在

《设计(文本)校审记录》（TB/ED-9920-214A-2018）上。

3.5 校审人认为设计文件质量低劣、不完整，不具校审条件时，可将文件退回设计人或校核人，并说明理由。  
3.6 各级校审人员签署必须在文件按校审意见修改后经过确认方可逐级签署，签署应字迹清楚，填写完整。  
3.7 校审中各级人员如有不同异议，应进行充分商讨取得一致，否则可逐级上报仲裁，最后设计服从校核，校核服从审核，审核服从审定。保留意见可记录在校审记录中。  
3.8 在设计文件内的“设计”、“校核”、“审核”、“审定”签署栏内不允许一人签两栏。  
3.9 在图纸和表格的同一签署栏内，若有两个设计人或校核、审核人时，可在同一栏内共同签署。  
3.10 成卷（如初步设计）对外发送的文件的首页（扉页）应由设计经理、专家团队评审人员审查批准后签字。

# 4 签署格式、程序和要求

4.1 设计图纸、规格表等文件应在其规定的栏目内签署。  
4.2 文字性设计文件的设计人员列在参加设计人员页中（不签署仅打字），当需要签署时，依据相关编制要求或导则进行签署页设置及签署。具体格式要求执行《能源工程技术有限公司设计文件格式规定》（9920/ZD1821A-2018）。  
4.3 设计文件的签署程序按制图-设计（编制）－校核－审核－专业会签－专业负责人-设计经理－审定的顺序进行，需要时公司领导签署。其中制图、专业会签、专业负责人、设计经理、审定，需要时才签署。前面岗位无签署时，后面岗位不得签署。  
4.4 签署必须用全名，不得用外文、代号、拼音字母等签字，字迹要工整清晰。  
4.5 纸版签字应用钢笔、墨水笔或碳素笔，底图上必须用黑色笔签署，对存档文件不允许代签他人姓名。签字 U 盾由本人保管，发现代签，

签字人、代签人一并处罚。出差人员需要代签时，在征得代签人员同意的前提下，由专业负责人、专业带头人或区域召集人双重确认后，由代签人员进行签署。

4.6 设计文件的签署见附表 1～6。

# 5 附则

5.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
5.2 本制度自颁发之日起实施。
