# 能源工程技术有限公司设计验证规定

# 1 总则

# 1.1 目的

为统一并规范工程设计过程中各级设计人员的设计验证工作方法，提高设计产品质量。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 由设计人员、校核人员、审核人员、审定人员和会签人员负责执行；  
2.2 专业负责人负责专业间互提资料的验证；  
2.3 设计经理负责验证记录的收集及存档。

# 3 验证方法及要求

3.1 设计文件的校核、审核和审定是在设计人员自校后按照各专业的校审细则进行。  
3.2 校核、审核、审定在电子版或纸质图纸中进行，专业间互提资料在原稿上校审。  
3.3 图纸、计算书及各种规格表的校核、审核和审定工作可在同一张图或表格上进行，也可视情况经修改后交下一级验证，其步骤和方法如下：

# 3.3.1 校核

图纸及各种规格表中的数字、符号图纸上的文字说明及计算书校核发现错误后圈出，在圈边注明流水号，然后按注明的流水号将错误及修改意见逐条填入《设计(文本)校审记录》中。

# 3.3.2 审核

发现图上错误的数字、符号、图形、说明和表格以及各种规格表中有错误时圈出，在圈边注明流水号。然后按注明的流水号将错误及修改意见逐条填入《设计(文本)校审记录》中。

# 3.3.3 审定

其步骤和方法与审核相同。

3.3.4对于图面复杂、错误较多的设计文件，校审人员可将重要的问题按上述方法列入校审记录，其它问题仅在图纸上表示清楚即可。  
3.4 区域设计部门与成员企业联合设计的校审

各燃气公司的设计人员将经过自校的设计文件电子版图纸及文字、表格发给校核人员，校核人根据设计要求对设计图纸进行校核，将重要的问题填写在《设计(文本)校审记录》上，并与设计人员沟通确认。校核人员可对原文件做必要的补充与更改，也可将设计(文本)校审记录及电子版文件一并发给设计人进行逐条修改。将修改后的设计文件送交审核人员，审核人员根据设计要求对设计文件进行审核，步骤同校核。

3.5 设计说明书和其它文字资料在电子文档中的校、审步骤和方法  
3.5.1 当发现文字叙述不确切、不全面时，应在校审记录中简要说明应补充、修改的内容；  
3.5.2 当发现错别字或明显的错误时，校核人用红色字体可直接修改，审核人用蓝色字体可直接修改，审定人用绿色字体可直接修改不用填写校审记录。  
3.6 在设计中采用新工艺、新技术、新材料时，还可根据项目中存在的具体问题，组织有关人员采用下列的方法进行设计验证：

（1）采用其他的计算程序或计算方法对设计计算进行验证；  
（2）与已证实的设计作比较。为了验证采用新工艺、新技术、新材料设计的合理性、正确性，可以与类似的、已经过工厂操作验证的设计相比较；  
（3）不同的技术内容可以采用不同的试验方式，如：新开发的工艺

技术经过中型试验鉴定后可以采用工业试验的方式，解决放大过程中的工程问题，验证工程设计的正确性；在施工现场可以采用试桩等方法获得设计数据；

（4）对发放前的设计文件进行评审。

# 3.7 会签

在施工图设计阶段，各专业根据本专业提出的委托条件，对相关专业完成的设计文件是否符合提出条件的要求进行核对。核对无误后进行会签。  
3.8 全部的设计输出文件按《能源工程技术有限公司设计文件签署规定》签署，其中校核人、审核人、审定人应对历次评审结论的落实情况进行验证，设计经理应进行监督检查。  
3.9 专业间互提资料应经验证签署，提出专业将正确的、通过校审的、签署齐全的原件给接受专业，自留复印件。委托资料不允许涂改，如有涂改无论在发放前或发放后均必须由修改人员签署姓名及日期。  
3.10 设计人、校核人、审核人之间的不同意见，以审核人员意见为准。设计人员可根据带有校审标记的图纸和文字资料原稿以及校审记录上的校审意见，在计算机上逐条修改。修改一条，在带有校审标记的图纸和文字资料原稿上对这一条打钩（√），并在校审记录表修改情况栏内说明修改情况。经与校、审人员协商可以不改的则在校审记录上简要注明原因并签署，必须有校、审人员在修改处的签署。  
3.11 设计文件底图需经设计人员自校、校审人员根据校审记录验证修改意见落实后签署、有关专业会签，设计经理验证各级签署（包括日期）齐全后交相关人员出版、存档。  
3.12 校核记录、审核记录、审定记录均采用同一格式的《设计(文本)校审记录》表，可共用同一张记录表，但必须按校核、审核、审定的顺序分开记录。当校审人员无意见时，应在校审记录上写明无意见。《设计(文本)校审记录》作为项目文件的内容之一存档。  
3.13经会签、签署完整的设计成品归档、交出版部门验收。

# 4 附则

4.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
4.2 本制度自颁发之日起试行，试行期为 6 个月。6 个月内若无修订，则自动转入实施。
