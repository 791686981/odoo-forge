# 新地能源工程技术有限公司石家庄能源化工技术分公司设计经理分级管理办法

# 第一章 总则

# 第一条 目的

为打通设计经理职业发展通道，激发其创值热情和自我提升的动力，为培养具有 EPC 项目视野的高水平设计经理打好基础，特制订此办法。

# 第二条 管理原则

（一）公正、公平、公开原则：设计经理的分级秉承公正、公平、公开原则，所有评定过程和结果均在评定小组成员的见证下透明操作，确保所有设计经理在同一标准下进行评定。  
（二）激励原则：设计经理的分级体现了对个人发展的激励效目管理能力。  
（三）引导与提升：引导一批爱岗敬业、乐于奉献、敢于担当、德才兼备的设计人员快速成长，尽快具备设计经理的能力。

# 第三条 适用范围

本办法适用于燃气化工设计中心正式聘任的设计经理。

# 第二章 分则

# 第四条 组织与职责

（一）综合智能能力群是设计经理分级管理的归口管理部门，主要职责：

1.负责对设计经理的分级工作进行系统规划，建立设计经理分级档案，对设计经理实施动态管理，建立能上能下、优胜劣汰的用人机制。  
2.负责组织成立设计经理评定小组，开展设计经理分级评定、评估等相关工作，并将评估结果及处理意见形成文件报一号位审批,报

备上级公司,并负责设计经理约定创值分享审视等相关工作。

3.负责参与评选人员基本情况的收集与汇总。   
4.经营、财务负责提供所管辖范围内评估项目的数据（具体详见评定标准）等信息。

（二）技术质量智能能力群、市场开发群是设计经理分级管理的协助部门，主要职责：

1.负责为综合智能能力群提供所管辖范围内评估项目的数据（具体详见评定标准）等信息。技术质量智能能力群同时负责网络设计平台设计经理资格管理、能力标签等信息动态管理。

# （三）设计经理评定小组

1.根据综合智能能力群的安排，对具备任职资格条件的设计经理按评定办法进行评定，并形成评定意见。  
2.设计经理评定小组人员组成：

组长：一号位

组员：理正团队成员、技术委员会成员、综合智能能力群、技术质量智能能力群、市场开发群及各设计业务群/专业群负责人。

（四）燃气化工设计中心一号位负责对设计经理分级评定的结果及各项问题的处理意见进行最终的审批。

第五条 设计经理任职资格要求、评定标准等相关说明

（一）设计经理职责详见新地能源工程技术有限公司石家庄能源化工技术分公司 EPC 项目设计管理办法。

# （二）设计经理分级

一级设计经理、二级设计经理、三级设计经理。

# （四）评定原则

1.设计经理的聘任须符合任职资格要求，特殊人员经一号位审批后方可聘任。  
2.根据设计人员从业经验，在初次聘任设计经理时，原则上不高于三级设计经理（特殊人员经一号位审批后可聘任为二级设计经理），依据评定标准进行评定，逐级晋升。  
3.一级设计经理在负责大型项目时可以申请配备设计副经理（具备三级设计经理以上的人员担任），经分管理正同意后可执行;当具备设计经理资格的人员，在项目中担任项目设计副经理职位时，所得分数是所参与项目得分的 50%。  
4.持有注册证书的设计人员均具有设计经理资格，可根据项目需要进行项目任命。

# （六）评定方式

# 1.评定周期

设计经理分级评定工作每年组织一次（1 月份），参考最近一年的工作业绩（截止时间为每年 12 月份（含）），由综合智能能力群牵头组织，对公司所有具备评定条件的设计经理进行全面、系统的评定，并根据评定结果进行相应的级别调整工作。

遇特殊情况需要临时组织评定的，由综合智能能力群拟定评定申请，上报一号位批准后组织评定。

# 2.评定方式

由综合智能能力群按本办法要求按期组织评定小组开展评定工作，本着公平、公正、公开原则，同级资格人员横向比较，严格参照2.5 评定标准进行评定。

符合晋级条件的设计经理，需进行个人答辩，答辩通过者方可晋级。

# 3.评定结果

按照表 2.5-1 的评定标准分别对每位设计经理进行评定，最终评定结果按照如下方式执行：

（1）综合评定得分 80（含）～100 分，具备晋级条件；  
（2）综合评定得分 60（含）～80 分，保级；  
（3）综合评定得分 0～60 分，降级，原则上三级设计经理取消其设计经理资格。  
（4）评定最终结果经理正团队成员、技术委员会成员评审确认后,进行聘任公示。

注 1：理正团队成员、各组织召集人、专业总工、技术委员会成员、能力群成员已聘任为设计经理的，如当年综合评定得分低于 60分，但因业务需要保留其设计经理资格，须经评定小组合议并报一位审批后方可保留原设计经理资格及等级。

注 2:因年度评定而降级的设计经理，所负责项目处于正在进行状态时，仍担任该项目设计经理，直至项目完成。

# 第六条 补充

（一）设计经理优先考虑专业晋级。

（二）严格按照级别对标的项目规模进行项目分配。  
（三）其他相关待遇按照燃气化工设计中心相关制度执行。

# 第三章 奖惩

第七条 对不认真按照本办法开展工作，在设计经理分级评定过程中徇私舞弊、弄虚作假、提供不实证明材料的个人，视情节轻重给予取消评定资格、罚款、降薪、降级直至解除劳动关系等处罚。

第八条 取得相应资格等级的设计经理，如果在工作中出现违规、违纪等情况，公司有权在当月即对其进行降级或取消其设计经理资格。

第九条 年度考评优秀的设计经理，视情况予以一定金额的奖励，由所在组织召集人申请，经综合智能能力群、分管理正审核，一号位审批后在当月工资中发放。

# 第四章 附则

第十条 本办法由综合智能能力群负责解释、修订。自颁布之日起实施。
