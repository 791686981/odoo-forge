# 能源工程技术有限公司《设计开工报告》编制规定

# 1 总则

# 1.1 目的

为使项目遵循统一的设计原则、设计规定及质量控制标准，明确可行性研究、初步设计和综合工程施工图设计的工程内容、规模、设计范围及专业分工、人员安排、进度计划等内容，确保项目顺利实施。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 设计经理负责编制项目《设计开工报告》。  
2.2 综合项目群各专业带头人协助设计经理进行项目人员安排及分工。  
2.3 综合赋能服务群、市场开发群协助设计经理进行设计依据、工程概况的收集。

# 3 《设计开工报告》编制要求

《设计开工报告》在项目初始阶段进行编制。后一设计阶段开工报告的内容可在前一阶段的开工报告上进行修改或补充，修改或补充部分应附其后。

3.1 可行性研究报告、初步设计《设计开工报告》的编制提纲

# 3.1.1 设计依据

根据项目的具体情况，列出下列有关文件的名称、文号、审批单位及日期：

（1）有关合同（设计合同、引进技术合同等）；  
（2）应遵守的规范、规章制度等；  
（3）各种报告及其批文（前期工作文件：项目建议书、可行性研究报告、厂址选择报告，环境影响评价报告及其批文，运输、水

源、电源的协议及批文等）；

（4）设计基础资料（自然条件、岩土工程初勘报告，地震基本烈度、水文资料和当地环保、安全卫生、消防标准和规定等）；  
（5）工程设计中采用的新工艺、新技术、新材料的试验报告或试验鉴定报告；  
（6）当地的概算指标；  
（7）有关会议纪要或批文等。

# 3.1.2 工程设计项目概况

（1）项目的性质和特点（说明工程设计项目属于新建、扩建、改建或分期建设以及属于国内联营或中外合资或其它特殊要求等）；  
（2）场址概况（场址地形、地貌、地理环境、水、电、汽的来源及可靠性，货物流向、运输方式、扩建预留地等）；  
（3）主要工艺技术要求，工艺方案、项目组成、规模以及对配套系统工程和生活辅助设施的要求；  
（4）协作条件或依托条件（公用工程和生活设施、机修和仪修等）；  
（5）本工程节能要求；  
（6）工程建设的总投资和总进度的控制要求，设计的形象进度。

# 3.1.3设计范围及设计分工

（1）列出项目组成（包括主体部分、辅助部分、生产福利设施部分）；  
（2）内、外部设计分工。

# 3.1.4 文件编制统一规定

（1）工程设计文件格式（工程项目名称，工程代号，主项号，项目编码（总承包工程），册、章的编排）；  
（2）初步设计的深度要求，设计文件密级划分的原则意见；  
（3）对各专业的设计技术统一规定的要求。

# 3.1.5 存在的问题及解决问题的建议措施。

3.1.6质量控制程序和要求见施工图《设计开工报告》中第 3.2.3 条。  
3.2 施工图《设计开工报告》  
3.2.1在初步设计《设计开工报告》的基础上，必要时需作修改或补

# 充，内容如下：

（1） 初步设计文件及其审查纪要和批复文件；  
（2） 引进技术的基础设计文件；  
（3） 有关本工程技术讨论的备忘录、会议纪要；  
（4） 设计合同；  
（5） 岩土工程详勘报告，设计范围的总图及其地形图，顾客提供的基础数据资料；  
（6） 主要设备订货情况；  
（7） 必要的内外部设计条件提供日期；  
（8） 投资控制指标：说明总投资及专业投资控制指标，提出对编制施工图设计预算或施工图修正概算的要求；  
（9） 设计进度：根据顾客对设计进度的要求，提出各专业互提委托资料进度的初步意见和设计成品文件交付的最终期限；  
（10）存在问题及解决问题的建议措施。  
3.2.2 对于只做施工图设计的项目，参照 3.1 和 3.2.1 的内容编制。  
3.2.3质量控制要求  
（1）设计评审执行《能源工程技术有限公司设计评审管理规定》，并列出评审大致时间，必要时列出应评审的设计技术方案，对于技术成熟、方案简单的项目，将评审与验证同步进行；  
（2）设计验证执行《能源工程技术有限公司设计验证规定》；当项目需采取特殊方法进行验证时，应加以说明；  
（3）确定质量目标；  
（4）确认的安排。

3.2.4当有分包方时，应明确双方的联络人、设计技术和质量问题的处理权限以及对供方的特殊要求等。  
3.2.5《设计开工报告》中应明确特殊的技术要求。

3.3 开工报告的批准

项目的开工报告，在专业负责人和综合赋能服务群的配合下，由设计经理负责编制，主专业审核人员审核，经综合项目群\常规设计

群召集人批准。当设计经理与审核人员为同一人时，审核人处由主专业带头人或主专业专家团队人员签字。对于总承包综合工程需报项目经理批准。

3.4 开工报告的发布  
3.4.1 开工报告由设计经理在开工会议上发布。  
3.4.2《设计开工报告》由设计经理报送总工程师、相关专业带头人。有供方时，应发至供方。

# 4 附则

4.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
4.2 本制度自颁发之日起试行，试行期为 6 个月。6 个月内若无修订，则自动转入实施。
