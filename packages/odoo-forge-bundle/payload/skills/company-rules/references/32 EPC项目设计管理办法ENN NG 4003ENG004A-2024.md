# 新地能源工程技术有限公司设计分公司EPC 项目设计管理办法

# 第一章 总 则

# 第一条 目的

设计管理是总承包项目管理的重要组成部分。设计文件是项目管理中采购、施工和试运行各阶段开展工作的主要依据，为有效提高设计工作在EPC项目中的执行效率，满足用户需求，实现项目创值增效，特制定本办法。

# 第二条 适用范围

本办法适用于新地能源工程技术有限公司设计分公司（以下简称“燃气设计中心”）参与的 EPC 项目。

# 第二章 分 则

# 第三条设计管理职责

# （一）设计经理

设计经理在项目经理领导下，负责组织协调项目的设计工作，全面负责设计的进度、质量、成本和协调等工作。其主要职责和任务如下：

1.研究、熟悉合同文件，确定设计工作范围，进行设计工作分解，提出设计工作分解清单。  
2.组织项目设计人员收集项目设计基础资料，落实设计条件。  
3.组织审查工程设计所必需的输入条件和设计基础资料，主要包括：设计依据、业主提供的设计基础资料和有关协作协议的文件。  
4.根据EPC项目总进度计划编制设计进度计划。  
5.参加编制项目设计开工报告和工程设计统一规定，各专业负责人配合。  
6.组织各专业对本项目进行方案编制，并积极开展优化设计工作，

组织设计方案评审，并协同采购、造价等人员进行设计优化。

7.组织各专业开展勘察、设计、会审、输出评审、文件发放、图纸审查等工作。  
8.设计结束后，组织各专业对本项目的设计基础资料、项目批复、往来函件、设计变更等资料进行整理和归档。  
9.组织设计人员参与施工图会审和设计交底，对重大设计变更组织设计修改。  
10.根据合同条款，配合 EPC 项目部开展施工期间的设计服务工作。  
11.工程结束后，组织各专业开展项目设计总结及项目设计复盘。  
12.如果设计项目需要进行分包时，协助项目经理对分包单位资质、能力等进行审查，对分包任务进行分包，组织各专业对分包界面、专业分工等进行确认。

# （二）设计项目组

1.设计经理由燃气设计中心根据工程规模、工程类型进行任命，并向EPC项目经理部报备。设计经理须具备所担任工程类型、规模的设计经理任职资格，合同中要求具备注册师执业资格的，要按注册师的执业规定执行。  
2.设计项目组由设计经理根据项目工程规模、工程类别牵头组建，各专业群召集人协助设计经理进行项目人员安排及分工。  
3.各专业负责人及设、校、审人员应由具备该类型工程设计资格的人员担任。专业负责人在设计文件上进行签署时，除了符合《能源工程技术有限公司工程设计文件签署规定》的要求，同时还应符合国家、行业及业务所在地相关部门的有关规定，实行注册师制度的专业，并按注册师的执业规定执行。  
4.设计项目组人员应认真研究和熟悉 EPC 合同的有关的内容，编写设计工作大纲。  
5.设计项目组应认真执行总承包设计管理办法，按照国家颁布的法律、法规、标准、规范以及公司的质量手册及相关制度进行设

计文件的设计、校核、审核、审定、会审等工作。

6.设计项目组负责项目设计内容的实施，解决项目实施中的设计质量和技术问题，对所设计图纸内容负责。  
7.设计项目组应根据 EPC 合同约定及项目整体进度计划要求制定设计项目进度计划，并按时完成相关设计工作。  
8.设计项目组应根据项目整体进度计划，配合施工进度，提供准确、完善的阶段性施工图纸和现场技术服务工作。

# 第三条设计过程管理

# （一）熟悉合同并参与合同交底

# 1.研究、熟悉合同文件

在项目初始阶段，设计项目组人员应认真研究和熟悉合同文件中与设计工作有关的内容，如：工程承包范围、设计工作任务、工程项目的基础资料和设计参数、采用的标准规范、工程进度、考核验收以及违约责任等等。

# 2.参与市场中心（或市场开发人）合同交底

项目开工前，设计项目组人员应参与由市场中心（或市场开发人）组织的合同交底，了解投标合同背景、范围、目标、执行要点及特殊情况处理（包括建设单位相关人员及背景），特别是工程的报价策略及单价确定方式，从投标、合同的角度分析项目设计、采购、施工过程中影响成本的控制节点，对不清楚、不明确的问题要与市场开发人员及投标人员充分沟通，并形成合同交底记录。

《合同交底记录》模板详见附件1。

# 3.专业内部投标交底

投标文件编制人员与设计人员不一致时，专业内部需进行投标文件交底，投标人员需向设计人员详细介绍投标方案、设计内容、设计记录、工程量编制、注意事项等，并填写投标人员与设计人员交接单。

《中标项目专业内投标人员与设计人员交接单》模板详见附件 2。

# （二）建立项目设计协调沟通程序

项目设计协调沟通程序由设计经理与项目经理部共同确定，指在合同文件的基础上进一步明确沟通协调机制，建立设计沟通、联络、报告制度，作为项目设计协调沟通依据。

设计经理和设计项目组成员应在项目经理部授权范围内参与设计相关内容的沟通联系，内容包括确定设计范围、组织评审设计方案，与对接人沟通、项目基础资料收集、设计方案确定、工程设计规定编制、设计文件评审、设计文件发放等。沟通记录文件由 EPC 项目经理部统一管理、统一接口、统一发放。

# （三）设计进度管理

1.设计项目组应根据合同约定及项目经理部要求编制有前置条件或绝对工期的项目设计进度计划，报 EPC 项目经理部批准后执行。  
2.设计工作需按照经批准后的计划开展，因设计原因影响设计进度的，按合同规定进行处罚；因业主原因、采购原因影响设计进度的，设计项目组可根据实际情况修正进度计划，并将修正后的进度计划重新提交项目经理部报备。  
3.设计进度计划应结合施工进度合理编制，对于已经开工项目，应按实际施工需要加大投入、加快进度，可按装置、单元、专业分批次出图，以满足项目整体进度要求。

# （四）设计质量管理

1.设计过程中应严格执行国家颁布的强制性标准、规范以及公司的质量手册、质量相关制度等规定。  
2.建立自查自纠制度，对设计文件和图纸中的错误或遗漏及时进行修改或补充，并承担由此造成的相关责任。  
3.对项目整体投资影响较大的设计方案应经项目经理部预算审核通过后再进行详细设计；对于设计方案的关键要点，明确提出工艺、质量、合格标准等要求。  
4.设计不得以任何方式明示、暗示、指定工程材料、设备的生产厂或供应商。

5.设计应积极利用新技术、新材料等科研成果，以及设计审查和咨询意见进行设计优化，最大限度地提高设计质量。  
6.设计项目组对其设计成果的质量负责，应严格执行《建设工程质量管理条例》之相关规定。  
7.对于EPC项目中有异议的技术问题或设计与业主、施工之间有不同意见，通过EPC项目技术评审会解决，EPC项目部无法解决的问题，以项目部名义及时报公司相关部门解决处理。

《EPC项目技术问题评审审批单》模板详见附件3。

# （五）设计文件管理

1.设计文件和施工图纸完成后应按照《设计分公司技术档案管理办法》及时进行归档，归档内容包括但不限于设计产品、项目实施中目会议纪要、审查意见、评审记录等。  
2.按分包工程招标、设备采购、物料准备等要求，及时提供相关设计文件、图纸和清单。  
3.按合同约定数量及时交付设计文件和施工图纸，同时按合同规定格式和时间提交设计文件电子文档。提供的设计文件电子文档，要与纸质文件内容一致。  
4.设计图纸上必须明确版次号及版次说明，以保证设计文件的有效性和真实性。当修改设计图纸时，在标题栏上部增加版次说明栏、更改版次号，并修改图纸目录，同时将修改情况以《设计补充、变更通知单》形式与修改图纸一同发放。

# （六）设计文件审查

1.项目各阶段设计文件完成后，应尽快做好报审准备工作，配合业主按照相关规定报主管部门审查、批准。  
2.设计文件不需要政府有关部门审查或批准的，应当经业主审查同意后提交施工。  
3.设计文件需政府有关部门审查批准的，应在审查同意修改完成向业主提交设计文件再组织施工。

4.对于相关部门的审查意见，设计项目组要严格落实修改工作，说明并报送审查部门，经同意后可不做修改。

# （七）设计成本控制

1.设计项目组应根据项目合同签订情况及项目运营策略，进行设计成本控制。设计成本控制应重点加强在设计人工时、设计质量、设计方案优化比选、材料种类/材料等级、可施工性审查、与其他专业接口管理等方面的控制。  
2.在保证设计方案合规合法、技术可行、经济合理、满足用户要求的前提下，设计项目组应积极倡导设计人员开展施工图优化设计工作，选择或推荐适宜的工艺、设备、材料，严格控制项目预算成本，避免过度设计，造成浪费。若项目预算超出项目目标成本，须报项目经理部批准后实施。  
3.需要进行限额设计的项目，在保证设计质量的前提下，对设计方案进行限额设计，设计方案的实施费用原则上按照项目成本目标对应子项的费用由造价工程师进行控制，确保工程预算不突破限额目标，超过限额目标的，须经项目经理部批准后实施。  
4.设计文件出件前，各专业须与投标文件材料进行比对，涉及材料变化、影响成本事宜需书面与项目经理部沟通，征得同意后方可出版。

# 第四条设计后续管理

设计产品交付后，设计项目组需按照《能源工程技术有限公司设计服务管理规定》开展设计服务工作。

# （一）设计变更管理

1.EPC项目设计变更应以会议纪要、变更审批单、设计委托书等书面形式作为变更依据。《变更审批单》模板详见附件 4。  
2.设计变更原因要具体明确，不得含糊其词，变更的文字内容及附图应清楚。  
3．设计变更必须遵守国家技术标准、设计规范、《建造智能业务

群项目管理指导手册》、《能源技术工程 HSE 变更管理规定》等相关规定，以保证设计文件的合规性和完整性，在确保工程质量和技术标准的前提下，对降低工程造价，加快施工进度，有利于工程管理等方面有明显效果时，可考虑对施工图设计进行优化和调整。

4.设计文件中的差、错、碰、漏的修正和对设计图纸复查完善的变更应以《设计补充、变更通知单》和变更图纸形式报相关方，并及时更换现场图纸。  
5．有设计代表在现场时，设计变更通知单中设计代表必须会签。  
6.设计变更图纸按燃气设计中心出版程序执行。

# （二）设计代表的派遣及要求

施工期的现场技术服务工作根据合同规定派遣相应的设计代表。设计代表主要为参加过本项目的设计人员，要求对项目具体设计情况熟悉，坚持原则，责任心强，具有实践经验、能够独立处理现场技术问题的能力、同时具有一定协调能力。现场技术服务期间，设计代表应遵守施工项目部相关制度管理规定，去施工现场时须做好安全防护措施。

# （三）设计代表责任与权利

# 1.设计代表的职责

设计代表按本办法做好现场技术服务工作，除应做好本专业设计代表工作外，还应协调各专业之间的关系，代表燃气设计中心参加现场各种会议，协调设计单位与各参建单位的关系。

（1）严格执行国家相关法律法规，贯彻落实技术标准、规范；  
（2）严格执行已经审批的设计文件，熟悉设计内容，充分了解各专业的设计原则、设计方案、设计意图和施工组织方案。  
（3）参与技术交底，向业主、监理、施工单位阐明设计意图；  
（4）对施工的设计图纸负责事先检查，及时将文件中存在的差、错、漏、缺及不合理的设计问题，协同各专业修正完善。  
（5）协调处理业主、监理、施工单位提出设计变更的要求，配合业主、监理、施工单位解决施工中出现的技术问题，提出安全、

经济、合理、易于实施的技术方案。

（6）应严格执行有关纪律，廉洁自律，提高工作效率，除现场问题应及时答复外，答复。  
（7）负责现场图纸有效性控制；  
（8）负责搜集和反馈现场的施工进度与质量信息；  
（9）作好设计代表工作日志和各项纪录（文字、图片、视频等形式）；  
（10）设计代表在现场技术服务结束返回公司后应编写现场技术服务工作总结，并交给设计经理汇总。

# 2.设计代表权利

（1）接收不改变原批准的设计原则和工程质量特性的设计变更申请。  
（2）纠正设计文件中的错误、缺陷、漏项等设计问题。  
（3）代表燃气设计中心处理有关设计的具体专业技术问题，  
（4）对涉及技术标准、工程参数、投资额的重大设计变更及时上报设计经理、燃气设计中心业务主管部门。  
（5）有权拒绝不合理、不正当的变更设计要求，有权拒绝签认未履行正常手续的变更设计。  
（6）不得擅自将未经签署、盖章的设计图纸交付监理、施工单位使用。

# （四）现场技术服务管理

# 1.准备工作

设计代表进入工地前，应熟悉设计文件，汇集设计遗留问题，以便现场沟通解决。

# 2.设计答疑

对于监理、施工单位就设计图纸提出的质疑，应积极答疑和及时处理。

# 3.图纸检查

在分项工程开始前，设计代表应对本专业和有关联的其他专业图纸进行一次检查，将设计差、错、漏、碰问题消化在施工前。

# 4.现场问题反馈

施工过程中对于施工现场发现的设计问题或提出的改进、优化建议，应以《施工现场问题反馈单》的形式反馈给设计项目组，设计项目组接到反馈单后应认真研究，对同意的问题，履行设计变更手续。对不同意的问题，应积极与相关方沟通，如意见分歧较大时，可报项目经理部进行磋商解决。

《施工现场问题反馈单》模板详见附件5.

# 5.施工现场巡视、检查

设计代表应经常到施工现场了解情况，配合施工，解决设计问题。对重要隐蔽工程及技术复杂对质量要求较高的分项工程，设计代表应配合施工单位加强质量监督，有必要时可进行影像留痕。

# 6.质量信息的收集和反馈

质量问题信息收集是设计代表的一项重要工作，设计代表在现场应积极搜集与设计有关的质量问题信息，对其中技术性差错或带普遍性的质量问题，要及时书面上报相应业务群，对套用的标准图及通用图纸，如发现错误，亦应做好记录，及时向相应业务群汇报，以便及时修正。

# 7.设计质量事故处理

凡在工程建设或试运行中发生质量事故或事故尚未弄清楚是否与设计有关，设计代表均应于 24 小时内以电话、邮件、传真等形式向燃气设计中心相应业务群报告，设计经理应及时组织有关人员进行研究分析，制定事故处理应对方案。设计代表应积极协助公司搜集现场资料，调查事故发生过程、分析原因，积极配合业主单位采取善后措施，尽快恢复正常施工程序。

# 8.设计代表日志编写

设计代表每天要做好设计日志编写工作，并存档备查。主要包括但不限于：

（1）每天在现场的技术服务工作内容；  
（2）现场召开的各种协调会议；  
（3）现场重大技术研讨会；  
（4）设计变更处理情况；  
（5）发现的设计质量问题及处理情况；  
（6）其他与项目相关的事宜。

# （五）现场服务人员管理

1.设计代表人员不宜变动频繁轮换，因故变动或替换，应与继任者做好工作交接。  
2.设计代表进驻和撤离现场的时间安排由设计经理与项目经理部协商决定。未经同意擅自离开工地者，视情节轻重予以惩处。

# （六）设计回访

1.项目竣工后，燃气设计中心相应业务群应安排相关人员选择适当时机进行回访，听取业主、监理及施工单位对设计质量的意见，总结经验教训,不断提高公司设计技术水平。  
2.回访后半个月内,由回访组长（或主要回访人员）编写设计回访报告。对于重大技术问题（例如与设计有关的生产事故）等还应编写专题报告。  
3.回访报告的编制应符合公司《设计回访报告》的相关要求。

# 第三章 附 则

1.本办法由燃气设计中心燃气业务群负责解释、修订。  
2.本办法自发布之日起实施。
