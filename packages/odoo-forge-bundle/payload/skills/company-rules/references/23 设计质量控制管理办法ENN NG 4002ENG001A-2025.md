# 新地能源工程技术有限公司石家庄能源化工技术分公司设计质量控制管理办法

# 第一章 总则

# 第一条 目的

为了加大设计质量的监管力度，确保工程项目设计质量的有效管理，降低设计质量发生概率，防止不合格产品的发生，提升企业品牌形象，特制定本办法。

# 第二条 适用范围

本管理办法适用于燃气化工设计中心设计质量控制的管理。

# 第二章 分则

# 第三条 组织与职责

（一）各级设计人员负责设计产品设计过程的质量管控，并对设计文件的质量负责。  
（二）设计经理组织各级设计人员根据设计过程管理相关制度及其他相关要求对项目实施管控，对项目总体质量负责。  
（三）各业务/专业组织负责组织相关设计产品的质量检查，收集质量问题、获取质量信息、管控设计产品质量。  
（四）技术质量智能能力群负责组织分公司级的设计产品质量检查，收集质量问题、获取质量信息、统计产品质量目标数据、监测设计产品质量。  
（五）技术委员会、业务/专业组织负责人、专业总工程师及由业务/专业组织负责人指定的具备审核、审定设计资格的人员按要求开展质量检查、评审、评定等质量管控工作，并对检查、验证等结果负责。  
（六）技术质量智能能力群负责人、总工程师、一号位（总经理）依次负责质量控制奖惩结果的审核、审批和批准。

第四条 设计质量控制主要包括成品质量检查和设计过程质量管控。

第五条 成品质量检查包括：季度检查、专项检查、归档检查。

（一）季度检查依据《新地能源工程技术有限公司设计成品质量评定规定》进行。  
（二）专项检查为不定期检查，由技术质量智能能力群结合实际情况及上级要求，制定专项检查工作方案，有针对性的组织实施。  
（三）归档检查由技术档案管理人员依据《新地能源工程技术有限公司石家庄能源化工技术分公司技术档案管理办法》进行。

第六条 设计过程质量管控分为产品交付前和产品交付后的质量管控。设计过程质量管控除应满足《质量手册》和《燃气化工设计中心质量分手册》中对设计策划、设计输入、设计控制、设计输出、设计更改等过程的管控要求，还应执行以下要求：

（一）产品交付前的质量管控是指项目策划、组织接口、输入资料、评审验证等过程的质量控制。各管控节点涉及人员依据下列相关规定开展工作，形成过程记录并随设计文件归档留存，确保记录的可追溯性。

1.开工报告的编制执行《新地能源工程技术有限公司设计开工报告编制规定》；  
2.接口管理执行《新地能源工程技术有限公司项目设计组织和技术接口规定》；  
3.专业之间发送、接收条件管理执行《新地能源工程技术有限公司项目设计条件管理规定》；  
4.设计文件验证方法及要求执行《新地能源工程技术有限公司设计验证规定》；  
5.设计文件签署要求执行《新地能源工程技术有限公司设计文件签署规定》；  
6.设计文件格式执行《新地能源工程技术有限公司设计文件格

# 式规定》；

7.设计文件的编号执行《新地能源工程技术有限公司设计文件及技术文件编号规定》；  
8.设计文件放行前的质量评定和输出评审分别执行《新地能源工程技术有限公司设计成品质量评定规定》和《新地能源工程技术有限公司设计评审管理规定》；  
9.压力管道设计文件执行《压力管道设计质量保证手册》、《压力管道设计管理制度》等压力管道设计质量保证体系的相关文件。  
10.压力容器设计文件执行《压力容器设计质量保证手册》、《压力容器设计程序文件》等压力容器设计质量保证体系的相关文件。

# （二）产品交付后的质量管控是指产品交付后的外部审查、设计变更、设计服务等过程的质量控制。

# 1.外部审查

（1）设计经理负责建设单位、外部审图机构的沟通协调，及外部审查结果信息的跟踪和反馈。  
（2）各专业人员接到审查意见后，积极查找原因，找出解决办法；在与外部审图人员沟通，达成一致意见后，进行回复或变更，并反馈设计经理。  
（3）外部审查意见回复和澄清应在项目组内进行验证和确认，必要时，还应在相关专业或分公司内进行评审。当审查意见中出现有违反规范强制性条文时，设计经理应及时通告技术质量智能能力群和专业理正。设计经理组织相关人员与外部审图机构及审查人员沟通，查找原因，寻求解决办法，必要时，提请分公司级评审。  
（4）外部审查意见、回复、变更图及审图合格证明等资料应作为项目级的质量信息进行管理，并及时归档。归档应满足《新地能

源工程技术有限公司石家庄能源化工技术分公司技术档案管理办法》要求，并明确各专业分别为第几次审查后达到外部审查合格。技术质量智能能力群技术档案管理人员对归档文件中的审查数据进行统计。

# 2.设计变更

（1）设计变更按变更原因分为设计原因和非设计原因。设计原因分为设计不合理、设计不合规、设计遗漏、计算错误、各专业配合失误等；非设计原因分为建设方要求、采购原因、施工原因等。  
（2）设计变更按严重程度分为一般设计变更和重大设计变更，具体划分详见《燃气化工设计中心质量分手册》中“设计更改”。一般设计变更由设计人员或现场设计代表组织实施，涉及到多专业变更时，须经相关专业设计人员会签。重大设计变更由设计经理提请分公司级评审，并按评审结论组织实施。  
（3）设计变更的实施应满足《燃气化工设计中心质量分手册》中“设计更改”、《新地能源工程技术有限公司设计文件及技术文件编号规定》、《新地能源工程技术有限公司设计文件签署规定》等相关制度的要求，还应满足《建造智能业务群工程建设项目设计变更管理办法》的相关要求。  
（4）各业务/专业组织应将设计变更信息作为本组织的质量信息进行收集，并对本组织的设计变更信息进行统计，明确设计变更的原因及问题属性等内容。  
(5)设计变更优先采用网络设计平台进行发起及获取相关编号，变更文件随出版文件进行归档。

# 3.设计服务

（1）设计服务为设计产品交付后对顾客的服务，主要包括设计交底、现场技术服务、消防及竣工验收、三查四定和设计回访等，执行《新地能源工程技术有限公司设计服务管理规定》。通过设计服务获取的质量信息应作为项目级质量信息进行存档和管理，并在

相关组织内进行统计。

（2）项目现场派发设计服务代表时，驻场服务代表填写服务日志，每月归档一次。如果多名代表轮换服务，服务日志应接转使用。现场服务结束后，现场服务代表一周内完成服务总结并归档。设计经理收集各专业设计服务总结形成报告并进行技术归档，组织完成相关问题的培训宣贯。

# 第七条 质量问题的评审和反馈

（一）项目组专业内无法解决的质量问题由专业负责人组织专业级评审；涉及多专业的质量问题由设计经理组织项目级或分公司级评审。设计评审相关要求执行《新地能源工程技术有限公司设计评审管理规定》。  
（二）当设计原因引起的重大设计变更、审图意见违反强制性条款及服务、设计服务等过程中直接或间接造成的质量问题时，设计经理或相关组织应及时向与质量问题相关的组织进行反馈，如发现质量问题可能造成公司或相关方的信誉或经济受损时，设计经理或相关组织还应及时报送技术质量智能能力群。  
（三）档案、晒图等非专业人员发现图纸存在低级错误时，依据错误内容及时反馈相关设计人员或设计经理。

# 第八条 质量信息的收集与应用

（一）技术质量智能能力群负责组织分公司级设计产品质量检查信息的收集、质量情况的分析及宣贯，必要时制定整改或改进方案并组织实施。  
（二）各业务/专业组织负责本组织质量检查、设计变更等质量信息的收集、分析及宣贯，必要时制定整改或改进方案并组织实施。

# 第三章 奖惩

# 第九条 处罚

（一）根据成品质量检查结果统计情况，对评定成绩为不合格

的设计文件进行处罚，扣除此设计文件相应绩效奖励，并给予相关人员 100～500 元的处罚，当出现较大技术问题或特殊事项时，可适当增加处罚额度。

（二）当外部审查意见中出现违反规范强制性条文问题时，给予相关人员 100～300 元/条的处罚，如因外部审查事宜给公司造成信誉或经济损失时，给予相关人员不低于 500 元的处罚。  
（三）设计原因造成的质量事故、不良质量风险，或设计质量原因造成的客户投诉、不满意等，应根据具体情况进行分析，并按公司相关事故、投诉及质量红黄线等管理规定要求制定处罚方案。

# 第十条 奖励

对设计过程质量管控有突出贡献或表现突出者，给予 500 元奖励。

每半年度对质量检查工作中表现突出、责任心强、问题专业、评审力度强、敢于暴露问题，并有效提升设计质量和设计水平的评审人员，给予 500 元奖励。

全年成品质量检查中，未出现合格及以下评定成绩、未被通报批评且评定优良率最高的设计文件，结合现场施工反馈的问题情况，按排名择优奖励，每人 1000 元。

# 第四章 附则

第十一条 本办法由技术质量智能能力群负责解释、修订。

第十二条 本办法自发布之日起执行。
