# 新地能源工程技术有限公司质量事故管理规定

# 第一章 总则

# 第一条 目的

为规范和加强新地能源工程技术有限公司（以下简称“公司”）质量事故的上报、调查、处理等工作，有效消除各类质量事故带来的不利影响，教育并警示他人，结合公司业务实际，特制定本规定。

# 第二条 管理原则

# （一）四不放过

事故原因未查清不放过、责任人员未处理不放过、整改措施未落实不放过、有关人员未受到教育不放过。

# （二）依法依规

事故调查处理必须严格遵守国家法律法规和相关规定，确保调查处理的合法性和合规性。

# （三）实事求是、尊重科学

事故处理必须以事实为依据，辅以科学的检测手段或方法，查明事故发生的内在原因和机理，发现事故发生的规律，制定预防事故重复发生的有效措施。

# （四）公开公正

事故调查及处理必须公正无私，既不包庇事故责任人，也不借机对事故责任人打击报复。事故处理结果要在一定范围内公开，接受监督，提高事故处理的透明度和公信力。

# 第三条 术语定义

# （一） 质量事故

由于工程服务（或产品）不符合法律法规、强制性标准或合同约定的质量要求，影响工程或产品功能正常使用或安全运行，并造

成经济损失、人员伤亡、社会影响等不良后果的事件，统称为质量事故。

现场发生人员伤亡的质量事故，事故处理按照 HSE 事故相关制度执行，事故调查与HSE管理部门联合开展。

# （二） 质量事件

在工程服务（或产品）建设或使用过程中出现的未达到事故等级的质量问题或缺陷，如不处理可能对工程（或产品）使用寿命、生产效率、系统运行等造成影响的，统称为质量事件。

# （三） 事故分级

# 第四条 组织职责

# （一）公司技术质量智能能力群

公司技术质量智能能力群是质量事故管理的归口部门，主要负责：

1.质量事故管理规定的编制、修订；  
2.按要求及时上报质量事故；  
3.统筹质量事故调查处理工作，组织一级（重大）、二级（较重大）、三级（较大）质量事故的内部调查处理，组织区域分公司四级（一般）质量事故的内部调查处理；  
4.监督指导各组织、事故单位对质量事故的规范管理；  
5.建立质量事故台账。

# （二）专业中心、专业分公司、区域分公司

根据职责分工负责组织、参与、协调或监督相关质量事故管理工作。主要负责：

1.专业中心、专业分公司负责本组织四级（一般）、五级（轻微）事故及六级（质量事件）的内部调查处理；  
2.区域分公司负责本组织五级（轻微）事故及六级（质量事件）的内部调查处理；  
3.协同、参与其他质量事故的内部调查处理；  
4.建立本组织质量事故台账并上报。

# （三）事故单位

作为责任主体，主要负责：

1.按要求及时上报质量事故，组织抢救、防止事故扩大、减少事故损失、保护事故现场和有关证据；  
2.全力配合事故调查组开展事故调查，或依据授权进行事故调查处理；  
3.与相关方的沟通协调和处理方案的现场实施；  
4.与事故相关承包商的协调、谈判和索赔；  
5.编制事故案例并上报。

# （四）质量事故调查组

作为质量事故调查责任主体，主要负责：

1.核实事故基本情况，包括事故发生的经过及直接经济损失；  
2.核查事故项目基本情况，包括项目履行法定建设程序情况、工程各参建单位履行职责的情况；  
3.依据国家有关法律法规和工程建设标准分析事故的直接原因和间接原因，必要时组织对事故项目进行检测鉴定和专家技术论证；  
4.认定事故的性质和事故责任；  
5.提出对事故责任者的处理建议；  
6.总结事故教训，提出防范和整改措施；  
7.提交事故调查报告。

# 第五条 适用范围

本规定适用于公司及下属各单位，承包商发生的质量事故也须按照本规定执行。

# 第二章 分则

# 第六条 事故报告

（一）事故发生后，事故现场项目负责人应当立即向本组织质量部门负责人电话或短消息报告，由各级质量部门负责人立即向本级组织一号位和上级部门报告，每级上报时间不超过2小时。

（二）事故单位应及时上报质量事故快报(见附件1)，并在24小时内逐级上报至公司一号位，同时向公司财运智能能力群做“工程一切险”的理赔报备。  
（三）在事故发生后超过24小时上报的算“迟报”，超过48小时的算“瞒报”，未按事故真实情况如实上报的按“谎报”处理。  
（四）质量事故涉及两家及两家以上单位的，各单位应分别上报。  
（五）事故报告后出现了新情况或新进展，应当及时补充报告。

# 第七条 事故调查

（一）事故发生后，按事故相应级别由责任组织成立质量事故调查组，调查组成员由相应专业、并与所调查的事故没有直接利害关系的内、外部专家组成。  
（二）调查组有权向事故发生单位及有关人员了解事故相关情况和索取有关资料，任何单位和个人不得推诿、拒绝或隐瞒。凡涉及公司商业秘密的，经判定后按照公司相关保密规定执行。  
（三）事故调查组成员在事故调查过程中应当诚信公正、恪尽职守，遵守事故调查组的纪律，保守事故调查的秘密。未经事故调查组组长允许，事故调查组成员不得擅自发布或透露有关事故的信息。  
（四）事故调查结束应形成《质量事故调查报告》（见附件2），事故调查组全体成员应在报告上签名确认，对调查报告真实性负责，报告应当附具相关证据材料。质量事故事件调查报告由事故调查责任组织批准后上报，上报时限要求如下：

1.一级（重大）、二级（较重大）、三级（较大）事故，在事故发生后30日内向公司技术质量智能能力群提交报告；  
2.四级（一般）、五级（轻微）事故，在事故发生后15日内向公司技术质量智能能力群提交报告；

3.对质量事件，责任单位在事件发生后7日内按规定存档；  
4.如遇特殊情况，经调查组组长批准可适当延长上报时限。

# 第八条 责任追究

（一）按照“尽职免责、失职追责”的原则，对事故相关责任人的失职行为进行严肃问责。当质量事故性质难以界定时，处罚按“就高不就低”原则处理。

# 第九条 事故处置及防范措施落实

（一）按事故调查报告整改防范措施对事故问题进行闭环处理，处理结果需要通过检查、验收和必要的检测鉴定，并有明确的质量事故处理鉴定意见后，报请工程监理或建设单位批准恢复正常施工或使用。  
（二）所有事故在处理决定下发后一个月内必须编制事故警示案例，组织召开事故检讨反思会，通报事故发生原因、经过、处置情况及有关责任人员处理情况，检讨反思不足，并做好记录备查。  
（三）事故发生单位及其他相关单位应当认真吸取事故教训，举一反三进行排查整改、落实防范措施，避免类似事故再次发生。

# 第十条 事故归档

事故快报、事故调查报告、处理决定、事故处置等相关事故资料由事故责任组织在事故处置完毕后7日内整理归档并永久保存，录入质量事故事件台账（见附件3），并即时上报至公司技术质量智能能力群备案。

# 第三章 罚则

第十一条 对发生二级（较重大）以上质量责任事故的单位，除对相关责任人追究责任外，取消该单位及主要责任人当年度所有评优评先资格，并在组织创值评估结果中进行扣分处理。

第十二条 对有下列行为的，视情节轻重，给予相关责任人行政降级、撤职或解除劳动关系处分，扣发6-12个月月度基线业绩分享。

1.隐瞒事故不报、谎报或故意迟报的；

2.事故发生后，不负责任、不积极组织抢救或抢救不力，造成更大损失的；  
3.无正当理由拒绝事故调查，毁灭、伪造证据，破坏、伪造现场，干扰调查工作或嫁祸于人的；  
4.在调查、处理事故过程中玩忽职守，徇私舞弊，擅自处理或包庇事故责任者或打击报复的；  
5.对事故（隐患）报告人、调查人和处理人进行打击、报复的；  
6.其它影响严重、性质恶劣的行为或情形。

# 第四章 附则

第十三条 本规定由公司技术质量智能能力群负责解释、修订。

第十四条 本规定自正式发布之日起，试行 6 个月；6 个月内若无修订，则自动转入实施。

第十五条 本规定中其他未尽事宜按国家相关规定执行。
