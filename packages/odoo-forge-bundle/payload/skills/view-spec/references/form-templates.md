# Form 视图规格模板

## 选择骨架

| 场景 | 骨架 | 判断依据 |
|------|------|----------|
| 字段少（5-8个），无明细行 | 简单 Form | 无 One2many、无 tab 需求 |
| 有明细行或多个信息分区 | 标准 Form | 有 notebook、有 One2many 明细 |
| 有 button_box、oe_title、多 tab、chatter | 完整 Form | 主要业务对象的核心表单 |

---

## 骨架一：简单 Form

适用于配置项、简单记录、弹窗表单。

```
┌── header ────────────────────────────────────────┐
│ [操作按钮]                   ○ 状态1 → ○ → ●   │
├── sheet ─────────────────────────────────────────┤
│                                                   │
│  ┌── group ────────┐  ┌── group ────────┐        │
│  │ 字段A  _______  │  │ 字段D  _______  │        │
│  │ 字段B  _______  │  │ 字段E  _______  │        │
│  │ 字段C  _______  │  │ 字段F  _______  │        │
│  └─────────────────┘  └─────────────────┘        │
│                                                   │
└──────────────────────────────────────────────────┘
```

**特征：** header + sheet(group×2)，无 notebook，无 chatter。没有工作流时可省略 header。

## 配套说明格式

线框图之后，必须附上以下说明表。

### 字段说明表

```markdown
**字段说明**

| 字段 | Widget | 备注 |
|------|--------|------|
| 字段标签 | widget名 | 默认值 / 可编辑条件 / 计算说明 / 关联过滤条件 |
```

**Widget 写法参考 `references/layout-rules.md` 第 6 节。** 常用的：

| 情形 | Widget |
|------|--------|
| 普通文本 | `char` |
| 关联对象（可搜索下拉）| `many2one` |
| 关联对象+头像 | `many2one_avatar` |
| 日期选择器 | `date` / `datetime` |
| 金额 | `monetary` |
| 多行文本 | `html`（富文本）/ `text`（纯文本）|
| 只读自动计算 | `monetary` + 备注"只读，自动计算" |
| 拖拽排序 | `handle`，备注"最左列" |

### 操作按钮表

```markdown
**操作按钮**

| 按钮 | 显示条件 | 行为 |
|------|----------|------|
| [按钮名] | 状态=xxx / 角色=xxx | 状态变为 xxx，发通知给 xxx |
```

### 状态栏说明

```markdown
**状态栏**：statusbar 显示 `状态1 → 状态2 → 状态3`。[异常状态名] 不显示在状态栏（异常态）。
```

## 字段分配原则

**左栏 group（业务主体信息）：**
- 关联对象（客户、供应商、项目、员工）
- 分类字段（类型、部门、类别）
- 核心业务字段

**右栏 group（辅助信息）：**
- 日期字段（创建日期、截止日期、审批日期）
- 金额字段（总额、预算）
- 负责人、审批人
- 自动编号字段

**Tab 分配：**
- Tab 1：One2many 明细行（用户最常看）
- Tab 2：次要字段、扩展信息（标签"其他信息"）
- Tab 3：纯文本备注（标签"备注"）
- Tab 4+：审批记录、变更历史等

## 填充示例

需求："合同管理表单，有合同基本信息、合同条款明细、付款计划"

```
┌── header ────────────────────────────────────────────────┐
│ [提交审批] [批准] [驳回] [取消]     ○ 草稿 → ○ 审批中 → ● 执行中  │
├── sheet ─────────────────────────────────────────────────┤
│  ┌── oe_title ─────────────────────────────────────┐     │
│  │  合同编号    HT-2026-001                         │     │
│  │  合同名称    新奥天然气EPC总承包合同（大字）       │     │
│  └─────────────────────────────────────────────────┘     │
│                                                           │
│  ┌── group（左）──────────┐  ┌── group（右）────────────┐│
│  │ 客户      [___________]│  │ 签订日期  [2026-01-01 📅]││
│  │ 合同类型  [EPC总承包 ▼]│  │ 合同金额  ¥ [_________]  ││
│  │ 所属部门  [_________ ▼]│  │ 有效期至  [2027-01-01 📅]││
│  │ 负责人    [👤 _______ ]│  │ 付款条款  [月结30天 ▼]   ││
│  └────────────────────────┘  └──────────────────────────┘│
│                                                           │
│  ┌── notebook ───────────────────────────────────────┐   │
│  │ [合同条款] [付款计划] [其他信息] [备注]            │   │
│  │ ≡ │ 序号 │ 条款内容      │ 金额       │ 备注     │   │
│  │ ──┼──────┼───────────────┼────────────┼──────────│   │
│  │ ≡ │ 1    │ [___________] │ ¥[_______] │ [______] │   │
│  │   [添加条款]                                       │   │
│  │                             合同总额  ¥0.00        │   │
│  └───────────────────────────────────────────────────┘   │
│                                                           │
├── chatter ───────────────────────────────────────────────┤
│  消息 / 活动 / 关注者                                     │
└──────────────────────────────────────────────────────────┘
```

**字段说明**

| 字段 | Widget | 备注 |
|------|--------|------|
| 合同编号 | `char` | 序号自动生成，不可手动编辑 |
| 合同名称 | `char` | 必填，显示在 oe_title 大字 |
| 客户 | `many2one` | 关联 res.partner，过滤供应商 |
| 合同类型 | `selection` | 必填 |
| 所属部门 | `many2one` | 关联 hr.department |
| 负责人 | `many2one_avatar` | 默认当前用户 |
| 签订日期 | `date` | 默认今天 |
| 合同金额 | `monetary` | 必填 |
| 有效期至 | `date` | 选填 |
| 付款条款 | `many2one` | 关联 account.payment.term |
| 明细行 > 序号 | `integer` | 只读，自动编号 |
| 明细行 > 条款内容 | `char` | 必填 |
| 明细行 > 金额 | `monetary` | 必填 |
| 明细行 > 备注 | `char` | 选填 |
| 合同总额 | `monetary` | 只读，汇总明细行金额 |

**操作按钮**

| 按钮 | 显示条件 | 行为 |
|------|----------|------|
| [提交审批] | 草稿状态 | 状态→审批中，站内消息通知审批人 |
| [批准] | 审批中，且当前用户是审批人 | 状态→执行中，记录审批日期 |
| [驳回] | 审批中，且当前用户是审批人 | 弹窗填写驳回意见，状态→已驳回 |
| [取消] | 非已完成状态 | 状态→已取消 |

**状态栏**：statusbar 显示 `草稿 → 审批中 → 执行中 → 已完成`。已驳回、已取消不显示在状态栏（异常态）。
