# 审查报告模板

统一使用下面的结构输出。

```md
# 审查报告

## 审查对象
- 文档类型：
- 审查模式：
- 文档位置：
- 审查依据：

## 整体判断
- 总体结论：
- 是否建议进入下一阶段：可以进入 / 补充后进入 / 暂不建议进入
- 原因：

## 关键问题
1. ...

## 重要问题
1. ...

## 优化建议
1. ...

## 建议加入页面《待确认事项》
- [ ] ...

## 审查覆盖范围
- 已检查：
- 未展开：

## 残留风险
- ...
```

## 使用规则

- `快速审查` 可以保留相同结构，但内容压缩
- `完整审查` 应完整输出所有章节
- `阶段 Gate` 必须明确写出“是否建议进入下一阶段”
