# Kanban 视图规格模板

## 两种形态

| 场景 | 形态 | 判断依据 |
|------|------|----------|
| 按阶段/状态拖拽流转 | 分列 Kanban | 有明确的阶段字段（Selection / Many2one） |
| 卡片浏览，无分列 | 平铺 Kanban | 不按阶段分，像商品列表一样排列 |

---

## Kanban 线框图：只画一张卡片

Kanban 不需要画出所有列，画好一张卡片 + 文字说明列/阶段即可。

### 分列 Kanban 卡片

```
┌──────────────────────────┐
│ 标题（名称/编号，加粗）   │
│ 关键字段1: 值             │
│ 关键字段2: 值             │
│ [标签1] [标签2]          │   ← 可选：many2many_tags
│──────────────────────────│
│ [👤]          ¥XX,XXX ⋮ │   ← footer：头像 + 关键数值 + 菜单
└──────────────────────────┘
```

### 平铺 Kanban 卡片（有图片）

```
┌──────────────────────────┐
│ [🖼 图片/图标区域]        │
│ 标题                     │
│ 字段1: 值                │
│ 字段2: 值                │
│──────────────────────────│
│ [👤]              ★★☆  │   ← footer：头像 + 星级
└──────────────────────────┘
```

## 配套说明格式

### 卡片字段说明表

```markdown
**卡片字段说明**

| 字段 | Widget | 位置 | 备注 |
|------|--------|------|------|
| 字段标签 | widget名 | 主体 / footer | 说明 |
```

位置说明：
- **主体**：卡片上方区域，显示关键信息
- **footer**：卡片底部分隔线以下，显示负责人/数值/菜单

### 列/阶段说明

```markdown
**列/阶段说明**

分组字段：`stage_id`（或 `state`），按以下值分列：

| 列名 | 含义 | 可进入条件 |
|------|------|-----------|
| 待办 | 初始阶段 | 创建即进入 |
| 进行中 | 执行阶段 | 手动拖入或点击 |
| 已完成 | 终止阶段 | 手动拖入 |
```

如无特殊进入条件，可以只列出列名和含义。

---

## 卡片内容选取原则

卡片空间有限，只放 **3-5 个最关键的信息**：

- **标题**（必有）：名称或编号，最重要的识别信息
- **1-3 个关键业务字段**：影响用户决策的核心字段
- **footer**：负责人头像（`many2one_avatar`）+ 关键数值（金额/进度/截止日期）

不要把所有字段都放进卡片，次要信息看 Form 视图。

## 填充示例

需求："项目任务看板，按阶段流转，显示负责人和截止日期"

**卡片线框图**

```
┌──────────────────────────┐
│ 任务名称                 │
│ 所属项目: XXXXX          │
│ 截止日期: 2026-04-15     │
│──────────────────────────│
│ [👤]                ⋮   │
└──────────────────────────┘
```

**卡片字段说明**

| 字段 | Widget | 位置 | 备注 |
|------|--------|------|------|
| 任务名称 | `char` | 主体 | 卡片标题，必有 |
| 所属项目 | `many2one` | 主体 | — |
| 截止日期 | `date` | 主体 | 逾期时显示红色（`remaining_days` widget）|
| 负责人 | `many2one_avatar` | footer | 显示头像 |

**列/阶段说明**

分组字段：`stage_id`，按以下值分列：

| 列名 | 含义 |
|------|------|
| 待办 | 新建任务默认进入 |
| 进行中 | 执行中的任务 |
| 已完成 | 完成的任务 |

- 可拖拽卡片在列间移动
- 列头显示 progressbar：已完成任务数 / 总任务数
- 最多显示 10 列，超出折叠
