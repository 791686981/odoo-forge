# Settings 视图规格模板

Settings 视图本质是特殊 Form，结构固定：左侧应用导航 + 右侧设置内容区。

---

## 骨架

```
┌──────────────────────────────────────────────────────────────────┐
│ 设置                                                    [保存]  │
├── app 导航 ──┬── 设置内容 ──────────────────────────────────────┤
│              │                                                  │
│  ● 当前应用  │  ┌── block: 分组标题1 ──────────────────────┐   │
│  ○ 其他应用  │  │                                          │   │
│  ○ 其他应用  │  │  [✓] 开关型设置项                        │   │
│              │  │      说明文字：这个开关控制什么            │   │
│              │  │      ┌── 子设置（开关开启后显示）───┐     │   │
│              │  │      │ 配置字段    [输入框/下拉]    │     │   │
│              │  │      └─────────────────────────────┘     │   │
│              │  │                                          │   │
│              │  │  配置项标签    [输入框 _________]        │   │
│              │  │      说明文字                             │   │
│              │  │                                          │   │
│              │  └──────────────────────────────────────────┘   │
│              │                                                  │
│              │  ┌── block: 分组标题2 ──────────────────────┐   │
│              │  │  ...                                      │   │
│              │  └──────────────────────────────────────────┘   │
│              │                                                  │
└──────────────┴──────────────────────────────────────────────────┘
```

---

## 设置项类型

| 类型 | 线框图画法 | 说明 |
|------|-----------|------|
| Boolean（开关）| `[✓] 标签` / `[ ] 标签` | 左侧显示开关，开启后展开子设置 |
| 非 Boolean（输入）| `标签    [输入框/下拉]` | 右侧显示输入控件 |
| Header 横幅 | 全宽横幅，写范围说明 | 区分"当前公司"/"当前网站" |

---

## 配套说明格式

线框图之后，附设置项说明表：

```markdown
**设置项说明**

| 设置项 | 类型 | Widget | 默认值 | 说明 |
|--------|------|--------|--------|------|
| 设置标签 | Boolean / Char / Integer ... | widget名 | 默认值 | 控制什么功能 |
```

子设置（开关开启后显示的从属配置）在备注中说明依赖关系。

## 填充示例

需求："合同管理模块的设置页面，包含审批设置和通知设置"

```
┌──────────────────────────────────────────────────────────────────┐
│ 设置                                                    [保存]  │
├── app 导航 ──┬── 设置内容 ──────────────────────────────────────┤
│              │                                                  │
│  ○ 常规     │  ┌── block: 审批设置 ───────────────────────┐   │
│  ○ 销售     │  │                                          │   │
│  ● 合同管理 │  │  [✓] 启用金额审批                        │   │
│  ○ 项目     │  │      超过指定金额的合同需要额外审批        │   │
│              │  │      ┌── 子设置 ─────────────────┐      │   │
│              │  │      │ 金额阈值    [¥100,000]    │      │   │
│              │  │      │ 审批人      [总经理 ▼]    │      │   │
│              │  │      └───────────────────────────┘      │   │
│              │  │                                          │   │
│              │  │  [ ] 启用多级审批                        │   │
│              │  │      开启后合同按金额自动匹配审批流程      │   │
│              │  │                                          │   │
│              │  └──────────────────────────────────────────┘   │
│              │                                                  │
│              │  ┌── block: 通知设置 ───────────────────────┐   │
│              │  │                                          │   │
│              │  │  [✓] 审批超时提醒                        │   │
│              │  │      超过指定天数未审批自动发送提醒        │   │
│              │  │      ┌── 子设置 ─────────────────┐      │   │
│              │  │      │ 提醒天数    [7]           │      │   │
│              │  │      └───────────────────────────┘      │   │
│              │  │                                          │   │
│              │  │  [✓] 合同到期提醒                        │   │
│              │  │      合同到期前自动通知负责人              │   │
│              │  │      ┌── 子设置 ─────────────────┐      │   │
│              │  │      │ 提前天数    [30]          │      │   │
│              │  │      └───────────────────────────┘      │   │
│              │  │                                          │   │
│              │  └──────────────────────────────────────────┘   │
│              │                                                  │
└──────────────┴──────────────────────────────────────────────────┘
```

**设置项说明**

| 设置项 | 类型 | Widget | 默认值 | 说明 |
|--------|------|--------|--------|------|
| 启用金额审批 | Boolean | boolean_toggle | 关 | 开启后显示金额阈值和审批人子设置 |
| 金额阈值 | Float | monetary | 100,000 | 依赖"启用金额审批"开启 |
| 审批人 | Many2one | many2one | — | 关联 res.users，依赖"启用金额审批"开启 |
| 启用多级审批 | Boolean | boolean_toggle | 关 | — |
| 审批超时提醒 | Boolean | boolean_toggle | 开 | 开启后显示提醒天数子设置 |
| 提醒天数 | Integer | integer | 7 | 依赖"审批超时提醒"开启 |
| 合同到期提醒 | Boolean | boolean_toggle | 开 | 开启后显示提前天数子设置 |
| 提前天数 | Integer | integer | 30 | 依赖"合同到期提醒"开启 |
