# Search 视图规格模板

## 两种形态

| 场景 | 形态 | 判断依据 |
|------|------|----------|
| 标准搜索 + 筛选 | 普通 Search | 大多数列表/看板 |
| 搜索 + 左侧分类面板 | 带 Searchpanel 的 Search | 需要像电商侧边栏的分类导航 |

---

## 形态一：普通 Search

Odoo 搜索 UI 由两部分组成：搜索栏（激活筛选显示为芯片）+ 点击展开的三列下拉面板。

```
搜索栏（激活筛选时）:
  🔍 [筛选芯片1 ×] [筛选芯片2 ×]  [搜索...]               [▲]

展开下拉面板（三列）:
  ┌── 筛选 ──────────────┬── 分组方式 ──────────┬── 收藏夹 ──────────┐
  │ ✓ 筛选项1（激活）     │  分组项1             │  保存当前搜索 ▼    │
  │ ──────────────────   │  分组项2             │                    │
  │   筛选项2            │  分组项3             │                    │
  │   筛选项3            │                      │                    │
  │ ──────────────────   │                      │                    │
  │   时间筛选1          │                      │                    │
  │   时间筛选2          │                      │                    │
  └──────────────────────┴──────────────────────┴────────────────────┘
```

**筛选列中的 `──────────────────` 就是 separator（分隔线），上下两组之间是 AND 逻辑。**

## 形态二：带 Searchpanel 的 Search

```
搜索栏:
  🔍 [搜索...]                                             [▲]

展开下拉面板（同上三列）

┌── searchpanel（左侧）──┬── 主视图区域（右侧）──────────────┐
│                        │                                   │
│ 分类字段1              │  （List / Kanban 视图）            │
│  ● 全部（25）          │                                   │
│  ○ 选项A（12）         │                                   │
│  ○ 选项B（8）          │                                   │
│                        │                                   │
│ 分类字段2              │                                   │
│  ☐ 标签1（3）          │                                   │
│  ☑ 标签2（7）          │                                   │
│  ☐ 标签3（2）          │                                   │
│                        │                                   │
└────────────────────────┴───────────────────────────────────┘
```

- `○/●` 单选（Many2one / Selection 字段，`select="one"`）
- `☐/☑` 多选（Many2many 字段，`select="multi"`）
- 括号内数字 = 该分类下的记录数

---

## Filter 逻辑（重要）

```
同组内（无分隔线）→ 互为 OR：
  [我的记录] [待审批] [已完成]
  选任一个都能出来

分隔线隔开 → 两组之间是 AND：
  [我的记录] [待审批]    ←── 组1，互为 OR
  ──────────────────          separator
  [本月] [本季度]         ←── 组2，互为 OR
  组1 AND 组2 同时满足才出来
```

## 配套说明格式

线框图之后，附以下三个说明表：

### 搜索字段

```markdown
**搜索字段**：输入关键词时匹配以下字段：
- 字段标签（field_name）
- 字段标签（field_name）
```

### 筛选器表

```markdown
**筛选器**

| 标签 | 过滤条件（白话）| 所在组 |
|------|----------------|-------|
| 筛选标签 | 说明过滤了什么 | 组1 / 组2 |
```

所在组用于判断 AND/OR：同组内 OR，跨组 AND。分隔线隔开即换组。

### 分组依据表

```markdown
**分组依据**

| 标签 | 字段 |
|------|------|
| 按xxx | field_name |
```

## 填充示例

需求："合同搜索视图，有部门侧边栏，按状态和时间筛选"

```
搜索栏:
  🔍 [合同编号 / 合同名称 / 客户名称          ]           [▲]

展开下拉面板:
  ┌── 筛选 ──────────────┬── 分组方式 ──────────┬── 收藏夹 ──────────┐
  │   我的合同           │  按部门              │  保存当前搜索 ▼    │
  │   待审批             │  按状态              │                    │
  │   执行中             │  按客户              │                    │
  │   已完结             │  按月份              │                    │
  │ ──────────────────   │                      │                    │
  │   本月               │                      │                    │
  │   本季度             │                      │                    │
  └──────────────────────┴──────────────────────┴────────────────────┘

┌── searchpanel（左侧）──┬── 合同列表（右侧）────────────────────────┐
│                        │                                           │
│ 所属部门               │  （合同 List 视图）                        │
│  ● 全部（25）          │                                           │
│  ○ 工程部（12）        │                                           │
│  ○ 设计部（8）         │                                           │
│  ○ 采购部（5）         │                                           │
│                        │                                           │
│ 合同类型               │                                           │
│  ☐ EPC（10）           │                                           │
│  ☑ 设计（8）           │                                           │
│  ☐ 监理（7）           │                                           │
│                        │                                           │
└────────────────────────┴───────────────────────────────────────────┘
```

**搜索字段**：输入关键词时匹配以下字段：
- 合同编号（name）
- 合同名称（name）
- 客户名称（partner_id.name）

**筛选器**

| 标签 | 过滤条件 | 所在组 |
|------|---------|-------|
| 我的合同 | 当前用户是负责人 | 组1 |
| 待审批 | 状态=审批中 | 组1 |
| 执行中 | 状态=执行中 | 组1 |
| 已完结 | 状态=已完成 | 组1 |
| 本月 | 签订日期在本月内 | 组2 |
| 本季度 | 签订日期在本季度内 | 组2 |

> 组1 内互为 OR；组1 AND 组2。

**分组依据**

| 标签 | 字段 |
|------|------|
| 按部门 | department_id |
| 按状态 | state |
| 按客户 | partner_id |
| 按月份 | date_signed:month |
