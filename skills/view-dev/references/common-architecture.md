## 定义

视图的架构由 JavaScript 框架解析的 XML 数据定义。

## RNG 约束

对于大多数视图，都有一个 `*.rng` 文件定义属性和可能的架构。

## 例外情况

某些视图不受此类文件约束，要么是因为它们接受 HTML 内容，要么是出于性能原因。
