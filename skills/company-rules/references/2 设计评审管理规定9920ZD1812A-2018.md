# 能源工程技术有限公司设计评审管理规定

# 1 总则

# 1.1 目的

明确规定设计和开发评审的职责和权限，使设计和开发中的任何问题得到解决，最大限度满足顾客的合理要求。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 项目经理负责组织并主持公司级评审。  
2.2 设计经理负责组织分公司级评审。  
2.3 专业带头人负责主持专业评审。  
2.4 总工程师、专家团队人员负责担任评审人。

# 3 评审工作的组织

# 3.2 公司级评审

3.1 评审工作应在设计方案提出并作好各项准备工作后进行,评审分为公司级、分公司级评审和专业评审。

由公司组织的新技术新工艺项目（规模相近，投入运行五次以内的）、大型综合工程和重大 EPC 项目（如：LNG 液化工厂、LNG 接收站、焦炉气制 LNG、煤制天然气等）、其他特殊项目的设计方案，由设计经理或专业带头人准备好相关评审资料，经分公司内部评审后提请技术质量赋能部组织公司级评审。技术质量赋能部组织进行评审,确定评审时间及参会人员。评审会议由总工程师主持。评审资料应提前三天发送参评人员。

# 3.3 分公司级评审

综合工程的评审由设计经理组织，项目组有关人员、专家团队相关技术专家参加。

特殊工程项目设计方案的评审由设计经理组织，分公司总工程师审批，分公司有关领导、项目组有关人员、专家团队相关技术专家、分公司技术赋能群参加。

# 3.4 专业评审

各专业设计方案的评审由专业带头人组织，专业带头人指定的评审人主持，相关人员参加。必要时，设计经理、分公司总工程师及有关领导参加。

3.5 单项工程项目设计方案的评审，由设计经理组织，区域召集人主持，项目组有关人员参加；对于技术简单、方案成熟的单项工程项目，以审核人员签署作为设计评审。  
3.6 当初步设计确认后，如果施工图设计的方案有较大变化，或按合同规定顾客要求公司在设计某一适当阶段进行评审时，由设计经理提请组织进行公司级或分公司级评审。  
3.7 在设计过程中发生需要研究解决的问题，可由设计经理、专业带头人、设计人员随时提请评审。  
3.8设计评审活动由设计经理组织形成评审记录，相关人员进行签署。

# 4 评审时机及评审工作的准备

# 4.1 评审时机

# 4.1.1 常规综合项目

（1） 设计方案评审；  
（2） 设计输入评审；  
（3） 施工图阶段随时遇到问题时的评审；  
（4） 设计输出评审；  
（5） 工艺流程及 PID 图发表前的评审；  
（6） 技术开发过程中关键设备总图的评审；  
（7） 危险区域划分图的评审。

# 4.1.2 大型综合工程和重大 EPC 项目

# （1）总图评审；

（2）建筑方案评审，包括结构形式，平、立面布置图，效果图；  
（3）4.1.1 条的评审

4.1.3 新技术、新工艺项目

（1）总图及设备平面布置图评审；  
（2）关键设备及材料评审；  
（3）自控方案评审；  
（4）消防及给排水方案评审；  
（5）变配电方案评审；  
（6）建构筑方案评审。  
（7）4.1.1 条的评审

4.1.4 特殊项目

根据项目情况确定

4.2 评审工作的准备

4.2.1设计评审由设计人员拟定设计方案或汇报提纲，评审主持人或组织者事先审查设计方案，研究是否具备召开设计评审会议的条件。  
4.2.2 如已具备召开评审会议的条件，评审组织者确定会议地点、时间及参加会议人员名单，并通知有关人员。  
4.2.3 如果暂不具备召开评审会议条件，评审主持人或组织者应通知有关人员限期作好准备工作。

5 评审的范围和内容

5.1 需要由各专业评审的范围和内容见各专业制定的设计评审范围和内容规定。  
5.2 需要由公司级、分公司级评审的范围和内容：  
5.2.1 初步设计阶段

（1）各综合工程的主要流程、平面布置、可重复利用已有设计或在已有设计的基础上进行修改的内容；  
（2）各综合工程的设计参数、单元组成、设计规模和采用的主要技术；

（3）各单元需要引进的设备、材料和技术；  
（4）各单元的自控水平和管理模式；  
（5）各专业各单元之间不协调、不统一或有矛盾的设计方案；  
（6）对重复利用已经过实际考验的成熟设计，仅作局部修改时；  
（7）各专业采用新工艺、新技术、新材料的设计方案；  
（8）各专业提出或设计经理认为需要组织分公司级评审的问题。

# 5.2.2 施工图设计阶段

（1）根据实际情况需对初步设计方案或初步设计的审批意见进行较大修改时，需对修改方案进行评审，并征得顾客同意，如有必要，报初步设计批准单位备案或批准；  
（2）对重复利用已经过生产考验的成熟设计，仅作局部修改时；  
（3）在设计过程中各专业之间有矛盾的设计方案；  
（4）各专业采用新工艺、新技术、新材料的设计方案；  
（5）设计经理或专业负责人认为需要分公司级评审的问题。

# 5.2.3 重大设计变更

5.3 单项工程项目设计评审的范围和内容视项目具体情况而定。

# 6 设计评审结论

6.1 经过评审，对评审内容做出予以通过和进行某些修改的结论。  
6.2 如果设计深度不够，无法做出结论，主持人应根据具体情况提出要求重新评审。

# 7 《设计评审记录》

7.1 设计评审可以任何方式进行，评审结果应作记录，可填写《设计评审记录》，或用《会议纪要》予以记录。设计人员提出的汇报提纲及有关资料应作为评审记录的附件。  
7.2 《设计评审记录》由主持人签署，参加评审人员应有签到记录。  
7.3 经公司级、分公司级及专业评审确定的方案，任何人不得自行更改，若发现新的问题，提请有关部门重新组织相应级别的评审。

7.4 《设计评审记录》作为项目文件之一存档。  
7.5 《设计评审记录》中的最终结论，应作为设计依据之一，认真实施。在设计验证过程中，各级设计验证人员应对最终结论的贯彻执行情况进行监督检查。

# 8 附则

8.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
8.2 本制度自颁发之日起试行，试行期为 6 个月。6 个月内若无修订，则自动转入实施。
