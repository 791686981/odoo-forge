# 新地能源工程技术有限公司质量风险管理规定

# 第一章 总则

# 第一条 目的

依据 9000 质量管理体系相关标准要求，为建立风险和机遇的应对措施，增强风险预防能力，特制定本规定。

# 第二条 适用范围

本规定适用于新地能源工程技术有限公司（以下简称“公司”）及下属各单位在质量管理体系活动中应对风险和机遇的活动。

# 第三条 释义

（一）风险：在一定环境下和一定限期内客观存在的、影响公司目标实现的各种不确定性事件。  
（二）机遇：对公司有正面影响的条件和事件,包括某些突发事件等。  
（三）风险管理：风险管理就是一个识别、确定和度量风险，并制定、选择和实施风险处理方案的过程。  
（四）风险严重度：风险发生后其所产生的影响的严重程度。  
（五）风险发生频度：风险出现的频率或者概率。

# 第二章 分则

# 第四条 组织与职责

# （一）公司一号位

组织建立质量风险管理机制，负责公司内各单位间的质量风险管理协调，满足相应的资源保障

# （二）技术质量智能能力群

1、负责建立公司风险管理制度，并进行维护。  
2、监督指导各单位的风险评估、应对风险的策划和应对风险措施的执行、更新情况。

（三）各单位（含智能能力群、业务中心、专业中心及区域分公司）

1、负责本单位的风险和机遇的识别及评估，并制定相应的措施以规避或者降低风险并落实执行。

2、建立风险管理档案。

# 第五条 风险和机遇识别

为全面识别和应对在生产和管理活动中存在的风险和机遇，各单位应建立识别和应对的方法，确认本单位存在的风险。评估的结果应在单位内部交流，并作适当记录。各单位在风险识别过程中应识别包括但不限于以下方面的风险：

（一）对公司经营环境、业务开发、目标顾客、订单要求等的变化带来的风险；  
（二）生产作业过程中的消安风险；  
（三）设备、模具、工装夹具、刀具等对产品质量造成的风险；  
（四）产品售后的风险；  
（五）产品设计开发阶段及变更过程的设计失效风险；  
（六）供应商管理和采购控制过程的风险；  
（七）过程检验和监视测量设备的管理过程的风险；  
（八）过程失效的风险。

识别后，根据实际情况进行应对管理。

风险分为运营风险（经营活动）和管理风险（体系运行）两大类。

# 第六条 风险分析

风险分析是对所确定的危害源有关的风险进行预估，针对不同的风险项目需选择应用不同的分析工具。

（一）选择风险评估的工具；  
（二）确定风险的因素；  
（三）界定风险因素的范围；  
（四）界定风险因素的类型；

# （五）确定采取的行动。

# 第七条 风险评估

各单位应对所识别的风险进行评估，根据评估的结果对风险采取措施，从而达到降低或消除风险的目的。

（一）风险评估的时机：一般情况下每年进行一次风险评估，但在以下几种情况发生时，可适时引入新的风险识别并更新。

1.在对新产品、设施、设备的引入过程中；  
2.内外部环境变化时；  
3.变更控制、偏差处理、客户投诉、纠正和预防措施时；  
4.法律法规政策方针的更新与变化等情况下。

# （二） 风险评估的方法及工具：

1.各单位根据各自业务特点选择使用合适的风险评估方法及工具，如：预先危险性分析法 PHA、失效模式与影响分析 FMEA、危险与可操作性分析 HAZOP 等。  
2.风险由两方面因素构成，风险评估是基于对风险发生的频次和风险严重程度这两方面而考虑得出的综合结论，评估结果需被量化。

# （1）风险的严重程度评价准则

严重程度用于评价潜在风险可能造成的损害程度，包括但不限于风险产生后会导致的危害有：

a.社会、政府对公司的惩罚；  
b.影响顾客满意导致订单流失；  
c.风险发生时导致的人身伤害；  
d.直接的经济损失；  
e.对公司形象的损害。

为便于识别风险所带来的危害程度，对风险的严重程度进行区分，风险严重度分为以下五类：

a.非常严重  
b.严重

c.较严重  
d.一般  
e.轻微

风险严重度的评价准则：

# （2）风险的发生频率评价准则

风险的发生频率是指潜在风险出现的频率，为便于识别和定义，将风险频度定义为5级，如下所示：

a.极少发生；  
b.很少发生；  
c.偶尔发生;   
d.有时发生；  
e.经常发生。

风险的发生频率的评价准则：

# （3）风险的可接受准则

风险可接受准则是通过计算得出的风险系数来判定风险是否可接受，通过对风险的严重度和风险的发生频率评价后，通过计算风险系数确定是否对风险采取措施。风险系数的计算如下公式：

风险系数=风险严重程度等级×风险频度等级

# 第八条 风险应对

风险应对的方法包括：风险接受、风险降低、风险规避。对风险所采取的措施应考虑尽可能的消除风险，在无法消除或暂无有效的方法或者采取消除风险的方法的成本高出风险存在造成的损失时，再选择采取降低风险或者风险接受的风险应对方法。

# （一）风险接受

风险接受是指公司承担风险造成的损失，一般适用于那些造成损法：

1.采取风险规避措施所带来的成本远超出潜在风险所造成的损失时；

2.造成的损失较小且重复性较高的风险；  
3.既无有效的风险降低的措施，又无有效的规避风险的方法时；  
4.按风险评估准则中计算得出风险系数低于 5 的低风险。

# （二）风险降低

风险降低即采取措施降低潜在风险所带来的损坏或损失。各单位应制定详细的风险降低措施来降低风险，若采取的措施能够有效的降低所遭受的风险，应将采取措施的记录进行保留或者写入文件进行归档，以便后期重复发生时作为改善的依据。当出现以下情况时，可采取风险降低方法：

1.采取风险规避措施所带来的成本远超出潜在风险所造成的损失时；  
2.无法消除风险或暂无有效的规避措施规避风险时；  
3.按本规定要求的风险评估准则中计算得出风险系数为 5 至 15之间的一般性风险。

# （三）风险规避

风险规避是指通过有计划的变更来消除风险或风险发生的条件，保护目标免受风险的影响。风险规避并不意味着完全消除风险，所要规避的是风险可能给我们造成的损失。一是要降低损失发生的机率，这主要是采取事先控制措施；二是要降低损失程度，这主要包括事先控制、事后补救两个方面。

# （四）风险管理的监督与改进

风险识别和评估活动是用于识别风险并综合考虑对风险应采取的有效措施，当风险系数过高时应采取风险规避或者降低风险，以减行，在制定措施时，应考虑以下方面的内容：

1.制定的措施应是在现有条件下可执行和可落实的；  
2.制定的措施应落实到个人，每个人应完成的内容应得到明确；  
3.应指派专人为措施的执行进度和效果进行跟进，确保采取的措

施被有效的落实。

# 第九条 风险沟通

为了更好的理解风险及相关问题和决策，各单位建立风险沟通机制，风险评估者、管理者以及其他各方积极就风险及其相关因素进行相互交流信息和意见。参与者可以在风险管理过程中的任何阶段进行交流，对于公司就质量风险管理进行通报时，可利用现有法规与指南所规定的已有途径。

风险沟通达成的基本目标：

（一）疏导风险和改变风险的进程。  
（二）危机预警和阻止风险的发生。  
（三）危机防范和降低风险的危害程度。

# 第十条 风险回顾

各单位应建立并实施对风险进行定期回顾的机制。风险管理是持续、动态的管理过程，需要周期性的回顾与监督，以确保风险管理过程的持续有效性和适用性。风险管理过程的结果应结合新的知识与经验进行回顾。风险回顾的频率取决于风险管理水平，但每年不得少于2次。

# 第十一条 风险管理档案

各单位应对每一个产品建立风险管理档案，该档案包括所有与风险管理有关各阶段的文件和记录。

# 第三章 附则

第十二条 本管理规定由技术质量智能能力群负责解释、修订。自发布之日起发布实施。
