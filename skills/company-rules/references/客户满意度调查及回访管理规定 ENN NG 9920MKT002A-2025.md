# 新地能源工程技术有限公司客户满意度调查及回访管理规定

# 第一章 总则

# 第一条 目的

为了对客户满意度调查及回访进行有效控制，保证新地能源工程技术有限公司（以下简称“公司”）服务管理工作有效开展，挖掘设计、采购、施工过程中存在的问题，维系客户关系，维护公司品牌形象，深度开发市场，使公司服务工作开展规范化、系统化，提升公司客户满意度，特制定本规定。

# 第二条 管理原则

（一）高度重视：公司全员须高度重视客户服务质量及满意度调查结果。  
（二）全员参与：通过向全体员工提供工具、计划，营造全员参与氛围，使其主动承担解决问题责任，不断发现问题并积极参与改进。  
（三）客户导向：客户服务以积极响应客户需求、不断提升客户满意为主导思想，兼顾成本控制和资源匹配的管理要求。  
（四）持续改进：对改进效果通过实践进行检验并固化执行，同时通过进一步监督与信息收集，持续对服务产品、过程和体系实施改进。

# 第三条 适用范围

本规定适用于新地能源工程技术有限公司所属各单位。

# 第二章 分则

# 第四条 组织与职责

# （一）客品智能能力群

1.是公司客户满意度调查及回访归口赋能部门  
2.负责制定公司客户满意度调查及回访规定、规范、指导方针，并监督执行。  
3.负责对客户满意度调查活动进行信息汇总与分析。

4.为公司客户服务各项活动提供相关赋能和资源支持。  
5.负责将客户满意度调查结果应用于业务中心/分公司考核。

# （二）各业务中心/分公司

1.为客户服务管理执行单位，负责执行公司客户满意度调查及回访工作。  
2.监督、指导各业务中心/分公司相关部门客户服务工作的开展。  
3.负责对本公司服务质量进行跟踪调查，并对调查及回访结果进行统计与分析，按时报送至客品智能能力群。  
4.及时处理调查及回访过程中发现的问题，并针对问题进行调查处理，及时反馈调查结果，提出改进措施。

# 第五条 调查方法

（一）工程设计/施工过程中，业务中心/分公司客户服务人员负责收集业主、监理和主管部门以面谈、电话、传真等方式提出的有关工程设计/施工质量、进度等方面的意见或建议，及时做好相应记录，同时，注重过程服务质量管理与控制。  
（二）各业务中心/分公司组织相关部门整理各项资料，对工程质量服务、进度服务、环境服务、安全管理、文明施工等方面进行客户满意度调查，并报客品智能能力群备案。  
（三）针对客户满意度调查中出现的各类问题，业务中心/分公司组织相关部门及时进行纠正，并提出改进/预防措施，同时就相关问题进行回访。  
（四）技术服务部门/业务中心/分公司，在公司技术、产品推广中，向客户提供相关技术指导和咨询服务等。  
（五）通过与业主的沟通，合理安排工程设计/施工进度计划，确保工程按时完成，并建立目标责任制度。  
（六）严格按照 ISO9000 质量体系认证的管理要求，全方面保证工程质量，确保工程在设计使用年限内符合各项质量指标，定期针对工程施工过程中的质量问题进行总结，避免类似质量问题重复出现。

（七）全面贯彻国家和公司有关工程安全相关管理制度，坚持安全第一、预防为主、综合治理的方针，依法规范工程的施工管理，保障人身和设备、工程安全，确保工程建设安全、高效、有序的顺利进行。

# 第六条 客户满意度

# （一）客户满意度调查表

# 1.调查表发放

各业务中心/分公司客户管理人员在每季度首月对公司的工程与服务业务开展客户满意度调查，发放客户满意度调查表。调查范围涵盖公司所属业务中心和业务型自驱组织。保证表单发放不重、不漏。

# 2.调查表回收

调查表格发放后，各业务中心/分公司客户管理人员在当月月底之题。

# 3.问题反馈

各业务中心/分公司客户管理人员将调查表问题清单在 3个工作日内反馈给问题对应各业务中心/分公司接口人。

# 4.问题整改

各业务中心/分公司接口人在收到调查表问题清单之后的5个工作日内将问题清单反馈给相关项目经理、项目组、机组或设计室，跟进问题整改，落实到责任人，登记《客户意见内部流转单》，收集整理后统一发送给客品智能能力群客户管理人员。

# 5.结果统计

客品智能能力群客户管理人员每季度将客户满意度调查结果统一进行总结，以调查表中各业务中心/分公司所有项目平均值为考核数据，用于每季度的业务中心/分公司考核工作。

# （二）客户满意度现场访问

1.针对公司较大型项目，由专职客户服务人员至项目现场访问，收集项目现场人员对我方工程服务质量、进度等相关意见、建议。

2.对收集到的信息进行分类，由业务中心/分公司客服人员反馈至相关项目部，由项目部负责人针对问题进行整改。  
3.问题整改完毕，须由项目部负责人就业主反馈的问题进行项目回访，以达到客户满意为目的。

# （三）客户满意度第三方访问

1.针对客户满意度调查客观性，各业务中心/分公司客服人员可对项目第三方人员进行访问，如监理单位、政府监管部门等其他了解项目情况的第三方人员。  
2.听取项目第三方负责人对项目最真实的意见，将问题项反馈至项目部，由项目部负责人针对问题进行整改，建立常态化的服务品质监管机制。

# 第七条 回访管理

# （一）制定回访计划

各业务中心/分公司依据自身业务范围，制定年度回访计划，包括回访内容、回访范围、回访时间等。

# （二）回访客户问题处理

回访过程中，对于客户提出意见、建议、投诉的，回访人员需填写《客户意见内部流转单》，并将单据进行处理。

# （三）回访信息管理

回访管理部门须建立回访信息台账，对客户评价结果、意见、建议及回访人员、回访时间、问题批转情况等进行及时、详细的记录。

# 第三章 附则

# 第八条 附则

（一）本制度由客品智能能力群负责解释、修订。  
（二）各业务中心/分公司须结合自身特点，在本制度的基础上加强客户服务管理。  
（三）本制度自发行之日起正式实施。

附件 1：

# 客户满意度工作要求

# 一、客户满意度调查对象选取要求

各分公司及业务中心每季度应依据自身业务范围：常规、小型综合项目以客户维度进行满意度调查，大型综合项目、装备集成项目、单独设计项目等以项目维度进行满意度调查，保证表单发放不重、不漏。

# 二、每季度客户满意度调查数量要求

内部客户：每年度覆盖所有内部客户

外部客户：每季度调查数量不少于在建外部项目数的 10%

大型综合项目：每年度覆盖所有大型综合项目客户

特殊业务依据自身业务范围适当调整客户满意度调查数量，原则进行满意度调查。

若在调查过程中客户存在交叉的，应以区域分公司为对接客户第一界面。保证表单发放不重、不漏。

# 三、客户满意度调查工作实施要求

客户满意度调查以发放问卷形式进行开展，问卷格式参考工具表单，各分公司/业务中心根据业务形式不同自行选择。问卷应包括明确的业主/项目名称，业主项目负责人签字，填写日期，分数等，列入《客户满意度调查报告》。

各业务中心/分公司客户管理人员在每季度首月对本公司的工程与服务开展客户满意度调查，发放客户满意度调查表。调查表格发放后，各业务中心/分公司客户管理人员在当月月底之前（如遇节假日顺延）回收调查表格。并将调查结果形成台账，汇总问题。各业务中心/分公司客户管理人员将调查表问题清单在 3 个工作日内反馈给问题对应各业务中心/分公司接口人。各业务中心/分公司接口人在收到调查表问题清单之后的5个工作日内将问题清单反馈给相关项目经理、项目组、

机组或设计室，跟进问题整改，落实到责任人，登记《客户意见内部流转单》，收集整理写入《客户满意度调查报告》，于每季度末月 20 前反馈至客品智能能力群。

客品智能能力群客户服务管理人员每季度将客户满意度调查结果统一进行总结，以调查表中各业务中心/分公司所有项目平均值为考核数据，用于每季度的业务中心/分公司考核工作。

# 四、《客户满意度调查报告》编制要求

《客户满意度调查报告》应参考附件提供的模板，包括但不限于本季度客户满意度调查基本情况，本月需求/投诉事件及处理情况概述，客户满意度汇总表、客户意见内部流转单、客户满意度调查原始表等

工作涉及的相关人员应对报告质量进行把关，并完成报告中签署回，由相关组织重新提交。

附件 2：

# 客户回访工作要求

# 一、客户回访对象选取要求

客户回访：各分公司及业务中心每年制定客户回访工作计划，认真选取有代表性、有二次合作意向及重点的客户，通过实地拜访、电话等形式进行客户回访。

# 二、每年度客户回访数量要求

重点客户及重点项目必回访，其余回访依据组织实际情况按需开展。

特殊业务依据自身业务范围适当调整客户回访数量，原则上每年度回访不得少于 10 个客户，若全部客户不足 10 个的应对所有客户进行回访。

若在回访过程中客户存在交叉的，应以区域分公司为对接客户第一界面。

# 三、客户回访工作实施要求

各分公司/业务中心年度开展客户回访工作，客户回访以现场拜访、电话等形式进行开展，并填写《客户回访单》。回访过程中，对于客户提出意见、建议、投诉的，回访人员需填写《客户意见内部流转单》，并将单据进行处理。各分公司/业务中心回访管理部门须建立回访信息台账，对客户评价结果、意见、建议、联系人、联系方式及回访人员、回访时间、问题批转情况等进行及时、详细的记录，并形成季度回访分析统计表，写入《客户回访报告》，于每年 12 月 20 日前反馈至客品智能能力群。

# 四、《客户回访报告》编制要求

《客户回访报告》应参考附件提供的模板，包括但不限于本年度客户回访基本情况、回访过程中产生的表单等。工作涉及的相关人员应对报告质量进行把关，并完成报告中签署确认。未按本通知要求开展工作或未按附件要求完成报告的，将予以退回，由相关组织重新提交。
