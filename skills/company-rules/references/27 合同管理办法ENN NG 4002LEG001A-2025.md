# 新地能源工程技术有限公司石家庄能源化工技术分公司合同管理办法

# 第一章 总则

# 第一条 目的

为规范合同起草、签订和履行，防范合同风险，支持各业务创值活动，维护公司合法权益，根据相关法律法规，结合新地能源工程技术有限公司合同管理办法及公司实际，特制定本办法。

# 第二条 释义

本办法所涉及的合同包括以公司名义签订、变更、解除的各类合同、协议及相关法律性文件。

# 第三条 适用范围

本办法适用于燃气化工设计中心各组织。

# 第二章 分则

# 第四条 组织与职责

（一）一号位对合同管理工作全面负责。  
（二）一号位（或委托的代理人）负责各类合同的签订。  
（三）总工程师负责把关合同中技术层面的相关内容。  
（四）综合智能能力群：

1.合同专业归口管理部门,负责制定合同相关的制度及规则并监督实施。  
2.组织公司各类合同、合作协议、保密协议等合同范本的起草、优化和更新。并负责合同价款、结算及支付等归口职责范围内相关内容的审核把关；合同法律风险的防范等相关内容的审核、赋能。  
3.合同签订授权委托的管理及合同评审授权的管理。

# （五）市场开发群：

1.统筹内外部市场合同起草、谈判、签约及履约管理，并对合同有效性进

行评估。

2.对合同签约客户的资质和信用情况等进行调查，并全面负责合同的起草和谈判。  
3.负责具体项目合同(项目及服务内容、范围、付款条件、质量要求、履行期限、履行方式和罚则等)的完善、组织评审、合规与谈判管理。  
4.对即将签约评审合同，须预先组织有关专业人员对收益结果进行测算并对过程执行及合同履约结果验证负责。

# （六）各组织：

1.结合具体实际业务，涉及采购-服务类合同与采购-物资类合同由需求组织发起，经营协助组织与审核；其他类合同根据合同类型进行组织与审核，行政类合同由行政组织与执行;法律、保险、金融、资金类合同由财运组织与执行;人力及劳务派遣类合同由人发组织与执行。  
2.负责对合同签约对方的资质和信用情况等进行调查，并全面负责合同的起草和谈判。

# 第五条 合同评审及签订

# （一）合同审核权限

合同审核按照公司授权流程归口负责组织发布的授权管理相关文件执行，授权文件以公布于公司平台的内容为准，合同经办人及会签人员应定期查询授权内容并严格遵照执行。

# (二)合同评审要求

1.所有合同线上评审，保证流程合规、线上留痕。不允许出现先签合同后补评审的情况发生，特殊情况须按照公司授权经一号位或业务组织负责人批准。  
2.合同评审前应做好前期调研工作，对于不符合合同主体资格要求或资信状况不佳的相关公司和个人，不得与其签订合同。  
3.标的额人民币5000元以下（含5000元）且能即时结清的交易可以不签订书面合同，但需甲方提供盖章版委托单。如法律规定必须采用书面形式的，须签订书面合同。

4.提交合同评审前,应该做好充分细致的谈判、起草等基础工作，确保提交合同评审稿的质量。对于未使用公司范本或内容复杂的合同，应由发起组织先行做好线下沟通。  
5.合同经办人应当将合同所应附的合同招标(议标)中标审批表等相关资料上传或关联合同评审系统，对于续签合同及签订补充协议的，须在合同评审系统链接原合同;对于项目相关的采购-服务类合同，须在合同评审系统链接该项目的销售合同。  
6.参与合同评审的人员应当从各自的专业角度对合同的相应内容进行审核(合同评审职责要求参见《合同审批授权表》)，对合同其他部分问题应尽到提醒义务。  
7.参与合同评审的人员应签署明确、具体的意见，禁止使用“原则同意”、“基本可行”等模糊性语言。  
8.合同审批人负责全面审核合同内容和上述相关组织审核意见，做出是否同意签署和履行该合同的审批，并负全面责任。  
9.合同经办人应按照合同评审人员提出的意见或要求对合同内容进行修改，并在系统内将修改意见及改稿情况反馈给相关人员、复审。  
10.合同评审实行时限审结制,合同经办人提交合同后，参与审核及审批人员须在接到待审合同后1个自然日内完成意见签署。  
11.经办人改稿环节，经办人应在5个工作日内完成改稿，如因特殊原因不能及时处理的(超过5个工作日)应告知合同管理员并于再次提交时将原因描述清楚。  
12.经办人为合同评审第一责任人,负责跟催合同审核会签进度  
13.合同管理员应尽到督促义务,严格监控合同评审时长,定期抽查，对评审时间过长的应问清原因，及时追责，督促改进。

# (三)合同签订

1.订立合同应采用书面形式，且公司已经制定合同范本的，应当采用合同范本，禁止无正当理由修改或弃用合同范本。在特殊情况下如需采用对方合同范本，应本着维护公司利益的原则进行协商，合同经办人必须充分说明理由，

且须经过法务人员对相应条款审核后签订。合同经办人应定期查询合同范本更新情况或直接向法务人员索取；基于业务需求，需采用非合同范本的，合同经办人应根据招标及谈判结果起草合同文本，并在合同评审系统中备注。

2.合同应该及时签订，不允许签订后补合同。  
3.合同审批通过后，须及时打印、签字、盖章。  
4.合同应先签字后盖章，签订时必须填写日期，不得签订没有终止日期的合同，不得签订自动续期的服务采办类合同。  
5.合同双方所盖印鉴须为公章或合同专用章。   
6.合同管理人员发现合同内容及签字、盖章、日期等与本办法不符，应提醒经办人立即采取书面补充协议方式予以补救。  
7.合同附件作为合同附属部分，其与主合同的效力差异，主合同应予以确定，并注明附件名称、数量。附件应盖章确认效力。  
8.单份合同文本达二页以上的须加盖骑缝章。  
9.签订合同时，签约组织应根据需要确定合同签约份数。

# 第六条 合同的履行、变更、作废或解除

（一）合同签订后，合同管理员通知市场开发人组织协调合同交底工作。具体流程详见《合同交底流程》。  
（二）各组织人员必须认真履行合同义务，切实维护我方权益，并根据有关交易习惯履行通知、保密等义务，但有下列情形之一的合同，各组织不得实施履行：

1.应当订立书面合同而未订立书面合同的。  
2.合同约定的生效条件未成立的。  
3.合同约定的生效期限未届满的。  
4.对方负有先履行义务，但尚未履行或履行不符合约定的。  
5.对方当事人资金信用状况恶化或其明确表明不履行合同义务的。  
6.合同无效或被撤销的。

（三）合同对价款（或报酬）、履行地点、履行费用承担等内容没有约定或约定不明确的，须及时与对方以书面形式协商确定。

（四）合同履行中，对方交付的标的物或提供的服务不符合合同约定、逾期付款、逾期供货等情形的，合同履行组织须按合同约定或在合理期限内向对方提出书面异议。  
（五）合同履行中，对方提出终止履行、合同变更、合同解除或其他履行异议的，我方须暂停合同义务的履行并按合同约定或在合理期限内做出书面答复。  
（六）合同履行中，对方出现经营状况严重恶化，资不抵债，或存在转移财产，抽逃资金，以逃避债务及有丧失或可能丧失履行债务能力的其他情形时，合同履行组织须通知财运等相关组织暂停合同履行(如款项支付等)，并向法务人员提交书面处理意见，经法务论证后，提交负责人决定采取相应措施。  
（七）合同签订时的客观情况发生重大变化，导致公司不能履行合同，或履行合同造成公司重大损失的，公司须向对方书面提出合同变更或解除请求。  
（八）合同无法履行，经办人须及时向本组织合同管理员提出变更、作废或解除申请。已作废的合同如我方已签字或已加盖合同章须将合同原件交至本组织合同管理员统一处理。  
(九)合同履行争议经各方协商一致的，须签订书面补充协议，书面补充协议视为新合同，须执行审核会签、审批、签署程序。

# 第七条 合同归档

# (一)合同编码管理

所有合同签订后合同编码由智慧建造平台统一生成。

# (二)合同文件管理

1.合同审批通过或收到对方盖章合同后应在二个工作日内进行后续盖章工作。合同盖章完毕经办人应在七个工作日内完成原件扫描（PDF格式）并上传扫描件，纸质版合同移交本组织合同管理员归档(经办人与本组织合同管理异地的时间从邮寄时间起算)。  
2.经办人为打印盖章环节的第一负责人，盖章合同应保证与评审合同完全一致，添加水印的应打印水印版合同;合同管理员负责合同文稿盖章前的核对与审定。

3.各组织应切实加强合同文件内容的保密管理，除岗位职责有相关要求外，合同文件查阅、借出管理须经综合智能能力群主管同意。  
4.各组织合同管理应建立健全合同台账，按照公司档案管理办法做好合同归档。

# 第三章 奖惩

第八条 业务负责人在签订合同前必须认真审查对方当事人的主体资格、履约能力及商业信用，否则依据公司有关制度严肃处理，相关领导承担失察责任。

第九条 严禁未按照本办法的程序办理合同评审、审批手续，严禁私自对外签订、变更或解除合同，否则无论是否造成严重后果，对私自签约人、合同章或公章管理人均予以严肃处理，分管领导亦应承担失察责任。造成严重后果，构成犯罪的，移交司法机关追究其刑事责任。

第十条 严禁将合同标的不正当拆分，以规避本办法的审批程序一经发现，对责任人予以严肃处理，分管领导亦应承担失察责任。

第十一条 严禁通过订立、变更或解除合同牟取私利，损害公司利益。对损公肥私、假公济私、以权谋私的行为，一经发现，依据公司有关制度从严处理。情节严重，构成犯罪的，移交司法机关迫究其刑事责任。

第十二条 合同管理员、审核人、签约人、公章管理人因玩忽职守对公司造成重大损失的，依据公司有关制度从严处理。情节严重，构成犯罪的，移交司法机关迪究其刑事责任。合同审核人员未出具明确意见对公司造成损失的，公司有权迪究该等审核审批人的责任。

第十三条 合同审批通过后，经办人未在规范时间内提交合同进行后续盖章工作的，一经查出，对经办人处以200元/份罚款，由此产生的法律风险及责任纠纷由经办人承担。

第十四条 合同双方签字盖章完毕，经办人应在五个工作日内将原件移交至本组织合同管理员并上传扫描件，对长时间未归档合同或未上传扫描件的，将对相关责任人进行处罚。

第十五条 合同经办人无正当理由延误合同归档及上传扫描件，对合同经办人处以50元罚款。如遇不可抗拒的自然条件或特殊时期，未能归档的须向合同管理人员说明原因。

第十六条 合同的工作流审批，应严格按照公司相关规定要求的期限内完成，否则将按公司相关规定对延期责任人进行通报、罚款等处罚。

第十七条 处罚落实

对所属员工的处罚措施由合同管理部门提出，组织一号位批准后，由人发部门在当月薪酬发放时落实。

# 第四章 附则

第十八条 本办法由合同管理部门负责解释、修订。

第十九条 本办法自颁发之日起正式实施。
