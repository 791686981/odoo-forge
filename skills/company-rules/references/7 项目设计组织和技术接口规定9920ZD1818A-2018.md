# 能源工程技术有限公司项目设计组织和技术接口规定

# 1 总则

# 1.1 目的

明确工程设计中设计项目组与项目部、各分公司、各自驱组织及赋能平台、各专业之间的接口关系，确保设计进度和质量。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 设计经理负责组织实施。  
2.2 与设计项目组有关的各自驱组织及赋能平台协同实施。

# 3 组织和技术接口分为外部接口和内部接口。

3.1内部接口是指分公司内部参与设计过程的各自驱组织之间的组织接口和技术接口，包括各专业设计分工范围及职责、各专业协作要求、各专业互提资料配合进度及会签/评审要求。  
3.2 外部接口是指分公司与参与设计过程的外部组织之间的组织接口和技术接口，包括协作条件、设计分工范围及责任、相互传递的形成文件的资料内容及提交进度以及相互间的联络安排及必要的文件。

# 4 设计项目组内各专业间的接口

# 4.1 组织接口

设计项目组各专业之间的组织接口体现在《设计开工报告》中。

# 4.2 技术接口

# 4.2.1 设计资料收集

4.2.1.1 综合项目基础资料由技术赋能群、市场开发群收集后交给设计经理；专业基础资料一般由设计人员提出清单，设计经理组织收集。

4.2.1.2 各种基础资料收集后，设计人员应对资料进行评审，确定是否满足要求。  
4.2.1.3技术调研的组织工作，由设计经理提出报告，说明调研的内容、目的，计划安排、人员组成等，由设计经理组织拟定调研方案，经综合项目群各专业负责人/常规设计群各区域召集人、综合项目群召集人/常规设计群召集人批准后方可执行,涉及到公司新技术/新业务的大型项目须总工程师批准。  
4.2.2勘察与科研试验委托   
4.2.2.1 设计经理在进行勘察委托、现场勘查之前，应召集各专业负责人或设计人员共同研究并协调本项目的勘察内容与要求，组织统一勘察。主体专业负责人提出统一勘察委托，相关专业会签。  
4.2.2.2需要配合设计进行科研试验项目的，由各专业负责人分别提出报告,经设计经理综合协调，确定委托原则，核实费用，经总经理审批后实施。  
4.2.3互提设计条件及资料  
4.2.3.1多专业的工程设计，根据《设计开工报告》确定的原则，由设计经理组织各专业间互提设计资料，各专业设计人按时提供相关委托资料，不能及时提供者，应说明原因，确定提供日期，否则应承担有关责任。  
4.2.3.2填写《专业设计任务通知书》并互提设计条件；其内容和深度应能满足接受专业正常开展设计工作的要求，提供人与接受人双方签字，设计经理应对各专业互提的设计条件进行综合审查。设计条件的编制、签署、发送、签收、更改、保管执行《设计条件管理规定》。  
4.2.4 图纸会签  
4.2.4.1 凡含两个或两个以上专业的设计项目，有配合关系的图纸，应进行审查，确认无误后进行图纸会签，对专业间的配合关系进行最终审查，会审由设计经理组织。  
4.2.4.2相关专业设计人员在确认设计图纸中没有与本专业发生矛盾的问题后，在图纸会签栏内签字，必要时增加设计经理总签。

# 4.2.5 设计变更

设计文件交付后，在施工之前或施工当中，任何专业发现原设计有明显问题，或因施工条件所限而需变更设计并影响相关专业时，必须报设计经理，由设计经理按《能源工程技术有限公司设计更改管理规定》组织相关专业变更。

# 5 设计项目组与各自驱组织之间的接口

5.1 根据项目的进度要求和设计项目组对专业人员的素质要求，综合项目群、常规设计群、市场开发群按公司发布的各级设计人员任职资格聘任情况，及时选配合格的人员参加各设计项目组。  
5.2 在各项目进行过程中，根据计划进度的要求，各自驱组织随时按设计工作量大小调配安排人员。  
5.3 综合项目群、常规设计群应做好业务建设工作，向各设计项目组提供适用有效的技术规定、标准、规范和标准图等，并根据技术发展、工作经验的积累不断完善和补充上述文件。  
5.4 在项目执行过程中，综合项目群、常规设计群应组织技术骨干和专家团队、常规技术指导团队成员参加技术方案的讨论和评审。  
5.5 设计文件按《设计文件签署规定》的要求签署。

# 6 设计项目组与各赋能平台之间的接口

6.1 技术赋能群  
6.1.1技术赋能群负责综合项目设计任务的下达。  
6.1.2 负责向设计项目组提供有效的技术管理规定。  
6.1.3 负责向设计项目组提供所需的各类标准、规范的有效版本，并保存常用的国内外标准和资料。  
6.1.4负责抽查工程设计项目质量管理体系文件规定的执行情况。  
6.1.5 负责组织设计图纸的质量评定。  
6.1.6负责设计文件的归档管理。  
6.2 综合赋能服务群

6.2.1 负责设计备案管理。  
6.2.2负责综合项目设计成品文件的提取、出版与寄送相关工作。  
6.3 市场开发群  
6.3.1 市场开发群配合设计项目组各项工作的开展。  
6.3.2市场开发群负责处理合同执行过程中顾客或公司对合同提出的修改补充和变更等问题。  
6.3.3设计经理在合同执行过程中对外发生合同纠纷时，或者当顾客或第三方未按合同规定提交有关资料并经设计经理协调无结果时，应将有关事项提交市场开发群。

# 7 设计项目组与顾客之间的接口

7.1 设计经理负责与顾客进行设计联络。  
7.2 合同签订前后，顾客提出超出委托内容、合同范围的要求，或执行中发现委托内容、合同内容存在问题，由设计经理会同市场开发群、技术赋能群与顾客协商，并对委托或合同进行修订。

# 8 设计项目组与供方的技术接口

设计经理代表设计项目组与供方进行项目设计联络，对设计过程中的技术问题进行协调解决。

8.1 设计项目组与综合赋能服务群应向供方提供设计基础资料，应遵守的规范和统一技术规定。  
8.2 设计项目组对供方的承包范围、技术、进度等提出明确要求。  
8.3 供方设计与分公司的总体设计出现矛盾时设计经理应进行协调。  
8.4 设计经理应及时组织对供方的设计进行验证。

# 9 设计项目组与供货单位的技术接口

9.1 设计过程中，需要供货单位提供报价资料和技术资料的,由设计经理组织或由有关专业负责人进行联系和催促,必要时应组织专业设

计人员参加供货单位协调会、技术谈判等。

9.2 对供货单位提供的技术资料(包括技术参数和图纸资料等),有关专业设计人员应进行审查和确认,如果有问题应及时与供货单位协商解决。对于涉及多个专业的重大设备和成套设备, 设计经理应组织相关专业进行审查和确认。

# 10 设计项目组与施工单位的技术接口

施工的设计交底、材料代用、设计修改等技术接口由设计经理、设计人员或现场设计代表负责实施。

# 11 工程总承包技术接口

11.1 综合工程  
11.1.1 在总承包项目中，设计项目组服从分公司及项目部双重管理。  
11.1.2 设计经理负责与项目部的沟通。  
11.1.3 初步设计完成后或设计方案确定后，设计经理与采购经理沟通，为其提供主要设备清单及相关信息。  
11.1.4 设计方案确定前，由设计经理组织进行评审，项目经理、采购经理、施工经理参加评审会，设计方案需经项目经理审批。  
11.1.5 施工图设计产品交付后，根据项目经理的安排，参加施工图的设计交底活动，说明设计意图，解释设计文件，明确设计要求。  
11.1.6 在施工过程中，设计项目组应在合同范围内及时提供必要的技术支持，处理施工中碰到的相关问题。  
11.1.7 设计文件出版部门将设计产品按相关要求发送至项目部。  
11.2 单项工程  
11.2.1 常规设计群各区域组织从任务委托单位接收设计任务委托书，根据设计任务委托书（合同）组建设计组，进行施工图设计。  
11.2.2 设计经理负责与顾客进行设计联络。  
11.2.3 常规设计群各区域组织将设计产品发给顾客。  
11.2.4 设计项目组负责工程的设计交底。

# 12 附则

12.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
12.2 本制度自颁发之日起实施。
