# 1.2 建设规模和设计范围

# （1）设计规模及项目组成

本工程为东光县隆缘 LNG加气站工程。本工程位于沧州市东光县

仓上村附近，站址东侧临104国道，北侧为民房，南侧为乡村道路，西侧围墙外55米处为铁路(津浦线)。本站为新建三级LNG加气站，站内分为3个功能区：对外加气区、工艺装置区、生产辅助区。工艺装置区主要工艺设施有：撬装式LNG双泵撬体1套，1座60m3的LNG卧式储罐；生产辅助区包括加气站房(控制室、空压机间、值班室、财务室、营业室、卫生间等)；对外加气区包括加气区罩棚及2台LNG加气机。

# （2）设计范围与分工

本工程设计内容由LNG卸车、储存、调饱和和加气过程的工艺设计、总图运输设计以及与之配套的建筑、结构、电气、电讯、自控、给排水及消防等设计。

本项目的LNG储罐和LNG泵撬设备由设备集成商成撬提供，设备及在工程建设和生产过程中应定期对消防设施进行检查，积极贯彻“以

防为主，防消结合”的方针，长期对职工进行安全和消防教育，提高职工的火灾防范意识，加强生产安全管理，实现安全生产。

正文文字格式严格按消防设计文件编制要求进行编制。
