# 能源工程技术有限公司工程设计资格管理办法

# 1 总则

# 1.1 目的

为了明确工程设计人员设计资格基本要求，对设计人员各级资格的申报、考核及确认做出规定，保证各级设计人员资格的有效管理，特制定本办法。

# 1.2适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1分公司各设计部门负责本部门设计人员资格的申报、应用。  
2.2分公司技术质量管理部门负责组织设计人员资格的考核及确认工作。  
2.3分公司总经理负责设计资格的审批。  
2.4分公司综合办负责区域设计部参加考核人员的接待工作。

# 3 各级设计人员的基本资格要求

# 3.1设计经理

# 3.1.1 综合工程

（1）大专以上或相当学历，具有五年以上设计项目设计工作经历；  
（2）掌握设计项目主专业及相关专业设计、技术标准，能结合工作正确使用；掌握、了解本专业及相关专业的新技术、新工艺和新设备的发展动向；  
（3）有组织多专业工程设计的经验和较强的组织协调及公关能力；  
（4）责任心强、有敬业精神，严格履行设计经理的职责；  
（5）具有中级以上专业技术职称；  
（6）有同类综合工程设计或校核经历。

# 3.1.2 单项工程

设计经理应具备单项工程设计资格。

# 3.2 专业负责人

3.2.1综合工程大专以上学历，五年以上本专业或相关专业设计工作经历；

（1）熟练掌握本专业及相关专业设计、技术标准，了解本专业和相关专业技术、工艺设备发展方向；  
（2）有一定的组织协调能力；  
（3）具有确定本专业各阶段设计方案和解决较复杂专业技术问题的能力；  
（4）具有消化本专业新技术、新工艺、新设备并将其应用到工程实践的能力；  
（5）具有本专业或相关专业中级以上专业技术职称。

3.2.2 单项工程的设计经理即为专业负责人。

3.3各级设计资格  
3.3.1 制图人

由实习生及其以上资格的人员担任。制图人必须具备本专业的一般基础知识，熟悉和掌握本专业的制图规定，并能运用 CAD进行设计。

3.3.2设计人（或编制人）

（1）有一定的专业设计知识；  
（2）能贯彻执行有关规程、标准等技术规范并能正确运用；  
（3）能在校核、审核人员指导下，独立完成设计工作；  
（4）设计人应具有大专以上学历，一年以上本专业或相关专业设计经历并经分公司考核评定符合要求的人员担任；  
（5）具有一定的外语水平，能使用办公软件和专业设计软件。

3.3.3 校核人

（1）具有助理工程师以上专业技术职称；  
（2）具有较全面的专业知识，并了解相应的工艺及相关专业知识；

（3）能贯彻执行有关专业技术规程、标准和法规；  
（4）校核人需具有三年以上本专业或相近专业设计经历，能正确运用专业技术标准、规范。

# 3.3.4 审核人

（1）熟悉并能正确运用本专业的有关规范、标准，并能正确指导设计、校核人员开展设计工作；  
（2）具有较全面的专业知识，并了解相应的工艺和相关专业知识；能够独立解决本专业的技术问题。  
（3）审核人需具有工程师以上技术职称、三年以上本专业或相近专业校核经历。

# 3.3.5 审定人

（1）具有高级工程师及以上技术职称；  
（2）具有丰富的工程设计、设备制造、工程施工、开工试运等经验；  
（3）全面掌握有关的标准、规范、熟悉国内外技术动向，能指导本专业技术人员解决复杂技术问题；  
（4）单项工程审定人，需有三年以上审核经历；  
（5）综合工程审定人，需具有七年以上设计经历，其中三年以上审核经历。

# 4 各级设计人员资格的管理

4.1 分公司技术质量管理部门，负责统一对分公司各级设计人员的资格进行集中管理；  
4.2 分公司技术质量管理部门负责及时更新《各级设计人员资格名单》，保证其有效性并由综合办公布。  
4.3《各级设计人员资格名单》作为工程项目的人员安排依据。  
4.4 分公司技术质量管理部门每年定期统一组织设、校、审人员的资格考核及确认工作。

# 5 各级设计人员资格的申报条件

各级设计人员资格的申报需满足本规定对设计人员的基本要求及以下要求。

5.1设计资格

5.1.1新毕业大学生及没有相关专业设计经历（设计经历在一年内）的人员，设计部门负责人根据设计人员的基本资格要求中的内容和设计能力考查合格后，向分公司技术质量管理部门提出资格考核申请。  
5.1.2相关专业设计经历在三年内并取得初级职称的新进设计人员，在试用期满后，由部门负责人对其工作能力和技术水平进行考核，提出资格建议，向分公司技术质量管理部门提出申请。

5.2校核、审核资格

5.2.1 设计经历在三年以上的新进设计人员在试用期满后，由部门负责人对其工作能力和技术水平进行考核，提出资格建议，参加分公司技术质量部组织的校审资格考核答辩。  
5.2.2对公司内部的资格聘任，部门负责人预先指定专人对申请人的设计图纸进行为期二个月的相应设计资格质量考查，确认其能胜任此项工作时，向分公司技术质量管理部门提出资格申请，参加分公司技术质量管理部门定期组织的校审资格考核答辩。  
5.3特殊申请资格人员，由部门负责人将预报聘任人员的业绩能力和特批理由，提交分公司技术质量管理部门，参加由分公司组织的相应的资格评审。

5.4 从事工程概算的审核人员，应取得国家颁发的相应注册资质的资格证书，并在综合办备案。

5.5注册工程师的资格由国家和各主管部门组织考试合格，并在相应主管部门注册后获得。  
5.6 特种设备设计资格：压力容器的设计、校核、审核资格，压力管道的审核、审定资格，首先应经过培训考核，取得国家质检总局或省

质量技术监督部门负责批准、颁发的资格证书，才符合压力容器、压力管道工程设计人员设计资格基本要求。

# 6 各级设计人员资格考核确认

# 6.1 资格申报

6.1.1分公司技术质量管理部负责定期组织资格申报工作。

6.1.2 设计部门负责人依据各级设计人员的基本资格要求，推荐部门具备相应能力的人员申请参加资格考核，将参加考核人员的基本情况，填写《设计资格申请表》报分公司技术质量管理部。

6.1.3分公司技术质量管理部根据各设计部门的申报材料，进行复审后，发布《参加各级设计人员资格考核人员名单》。

# 6.2资格考核

# 6.2.1 设计资格

（1）设计资格的考核由各设计部负责人负责组织完成，考核分为专业基础知识笔试和设计图纸答辩两项；  
（2）专业基础知识笔试的试卷由分公司技术质量管理部从公司题库抽调，由设计部门负责人组织完成考试；  
（3）设计图纸答辩由部门负责人和部门内的校核、审核三人以上组成答辩考核小组，由部门负责人兼答辩组长，负责完成本部门的答辩考评工作；  
（4）考试结束后，设计部门负责将试卷、《答辩考核表》、《设计人员资格聘任表》提交分公司技术质量管理部。  
（5）专业基础知识笔试试卷由技术管理人员根据题库的标准答案评分。  
（6）分公司技术质量管理部依据考核成绩，在《设计人员资格聘任表》中签署聘任建议后，报分公司技术分管领导审核，总经理审批。  
（1）6.2.2 校核、审核人员资格的考核由分公司技术质量管理部门

负责组织实施，明确考核答辩地点、时间及人员；

（2）考核答辩项目由分公司技术质量管理负责抽取；  
（3）考评组成员，由分公司技术质量管理部门负责协调组建，由设计经验丰富且具有审核资格的人员组成考评组，负责完成所有参加答辩考核的评分工作。  
（4）分公司技术质量管理部门负责将所有的考核评分进行汇总，填写完成《设计人员资格聘任表》并签署考核结果及聘任建议。

# 6.2.3对特殊申请校、审核人员资格的认定

分公司技术质量管理部门根据设计部门负责人提交的特批资格申请，组织特批资格评审会议；分公司有关领导及部门负责人参加评审并作出评审决定，由分公司技术质量管理部门统一发布。

# 6.3资格审批确认

6.3.1相关设计部门负责人和分公司技术质量管理人员共同填写完成《设计人员资格聘任表》，报分公司技术质量部门负责人进行审核。  
6.3.2经分公司技术质量管理部门负责人审核后的《设计人员资格聘任表》，报分公司技术分管领导审核，总经理审批。  
6.3.3 分公司技术质量管理部门根据审批后的《设计人员资格聘任表》，统一发布《关于设计人员资格聘任通知》。

# 7 罚则

对于在工作中不负责任，设计中出差错造成较大经济损失，或重复出现设计错误无明显改进，以及业绩考核不能胜任现岗位的或严重影响我公司声誉的设计人员，应降低或取消其资格。

# 8 附则

8.1本制度由公司技术质量部归口管理，石家庄能源化工技术分公司负责解释、修订。  
8.2本制度自颁发之日起实施。
