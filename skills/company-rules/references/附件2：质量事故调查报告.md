# 一、事故基本信息

1.事故名称：  
2.事故单位：  
3.事故地点：  
4.事故时间：  
5.事故等级。

# 二、事项项目基本信息

1.事故项目情况：  
2.事故项目各参建单位概况：

# 三、事故详细经过

1.事故发生的详细经过 （包括事故发生前、 事故发生过程中、 事故发生之后的所有情况和条件，附事故现场示意图、工艺流程图、设备图等）；  
2.事故发生后采取的应急措施。

# 四、事故损失统计

1.直接经济损失；  
2.间接经济损失。

# 五、事故原因及性质

1.直接原因

# 2.间接原因

（在分析事故时，应从直接原因入手，逐步深入到间接原因，从而掌握事故的全部原因。再分清主次，进行责任分析。）  
3.事故性质:认定为责任事故或非责任事故。

# 六、事故责任分析、认定及处理建议

认定事故性质为责任事故的，事故调查组应当依照下列规定确认事故发生单位相关人员的责任：

1.从事生产、作业的人员违反相关法律法规标准和公司相关质量规定，导致事故发生的，是直接责任者；  
2.对生产、作业负有组织、指挥或者管理职责的负责人、管理人员违反质量相关法规标准、规章制度，导致事故发生的，是领导责任者；  
3.在直接责任者和领导责任者中，对于事故的发生起决定性、关键性作用的，应当承担主要责任，其他人员应当承担次要责任。

# 七、事故整改防范措施

（从技术、管理等方面入手，针对每个原因至少制定一个整改行动措施，遵循‘SMART"原则，清楚传达整改方案或措施。）

# 1.技术措施

（如结构加固补强、缺陷修复、材料或设备替换、返修或返工等。）

# 2.管理措施

（如强化技术规范与设计复核；加强材料进场检验和施工过程管控；

引入智能化检测设备；完善质量管理或作业流程；开展全员质量培训与考核等。）

3.明确整改时限要求。

# 八、附件

1.调查组成员名单、单位及签名；  
2.事故造成的直接经济损失情况（附计算依据）；  
3.现场勘验、技术鉴定及物证、人证材料，主要包括：现场调查记录；现场照片、示意图；与事故有关的物证、人证资料及事故责任人的自诉材料等。  
4.其他需要载明的事项及与事故有关的材料。
