# 化工设计分院投标管理细则

# 第一章 总则

# 第一条 目的

为了完善建造智能业务群化工设计分院投标活动，规范投标流程的过程管理，促进投标科学决策，防范投标风险，保证投标质量，特制定本管理细则。

# 第二章 分则

# 第二条 组件

（一）由市场开发群负责，根据投标项目的类型和规模需要，向公司申请组建设计投标小组。  
（二）投标小组的技术负责人、设计经理以及相关专业的技术人员由主管生产的部门进行选定或推荐。  
（三）投标小组的主要成员有：小组组长、技术负责人、设计经理、商务经理、相关专业技术人员。如果有特殊需求，根据实际情况进行增设小组成员。

# 第三条 任职要求及职责

# （一）、投标小组组长的任职要求和职责

1．投标小组组长由公司决策层推荐确定  
2．投标小组组长为公司管理团队成员，有丰富的相关项目执行和投标组织经验。  
3．负责项目整体投标文件编制的进度、质量。参加投标小组组织的重要工作节点会议，并主持投标文件的最终审定会议。  
4．对项目最终投标文件的完整性、准确性负责。有权对投标小组的工作进行监督、指导，并且提出相关改进建议。  
5．负责将商务报价提交公司决策层最终审定确认。

# （二）投标技术负责人的任职要求及职责

1．投标小组技术负责人由公司总工程师推荐确定。  
2．投标小组技术负责人必须为公司技术专家团队成员，有丰富的相关项目执行和投标组织经验。  
3．负责投标技术文件的最终审定工作，对项目投标技术文件的完整性、准确性负责。  
4．负责主持由投标小组组织的重要工作节点的技术交流会议。  
5．有权对投标小组的工作进行监督、指导，并且提出相关改进建议。

# （三）投标小组设计经理主要职责

1．投标小组的设计经理由公司主管生产部门推荐选定。  
2．设计经理应具备相关专业高级工程师职称，取得了公司对专业岗位的资格认证，主持或参与完成过同类项目或类似项目的设计生产工作。  
3．参与或主持过同类项目或类似项目的投标工作。

4．负责组织投标小组对招标文件的解读。负责汇总各专业对招标文件需招标单位解答的问题清单。如果有项目现场踏勘需求，提出调查了解的项目清单和现场踏勘人员名单。  
5．对投标小组工作的进度和质量负责。对投标小组成员提出明确的投标分工需求、配合需求、进度需求和质量需求，对不能胜任工作的小组成员有权向相关部门提出更换。  
6．组织投标文件编制过程中关键环节的会议，并形成会议纪要。  
7．组织汇总各专业投标文件，最终形成的投标文件技术标书部分。  
8．负责对投标小组成员在项目投标过程中配合、进度控制、质量控制进行考核，负责统计设计团队成员工作量并将结果对主管生产部门汇报，计入月度和年度考核项。

# （四）商务经理主要职责

1．投标小组商务经理为项目的主要开发人员或由市场开发群推荐的市场开发人员。  
2．商务经理要具备良好的商业素养，跟招标单位保持顺畅的沟通。负责收集整理信息，向公司提交组建投标小组的申请。  
3.负责协调市场开发群进行投标报名、购买标书、下载招标文件，及时将招标文件和招标单位对项目的需求传达给投标小组设计经理。  
4.负责投标小组与招标单位的沟通联络工作。负责协助设计经理制定项目投标的分工、进度要求和质量要求。  
5.负责组织投标商务文件的编制工作，对项目商务投标文件的完整性、准确性负责。  
6.负责根据市场竞争情况和项目特点提交对外报价建议。  
7.负责联合设计经理组织对投标活动进行整体复盘，并编写复盘报告向公司决策层汇报。

# （五）各专业相关成员任职要求及职责

1．各专业相关成员具有五年以上（含五年）专业设计经验，具有同类项目或类似项目的设计经验，取得了公司对专业岗位的资格认证。  
2．按照投标小组的要求进度完成分派的工作任务。  
3．按照投标小组的要求参加投标关键环节的技术沟通会议。  
4．按照小组要求的质量标准交付技术文件。组织本专业部门的专家人员对交付的技术文件进行专业校审，对交付的技术文件质量负责。  
5．负责完成投标小组分配的其他专业工作。

# 第四条 投标小组工作全流程的过程控制

（一）投标小组组建 投标小组的组建由公司市场开发群发起，向生产主管部门下发投标小组组建联络单。由生产主管部门明确投

标小组成员。任务单格式 见附件。

同时需要提供的资料主要有：招标文件（包含技术附件及相关图纸等）。

投标小组组建完成后的运行和管理工作由投保小组组长负责。

# （二）工期控制

1．投标小组组长接到招标文件后 2 日内，组织召开投标启动会。会议根据招标文件内容要求和项目特点，由设计经理对投标小组设计参与成员进行任务分派，确定技术标书的工期要求和重要工作节点，明确投标技术文件审核提交的时间节点表。由商务经理提交商务标书的工期要求和重要工作节点，明确投标商务文件审核提交的时间节点表。  
2.设计经理是投标文件编制技术部分的负责人，对技术文件按时间要求提交承担主要责任。各专业相关成员是各个专业分项工作的负责人，严格按照投标小组会议确定时间节点要求完成相应投标文件编制。  
3.商务经理是投标文件编制商务部分的负责人，对商务文件按时间要求提交承担主要责任。同时协助设计经理检查投标文件编制的时间节点的完成情况。商务经理有义务提醒执行中发生的偏差，将相关情况直接反馈到投标小组设计经理。

# （三）质量控制

1．设计经理对投标文件技术文件部分质量负责；商务经理对投标文件商务部分质量负责，商务经理将两部分文件合并。投标小组组长对投标文件的完整性负责。  
2．投标小组对编制的投标文件实行三级质量管控。

商务文件部分由商务经理组织市场开发群商务文件编制人员严格按照招标文件商务标书的要求，进行商务标的编制。初稿进行自审后由市场开发部门组织相关人员进行集中审查，具体负责人为市场开发群负责人，并签字确认。

技术文件部分由各专业成员对严格按照招标文件技术标书的要求和投标小组确定的质量要求，进行本专业技术标的编制。初稿完成自审后由所在专业科室组织相关人员进行集中审查，具体负责人为各专业科室负责人，并签字确认。

投标文件的终审由设计经理联合商务经理组织公司的相关专家团队和所有投标小组成员进行集中，具体负责人为投标小组的投标小组组长。全体参会人员在投标文件终审会表鉴字确认。

# （四）投标小组文件传送管控

1.由项目商务经理建立投标项目微信群，成员仅限于投标小组成员。  
2.项目投标微信群里发送的文件均为一般性文件。投标文件中涉及技术专利、报价等绝密文件严禁在微信群发送。  
3. 投标小组相关专业提供资资料需求在微信群里传送（需要注明反馈时间），需要进行反馈的相关人员按照要求及时反馈。  
4.商务经理按照招标文件中投标文件格式要求，梳理编制投标文件框架，并负责发送给设计经理。由设计经理负责发送各专业投标小组成员。

# （五）项目投标报价

1.设计经理负责核算生产成本，并依据设计生产成本提出项目设计报价建议。  
2.商务经理负责根据市场竞争情况和项目特点，提出项目设计报价建议。  
3.投标小组组长负责收集设计经理和商务经理双方报价建议，提交公司决策层，最终对外项目设计报价由公司决策层确认。

# 第五条 投标过程会议设置

（一）投标过程中每次会议均要形成会议纪要，参会人员会签表。  
（二）投标启动会 收到招标文件后，召开项目投标启动会。由

投标小组组长主持、投标小组全员参与。 会议重点工作是针对招标文件进行解读，分析项目的特点和招标单位的需求。

（三）专项会议 在招标文件编制过程中，投标小组组长或其他成员根据需要组织专项会议。会议由设计经理主持，由投标小组相关人员参与。会议重点工作是针对招标文件答疑回复解读，工作进度协调，工作重点技术问题集中讨论等。  
（四）投标文件终审会 在投标文件编制完成后，由投标小组组织按照招标文件的要求对投标文件进行最终评审。投标文件终审会由项目商务经理联合设计经理组织，投标小组组长主持，公司相关专家人员和投标小组全体成员参加。

会议重点工作是针对投标文件的完整性、符合性以及招标单位需求的满足性几个方面进行评审。涉及专业技术问题的由投标小组专业成员修改后，交由市场开发群最终定稿。

（五）会议形式：提倡线下会议，尽量避免线上会议。  
（六）会议通知：会议通知均应在召开前一天传达到所有参会人员。

# 第六条 封送投标文件

由市场开发群按照招标文件规定进行投标文件密封和标记要求进行投标文件的打印、装订、盖章、封送及递送。

如果是电子投标，由市场开发群按照招标文件规定进行电子标书的上传，摸拟解密。

# 第七条 投标文件档案管理

（一）由市场开发群负责建立投标项目管理台帐，实施台帐管理。  
（二）项目投标组长、设计经理和商务经理负责将投标活动情况组织总结复盘。  
（三）市场开发群负责投标文件的归档与保管工作。

# 第八条 投标纪律与保密

所有投标资料均为内部文件性质，必须严格保密，严防丢失，未经批准，不得向无关人员泄漏。

# 第三章 奖惩

第九条 对于投标文件编制工作积极配合，认真负责，对标书质量提高有贡献的人员视情况程度根据公司的规定进行奖励。

第十条对于投标编制工作不配合、推诿、拖延、不认真人员，视情况严重程度实施口头教育、通报批评、考核降级等处罚。由于个人失误造成严重后果的按照公司规定进行处理。

# 第四章 附则

第十一条 本管理细则由化工设计分院综合智能能力群负责解释、修订。

第十二条 本办法自发布之日起试行，试行期为 6 个月，6 个月

内若无修订，则自动转入实施。
