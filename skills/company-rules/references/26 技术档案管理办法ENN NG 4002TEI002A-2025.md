# 新地能源工程技术有限公司石家庄能源化工技术分公司技术档案管理办法

# 第一章 总则

# 第一条 目的

为了完整、科学、高效地建立、健全档案管理工作创值与赋能，并充分、有效地发挥技术档案作用，及时准确地为企业发展和科学研究工作服务，结合实际经营生产情况，特制定本管理办法。

# 第二条 分类

燃气化工设计中心档案按载体格式分为实体档案和电子档案。按内容分设计成品文件（包括项目建议书、项目申请报告、可行性研究报告、规划、初步设计、各类专篇、市场调研及分析、节能评估报告、项目建议书、社会稳定性分析、项目简介、方案设计、效果图、工艺包、竣工图设计、技术规格书、施工图设计、设计模型（如应力、水锤、结构、三维、数据库等）、仿真分析文件、项目文件（包括项目实施的基础条件、质量记录、设计补充、变更通知单、计算书、审查意见、图纸审查问题回复、外来定型设备图纸、非定型设备图纸、项目的有关批复文件、工程地质勘察报告、项目往来函件、项目会议纪要等）及设计相关的外来资料等。

# 第三条 管理原则

（1）安全保密原则  
（2）完整真实原则   
（3）归档及时原则

# 第四条 适用范围

本办法适用于燃气化工设计中心所属的设计产品、项目设计实施中的质量记录以及设计所依据的基础资料、外来资料等文件归档管理，区域分公司技术档案管理可参照执行。

# 第二章 分则

# 第五条 组织与职责

# （一）设计文件归档

1.设计人员负责将本专业电子文档（设计成品文件、项目文件）归档至指定图档管理平台中，应采用网络设计平台归档，非平台归档应经公司专业理正批准后，在网络设计平台归档环节进行说明。档案管理专员接收到线下归档文件后，经验收入库后视为“档案接收”；  
2.设计经理负责将项目级项目文件收集，并按要求归档；项目级项目文件包括但不限于设计任务委托（特殊情况必须在归档表上备注）、设计输入评审表、设计开工报告、进度计划、会审记录、设计评审记录/质量评定记录、顾客沟通记录、顾客提供（设计）资料、设计交底记录、竣工验收记录、设计回访记录等文件；  
3.晒图员负责纸质设计文件的出版及整理，并将需要归档的纸质档案移交给本组织档案管理专员。

# （二）归档文件的验证及安全

1.设计经理负责审核所属项目设计文件归档的及时性、专业完整性；  
2.专业归档人员负责本专业上传归档文件的及时性、完整性、准确性；  
3.技术质量智能能力群负责对归档至本中心的项目设计文件进行接收、检查和档案管理，设计文件经检查无误，档案入库后，视为接收，流程闭合。  
4.中心档案管理专员对图档管理平台导出的区域分公司的常规项目电子版归档文件进行备存（各区域负责本组织归档设计文件的检查和管理）;  
5.在技术档案归档、管理过程中，所有涉及人员均应严格遵守、执行公司相关保密规定；  
6.图档管理平台应采取有效技术手段和管理措施，且严格控制权

限，确保平台归档技术档案安全。

# 第六条 技术档案管理要求

# （一）归档范围

# 1.设计成品文件

（1）电子文件（包括但不限于各类文本、方案设计、效果图、工艺包、计算书、施工图文件、审图意见及回复、变更文件、竣工图等）;  
（2）实体文件（包括但不限于各类文本、方案设计、效果图、施工图等）。

# 2.项目文件

（1）电子文件（包括但不限于设计各阶段实施过程中的项目质量表单、基础资料设计输入资料等）；  
（2）实体文件（包括但不限于甲方外来设计参考资料、地勘报告、沟通记录、交底记录等）；纸质版文件要求扫描后与电子文件一并归档，有需要归档的实体技术档案应自行整理后交给本组织档案管理专员保管；

3.凡使用新地图框、设计印章或各级设计人员签署的各种类型的技术文件均须按技术档案管理要求进行归档；  
4.其它文件（包括但不限于设计过程文件）；  
5.不同阶段设计文件具体归档要求详见《设计文件（电子版）归档清单》（附件 2）。

# （二）归档时间

1.电子版设计成品文件（包括计算书）和项目文件应在设计完成后进行归档；  
2.需要归档的实体文件应由设计人员或其他相关人员移交至本组织档案管理专员归档，并办理移交手续。

# （三）归档份数

1.不具备电子化的实体设计文件须归档一份；  
2.电子档案按照本办法要求的格式各归档一份。

# （四）实体文件的归档要求

1.卷宗整齐、完整、无破损、污迹等；  
2.归档文件应做到编制、整理及时，内容完整、项目齐全，字迹工整、图样清晰，签字、盖章手续齐全，确保档案的真实性、准确性、完整性、系统性；  
3.设计成品文件应真实、有效，与相应归档电子设计成品文件除印章外其他内容必须完全一致;  
4.归档实体设计资料一般须为原件，因故无法归档原件的可归具有凭证作用的复印件并标明原件存放位置或情况说明；  
5.档案装订幅面一般采用 A4 幅面，必要时可采用 A3 幅面；  
6.设计产品及项目文件需放置在专用的档案盒内，并编制卷内目录，盒脊要注明档案类别号、档案号、卷册号、年度、密级、保管期限、档案名称等内容，便于检索，并编制检索目录。

# （五）电子文件的归档要求

1.最终版电子文件归档类型有 Office 文档、CAD、PDF、JPG 等格式文件，其中 office 文档、CAD 文件为原文件；PDF 文件为出图文件，两种格式须并存。PDF 文件归档时须分图、签名齐全并保证有效，具备电子印章条件的需加盖电子印章；  
2.如专用软件产生的电子文件须归档源文件格式，同时在可操作的情况下归档轻量化版本和可视化文件，确保其可被浏览、审查，而无须依赖原始专业软件环境查看，并在《归档文件验收表》（附件 1）中备注所使用的软件名称及版本号。  
3.电子文件归档应严格执行《设计文件归档流程》（附件 3）和《审图文件归档流程》（附件 4），创建归档文件夹参考《归档文件夹目录结构及说明》(附件 5)；  
4.在项目设计完成后须将设计成品文件、项目文件同步归档；设计文件应由档案管理专员确认归档完成后出图文件方可流转相关环节，特殊紧急项目可由设计经理与档案管理专员沟通后，先归档电子

图、再后补项目文件，后补时限为一周工作日（五天），如未按时完成归档，出现两次以后则失去享受此通道资格；

5.设计文件补充、变更后，重新出版图纸目录（每个专业的设计图必须要有图纸目录）；变更目录的备注栏必须标明有效版次和作废版次，《设计补充、变更通知单》必须写入目录中；最新版次的目录必须承接上次目录的所有内容；  
6.项目若有多次补充、变更，则不同版次的电子图都应进行归档。补充、变更图的归档形式与原版电子图一致；当因设计原因更改发生在设计成品文件已归档，但还未交付顾客或相关方（不涉及升版），设计经理组织补充/修改已归档文件时，需明晰情况说明会同设计文件一并提交归档；  
7.与该项目、该专业无关的图或文字必须删除干净。对设计过程中图纸留存的，必须进行备注说明；  
8.非最终本版设计过程文件归档时须做好相应记录说明；  
9.归档人员在进行归档过程中电子文件应查杀病毒后进行归档，应确保电子文档的安全、完整、有效性。

# 第七条 保管与利用

（一）完善技术档案保护技术，延长档案寿命，对已破损和字迹褪色的档案及时修复。

# （二）保管期限

档案保管期限划分为永久和定期，定期分为 10 年、30 年，技术档案相关文件保管期限应参照《建造智能业务群档案管理规定》中有关要求。保管期限为定期保存的档案，到保管期满后还须复查一次，如发现有继续保存价值的，仍应保存下去，有的延长保管期限，有的转为永久保存。

# （三）检索

技术档案检索工具，是揭示技术档案的内容和成份，以保证借阅技术档案工作顺利进行的一种指引性工具。在技术档案归档完成后，

设计人员可在指定项目归档台账及图档管理平台中查询项目相关信息。

# （四）借阅

1.档案管理专员须严格履行档案查（借）阅登记手续，并负责对归还的档案进行检查核实，确认完整无损后方可办理借阅注销手续；  
2.借阅期限不得超过三十个工作日，到期归还；如需再借，须办理续借手续，续借最长不得超过三十个工作日；  
3.借阅已归档技术档案时，应严格执行相关调用流程；专业/设计人员中心内部参考时，执行《技术档案内部调用流程》（附件 6）；业主方及外部需要时，执行《技术档案外发流程》（附件 7）；内外部借阅均须按要求填写《设计文件外发、借阅审批单》（附件 8）后提交相关人员审批；借阅实体技术档案，还应在《档案借阅登记表》（附件 9）登记；案卷借出时要以代卷卡补位，案卷归还后，及时归位并取出代卷卡，档案归还应执行《技术档案归还流程》（附件 10）相关要求；  
4.提图加晒：经营管理人员办理提图手续后，向档案管理专员提图；  
5.档案查（借）阅者对借阅资料的安全、完整性负责，不得标记、涂改、折损、转借。

# （五）技术档案的销毁

档案管理专员定期对档案资料进行清理鉴定，对过期、无使用价值的档案进行清理，列出清单，报中心一号位(总经理)审批后，报废的技术档案放置待报废区，保存六个月后销毁，并在相关登记台账上清除。

# （六）档案库房管理

1.档案库房实行专人管理，非管理人员未经许可不得入内；  
2.库房内的档案柜要统一编号。入库的档案分大类有系统地上架排列；做到存放有序，查找有据；

3.爱护档案，在整理、搬移、上架和提供利用过程中，轻拿轻放，保持档案整洁，发现破损时，及时修补；  
4.认真落实防火、防盗、防尘、防潮、防强光、防虫、防鼠、防有害物体等“八防”措施。库内温度应控制在 14-24℃，相对湿度应控制在 45-60％。保持库房清洁，每月全面清扫一次；  
5.库房内严禁烟火，禁止存放与档案无关的其它物品；  
6.库房内安全用电：严禁走明线，离开库房时，检查关闭电源、关锁门窗；  
7.定期对库藏档案进行一次全面检查，掌握具体保管情况，为改善管理积累资料提供最新数据报表。

# 第三章 罚则

第八条 有下列行为之一的，将根据实际情况给予责任人罚款、赔偿损失等相应的处罚，构成犯罪的，依法追究其法律责任。

（一） 不按规定归档或不按期移交档案，影响工作或造成档案流失的;  
（二） 损毁、丢失档案的;  
（三） 涂改、伪造档案的;  
（四） 明知所保存的档案面临危险而不采取措施，造成档案损失的;  
（五） 擅自提供、抄录、公布、销毁档案的;  
（六） 擅自出卖或转让档案的;  
（七） 倒卖档案牟利或将档案卖给、赠送别人的;  
（八） 其他不执行本制度相关规定的。

# 第四章 附则

第九条 本办法由技术质量智能能力群负责解释、修订，如有未尽事宜按公司相关制度执行;

第十条 本制度发布之日起实施。

# 附件 2
