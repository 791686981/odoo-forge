# 能源工程技术有限公司设计服务管理规定

# 1 总则

# 1.1 目的

规范分公司的设计服务过程，设计产品交付后对顾客的服务、质量信息收集等活动做出相应规定。取得建设方、施工方和使用中的设计质量信息、不断提高设计质量和服务水平，最大限度使顾客满意。

# 1.2适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1各设计部门负责人和设计经理负责组织有关人员实施设计服务的具体工作，做好服务记录并负责归档。  
2.2 分公司技术质量管理部门从市场部接收顾客对质量信息的反馈,负责组织质量信息及技术服务信息的收集及分析。  
2.3经营管理部负责收集各燃气公司设计协同考核表，负责组织质量信息及技术服务信息的收集及分析。  
2.4分公司经营管理部门负责工程进度、沟通协作等信息的收集及分析。

# 3 配合施工

配合施工内容一般包括：设计交底、解决施工中的有关设计问题、现场技术服务、工程竣工验收、质量信息的收集和反馈。

# 3.1 设计交底

3.1.1工程项目在施工前根据顾客工程进度情况，及时参加设计交底。设计经理组织设计人员拟定设计交底提纲，向施工单位进行设计交底，设计交底内容包括介绍设计意图和设计内容，提出施工、安装、加工质量要求、注意事项及建议，听取施工、加工等部门对设计的意见。对施工、监理等单位提出的问题，交底人员应给予答复或变更设

计。

3.1.2区域设计部门与成员企业联合设计的一般项目的施工交底由燃气公司的设计人员完成，当施工出现设计方案变动或违规等问题，应及时与审核人员联系；工福户的施工交底（现场服务）或难度较大的工程技术服务，区域设计部门负责人安排能胜任的人员进行现场服务。  
3.1.3设计交底完成后，设计人员应收集交底记录，或者填写《设计交底记录》，交由设计交底组织单位、参加交底的施工单位、设计代表等签字确认；单项工程交底记录交部门负责人审阅后交设计经理，综合工程的设计经理负责收集整理各专业《设计交底记录》，作为项目文件存档。  
3.1.4对于顾客要求交底却无法到现场交底的单项工程，设计经理或设计人员可采用电话或其他通讯方式进行设计交底，交底内容做好记录，必要时将交底记录交甲方确认。  
3.1.5交底过程中有设计补充、变更内容的要执行《技术工程项目变更管理规定》。  
3.1.6《设计交底记录》按项目文件存档。  
3.2现场技术服务及设计质量信息收集  
3.2.1根据合同（协议）规定或顾客要求，由相关部门选派适宜的人员到现场配合施工，解决施工中的技术问题及设计变更等事宜。  
3.2.2 现场技术服务人员负责解决施工现场出现的设计技术问题,在设计单位和业主之间进行及时、准确地沟通,并通过良好的沟通、服务协助实现高质量的设计意图。现场技术服务人员必须具有良好的专业技术水平和较高的综合素质,把握质量、原则与服务之间的关系,树立进度意识、成本意识和形象意识。  
3.2.3 现场技术服务人员应填写《现场技术服务记录》，经甲方确认后按项目文件存档。  
3.2.4现场技术服务人员收集有关质量信息，总结设计中存在的问题及建议交设计经理、部门负责人和分公司技术质量管理部门各一份，

由部门负责人负责组织制定纠正和预防措施，分公司技术质量管理部门负责跟踪纠正或预防措施的执行情况。

3.2.5现场技术服务人员负责组织现场设计变更的编制并对发往现场的变更进行总体管控。  
3.2.6工程联络单的管控要求同设计变更单的管控，并作为项目文件进行存档。  
3.2.7现场技术服务人员在施工现场进一步勘验设计与实际的出入，并根据现场实际情况进行设计完善，纠正设计的失误或不合理情况。  
3.2.8质量信息反馈单及纠正或预防措施验证记录在分公司技术质量管理部门存档。

# 4 参加工程竣工验收

根据国家（地方）规定、合同规定或顾客要求，设计经理或专业部（室）负责人选派专业负责人或设计人员参加竣工验收。验收时应严格执行质量标准和设计要求，对施工中存在的问题应提出补救措施，必要时可要求返工。

# 5 设计回访

5.1综合工程或特殊工程在工程项目竣工后，适时进行设计回访。  
5.1.1每年由分公司技术分管领导根据分公司设计项目情况，确定有代表性的项目，组织进行现场设计回访。  
5.1.2主体专业部门负责人根据设计项目情况，不定期组织进行电话回访。  
5.2各设计部门每年至少组织两次现场设计回访。  
5.3确定现场回访时，由设计经理组织各专业（或相关）人员到现场进行查看、回访，听取顾客及施工单位的意见，找出设计中存在的问题及成功的经验，对现场适当拍照，回访结束后，由设计经理组织各专业（或相关人员）对回访中收集的资料进行整理、分析，编写《设计回访报告》,交分公司技术质量管理部门一份。

5.4回访可以到现场、电话联系、发放调查表等方式进行。

# 6 设计总结

# 6.1综合工程

6.1.1根据项目特点，在项目竣工后，设计经理可视整个项目在设计、施工中存在的问题进行总结，形成《设计总结报告》，经部门负责人审阅后，交分公司技术质量管理部门一份。  
6.1.2分公司技术质量管理部门根据项目总结内容，组织设计项目总结交流会，为今后的设计工作提供参考依据。  
6.1.3《设计总结报告》的内容包括设计、施工过程中的质量信息与改进措施，以及设计成功的经验和不足等。  
6.2 各设计部门负责人负责部门内部工程设计质量信息的收集与整理，在月度工作总结技术质量信息中汇报，主要内容为该月质量评定情况（包括评定比例、优良率、合格率）、出现的质量信息分析如与顾客在设计上的争议、标准规范执行情况，出现的设计质量问题及如何处理，或要求公司怎样处理等信息和成功的经验进行总结。  
6.3总结报告及时交分公司技术质量管理部门一份。

# 7 质量信息的应用

7.1分公司技术质量管理部门负责从分公司投诉管理部门处收集顾客投诉信息，并作为管理评审的输入。  
7.2分公司技术质量部负责从市场部接收顾客满意度分数统计结果。  
7.3 分公司技术质量管理部门将所有信息传达至各设计部（室），进行内部交流沟通。  
7.4通过信息分析，寻找质量管理体系业绩的改进机会。

# 8 附则

8.1本制度由公司技术质量部归口管理，燃气技术分公司负责解释、修订。

8.2 本制度自 2015 年 8 月 25 日发布实施。
