# 新地能源工程技术有限公司质量管理规定

# 第一章 总则

# 第一条 目的

为加强新地能源工程技术有限公司（以下简称“公司”）质量管理工作，建立公司质量管理组织架构，明确质量管理职责，确保工程产品与服务符合履约要求，达到高质量交付标准，依据国家相关法律法规、行业标准及质量体系要求，并结合公司实际，特制定本规定。

# 第二条 适用范围

本规定适用于公司及所属各单位的质量管理活动。

# 第三条 基本原则

# （一）依法依规，高质交付

遵守国家法律法规、工程行业相关标准规范，确保工程产品与服务交付质量合法合规要求。

# （二）客户为本，需求导向

以满足客户需求为核心，精准识别客户期望，明确质量管理措施，确保产品与服务按约交付，保证客户运营安全。

# （三）全员参与，组织协同

提高全员质量意识，落实质量规则与标准，强化组织协同，覆盖项目全生命周期质量管控。

# （四）事前预防，过程管控

重视对产品与服务过程的管控，加强对关键工序、关键质量控制点的事前、事中控制，防范质量问题或缺陷。

# （五）目标牵引，持续改进

建立质量改进的长效机制，通过质量目标分析、问题溯源、纠正措施等手段，推动质量水平螺旋式上升。

# 第四条 组织与职责

# （一）公司总经理

作为公司质量管理的第一责任人，全面负责公司的质量管理工

作，主要负责：

（1）组织制定公司质量方针、目标、策略和计划；  
（2）对质量管理工作所需资源提供支持，进行关键事项决策；  
（3）公司质量事故处理决定的审批。

# （二）公司总工程师

作为公司质量工作的核心推动者，协助公司总经理全面推动公司质量管理工作的开展，主要负责：

（1)组织制定、完善公司质量管理制度体系，建立并优化质量管理组织机构；  
（2）推动质量方针、目标和各项质量工作的有效实施，确保公司质量管理体系的有效运行和持续改进；  
（3）组织对公司各单位质量工作的监督、检查、分析与改进；  
（4）公司质量工作的协调、相关资源的调配；  
（5）质量事故处理的督导与跟踪。

# （三）技术质量智能能力群

作为公司质量管理工作的归口部门，主要负责：

（1）制定、修订和完善公司质量管理制度，并推动实施；  
（2）组织制定公司质量目标，并负责目标的收集、统计、分析与改进；  
（3）组织开展各项质量管理活动，监督、检查各业务组织质量管理工作；  
（4）质量事故的统计分析，质量事故的统筹调查处理。

# （四）专业中心、专业分公司

责推动落实本组织的质量工作，主要职责：

（1）结合专业项目需要，明确本专业的质量管理职责，配备质量管理人员；  
（2）落实公司质量管理制度，制定本专业质量管理要求并推动实施；

（3）负责本专业质量目标的设定、收集、统计、分析与改进；  
（4）负责组织本专业质量知识与技能的培训；  
（5）监督、检查本组织及区域分公司相关专业质量工作开展；  
（6）依据公司制度要求对相关质量事故/事件进行调查、处理及上报，并跟踪、监督整改措施的落实情况。

# （五）区域分公司

区域分公司负责人为本组织质量管理第一责任人，负责推动落实本组织的质量工作，主要职责：

（1）建立本组织的质量管理机构，明确质量管理职责，配备质量管理人员；  
（2）贯彻执行公司质量制度与专业质量管理要求；  
（3）负责本组织质量目标的收集、统计、分析与改进；  
（4）监督、检查本组织相关专业质量工作开展；  
（5）组织开展并参加公司及相关专业的质量培训；  
（6）按公司制度要求对质量事故进行调查、处理和上报，跟踪和监督质量事件的处理情况。

# （六）项目经理部/城市项目部/设计项目组

项目经理为本组织的质量管理直接责任人，负责本组织的质量工作，主要职责：

（1）根据工程合规需要和组织要求，设置质量管理角色，明确职责；  
（2）贯彻执行公司各级质量制度及工作要求，对项目交付全过程进行质量检查，及时发现和纠正质量问题；  
（3）对项目质量管理工作进行总结复盘，持续分析与改进；  
（4）组织参加各级质量培训；  
（5）及时调查、处理质量事件，按要求上报质量事故，并配合事故调查，执行事故处置方案。

# 第二章 分则

# 第五条 质量管理工作要求

# （一）双体系建设与运行

# 1.质量管理体系

（1）公司各类业务的质量管理过程应符合 ISO9000 质量体系要求。  
（2）由技术质量智能能力群组织建立质量手册、质量分手册，设定质量目标，任命质量管理人员，组织开展年度内审、管理评审及外部审核，保证公司质量管理体系的有效运行。  
（3）各业务质量工作按质量体系要求实施，详见公司《质量管理体系运维管理办法》ENN NG 9920/TEI002A-2025。

# 2.特种设备质量保证体系

（1）公司属于特种设备行业，相关工程项目及装备的设计、制造、安装、检验等过程应按国家《特种设备生产和充装单位许可规则》TSG07等要求合法合规开展。  
（2）由技术质量智能能力群统筹压力管道设计、压力容器设计、压力管道安装、锅炉安装、压力容器制造、压力管道元件制造等六项体系的建设，组织建立质量手册、质量分手册，设定质量目标，任命各级质量监督及控制人员，组织开展年度内审、管理评审，并接受外部审核；监督人员履职及特种设备质量安全风险“日管控、周排查、月调度”等机制落实情况。  
（3）各业务相关工作按特种设备质量保证体系要求实施，详见公 司 《 特 种 设 备 质 量 保 证 体 系 运 维 管 理 规 定 》 ENN NG9920/TEI008A-2025。

# （二）质量风险识别

各组织应全面识别、分析、评估本组织在生产和服务过程中存在的质量风险，制定应对措施以降低或消除风险，并建立风险定期回顾机制，对风险进行持续、动态的管理，确保风险管理过程的持续有效性和适用性。具体要求详见公司《质量风险管理规定》ENN NG

9920/TEI003A-2025。

# （三）质量目标的制定、收集、分析与改进

1.质量目标的制定应结合各业务实施中关键质量控制点进行设置，目标应清晰、可量化。  
2.各级组织应结合公司整体质量目标，并依据组织业务情况进行质量目标的制定和分解，落实到相应岗位和具体项目中。  
3.由技术质量智能能力群每月对各组织的质量目标达成情况进行收集、分析与评价，并向各组织反馈存在问题和改进意见，跟踪监督各组织纠偏措施落实情况。质量目标的收集、统计应从线下填报向线上系统自动提取转变。

# （四）各类业务质量工作要求

各业务组织应按下列相关业务质量要求开展工作。

1.设计业务质量工作要求

（1）对设计人员的资格申报、考评与聘任实施全流程监管；  
（2）对项目策划、设计接口、校审验证、外部审查、设计变更及设计服务等关键环节实施全过程质量控制；  
（3）定期组织设计产品的质量检查与评定，系统收集质量问题，落实闭环整改，并推动设计质量持续改进。

具体执行公司设计业务相关质量制度。

2.采购与物资管理业务质量工作要求

（1）建立采购供应商准入、评价、淘汰机制，规范采购流程；  
（2）对关键设备物资实施催交监造及检验；  
（3）制定生产类设备配置标准，定期组织盘点、抽查，做好使  
（4）依法对检验用计量器具的采购、检定、保养及报废等进行管理；  
（5）规范设备材料的进场验收、储存、保管、发放、回收、不合格品处理等管理。

具体执行公司采购与物资业务相关质量制度。

# 3.工程交付业务质量工作要求

（1）按项目施工图纸、标准规范等开展质量策划制定相关质量措施并实施，对施工全过程进行质量控制；  
（2）严格审查施工人员资质，如特殊作业人员持证上岗情况；  
（3）建立分包商准入、评价及退出机制，实行动态管理；对分包项目全过程、分包人员及现场使用的机具设备实施质量管控；  
（4）协同验收进场机具、材料及设备；  
（5）组织工程预验收，参与业主组织的竣工验收并落实整改项。具体执行公司工程交付业务相关质量制度。

# 4.穿越与检测业务质量工作要求

（1）严格特殊岗位人员培训考核及持证管理；  
（2）规范施工设备全周期管理；  
（3）实施穿越、检测工程专项质量策划，分类制定施工方案，严格按质量策划开展施工作业，对施工过程、变更进行控制。

具体执行公司穿越与检测业务相关质量制度。

# 5.装备集成业务质量工作要求

（1）做好特种设备作业人员管理；  
（2）对生产设备、检验设备或器具等进行全生命周期管理；  
（3）对原材料和设备按要求进行验收、入库、保管和发放；  
（4）产品的生产过程执行三检制、检测调试、总体检验，明确不合格品的检验、判定及处理要求，定期进行统计分析及改进。

具体执行公司装备集成业务相关质量制度。

# （五）质量管理活动

1.质量培训。由各组织制定年度培训计划或针对突出质量问题按需组织专项培训，对全体员工持续开展国家质量法律法规、行业标准、公司质量管理制度、质量标准等培训，建立培训台账及考核记录，报备技术质量智能能力群。  
2.“质量月”主题活动。按照国家相关规定开展“质量月”主题活动，由技术质量智能能力群结合业务实际制定活动方案并推动实

施，召开启动与总结会议、开展主题宣传、知识竞赛等活动，各业务组织应积极承接并按活动方案落实。

# 3.质量检查、问题处理及整改

（1）各级组织应开展定期/不定期检查、专项检查活动，制定相应检查方案，明确检查目的、检查工具、检查对象等，做好检查问题记录及跟踪整改；质量检查可与其他专业检查活动联合开展，并运用数智化工具对检查、整改情况进行上报、统计与跟进。  
（2）对业务实施、检查中发现的质量问题，由责任单位对相关问题进行整改，并分析根本原因，制定或完善相关机制与标准，杜绝类似问题再次发生；对典型质量问题定期进行复盘、分析，制定预防或解决措施，形成质量管控工具或产品。  
（3）对检查中发现的违反质量禁令的行为、严重质量问题等依据公司《工程质量禁令管理规定》ENN NG 9920/TEI012A-2025、《质量事故管理规定》ENN NG 9920/TEI014A-2025 等予以处罚。

# （六）质量事故报告及处理

发生质量事故，各级组织应按公司《质量事故管理规定》ENN NG9920/TEI014A-2025要求逐级上报，按事故分级授权组织专家组进行调查，按照“四不放过”等原则进行处理，依据事故处置方案进行落实整改。建立质量事故事件台账，做好事故反思与记录。

# 第三章 奖惩

# 第六条 奖励

（一）鼓励各业务组织积极开展市级、省级、国家级等各类优质奖项申报制度执行。

# （二）质量工作奖励

1.技术质量智能能力群定期（季度或年度）结合各业务组织质量目标达成情况、质量检查结果排名等，对表现特别突出的个人和组织申请相关荣誉与奖励。

2.对及时发现质量问题或隐患、避免重大质量事故发生的个人，由所在专业中心或分公司给予个人 500-2000 元奖励。 由所在专业中心或分公司给予个人500-2000元奖励。

# 第七条 处罚和责任追究

对未按本规定落实质量管理要求的组织，在年度、季度组织创值评估结果中进行扣分处理；对未按要求开展质量工作的个人，按照新奥天然气股份有限公司《员工违规违纪行为处罚办法》ENN NGHR002B-2024等制度进行处罚和相关责任追究。

# 第四章 附则

第八条 本规定由技术质量智能能力群负责解释、修订。

第九条 本规定自正式发布之日起，试行 6 个月。6 个月内若无修订，则自动转入实施。
