# 设计文件晒发管理办法

# 第一章 总则

# 第一条 目的

为高速有效完成设计文件晒制发放工作，规避设计文件晒发风险点，提高业主的满意度，结合公司实际，特制定本办法。

# 第二条 适用范围

本办法适用于燃气化工设计中心设计文件晒发管理的相关工作。

# 第二章 分则

# 第三条 权责划分

（一）综合智能能力群为设计文件晒发管理归口组织。  
（二）相关主管领导负责晒图申请流程审批。  
（三）设计经理或各专业设计负责人在网络设计平台发起晒图申请。

# 第三章 细则

# 第四条 晒图流程

# （一）设计文件正常晒制

1.综合项目：设计经理或各专业设计负责人在网络设计平台所属项目下发起晒图申请，选择平台归档文件、填写晒图详情和晒图信息并提交申请；综合智能能力群-经营收到晒图申请后审核晒图数量及项目经营状态；综合智能能力群-政联收到晒图申请后审核图纸类型及数量；晒图员收到晒图申请后，根据晒图申请要求下载晒图文件进行晒图并在平台维护晒图信息。  
2.燃气常规项目：设计经理或各专业设计负责人在网络设计平台所属项目下发起晒图申请，选择平台归档文件、填写晒图详情和晒图信息并提交申请；晒图员收到晒图申请后，根据晒图申请要求下载晒图文件进行晒图并在平台维护晒图信息。

# （二）加晒设计文件（平台项目）

1.综合项目：设计经理或各专业设计负责人在网络设计平台所属项目下发起晒图申请，选择平台归档文件、填写晒图详情、晒图信息、加晒原因并上传晒图依据后提交申请；综合智能能力群-经营收到晒图申请后审核晒图数量及项目经营状态；综合智能能力群-政联收到晒图申请后审核图纸类型及数量；经过业务理正审批后系统推送到晒图员；晒图员收到晒图申请后，根据晒图申请要求下载晒图文件进行晒图并在平台维护晒图信息。  
2.燃气常规项目：设计经理或各专业设计负责人在网络设计平台所属项目下发起晒图申请，选择平台归档文件、填写晒图详情、晒图信息并提交申请；经过设计负责人审批后系统推送到晒图员；晒图员收到晒图申请后，根据晒图申请要求下载晒图文件进行晒图并在平台维护晒图信息。

# （三）加晒设计文件（非平台项目）

1.综合项目：设计经理或各专业设计负责人在网络设计平台“晒图管理”界面发起晒图申请，填写晒图详情、晒图信息、加晒原因并上传晒图依据后提交申请；综合智能能力群-经营收到晒图申请后审核晒图数量及项目经营状态；综合智能能力群-政联收到晒图申请后审核图纸类型及数量；经过业务理正审批后系统推送到晒图员；晒图员收到晒图申请后，将晒图申请审批单（截图）提交至档案管理员，向档案管理员提图，根据晒图申请要求进行晒图并按照平台要求维护晒图信息。  
2.燃气常规项目：设计经理或各专业设计负责人在网络设计平台“晒图管理”界面发起晒图申请，按照平台要求填写晒图详情、晒图信息、加晒原因并上传晒图依据后提交申请；经过设计负责人审批后系统推送到晒图员，晒图员收到晒图申请后，将晒图申请审批单（截图）提交至档案管理员，向档案管理员提图，根据晒图申请要求进行晒图并按照平台要求维护晒图信息。

# （四）设计文件发放

# 1.电子版施工图发放

由业主或设计经理提交《设计文件外发、借阅审批单》（详见技术档案管理办法）写明原因，经设计经理签字及相关主管领导签批后递交至档案管理员，档案管理员将电子版设计文件直接发送至业主邮箱。  
2.纸版设计文件发放由晒图员将需要发放的设计文件进行登记，填写《燃气

化工设计中心设计文件发放登记表》电子版留存，连同设计文件一起邮寄给业主签字确认，并收集回签字版留存。

# （五）申请单的管理

1.《设计文件外发、借阅审批单》由档案室收集整理并存档。  
2.《燃气化工设计中心设计文件发放登记表》由晒图室收集整理并存档。

# 第四章 罚则

第五条 按本办法中规定的晒制流程开展设计文件出版发放工作，未按本办法执行的一经发现将严肃处理。

# 第五章 附则

第六条 本办法由综合智能能力群负责解释、修订。

第七条 本办法自颁布之日起实施。

# 设计文件晒发节点把控说明

1.每天整理晒制好的设计文件核对项目名称数量进行登记  
2.根据项目的轻重缓急及时与每个设计经理随时沟通，根据网络设计平台晒图申请审批单晒制发放设计文件。  
3.设计文件核对无误后一次邮寄给甲方（同时附《燃气化工设计中心设计文件资料发放登记表》），随时跟踪设计文件邮寄、到达的时间与甲方联系人沟通跟催设计文件发放登记表反馈。  
4.燃气项目要及时登记，发放时认真核对项目名称设计文件数量确定无误后由取图人员签字确认取走。  
5.设计变更通知单根据设计经理或经营对接人员提供的数量及时复印盖章登记，与设计经理或经营对接人员确认邮寄地址一次寄出，并跟催《燃气化工设计中心设计文件资料发放登记表》的反馈。  
6.项目登记清楚便于查找无漏或模糊登记，凡是晒图室出去的每个设计文件要有盖章，并登记将《燃气化工设计中心设计文件资料发放登记表》登记准确清楚，便于以后查找。
