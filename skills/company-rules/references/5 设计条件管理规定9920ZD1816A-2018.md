# 能源工程技术有限公司设计条件管理规定

# 1 总则

# 1.1 目的

规定设计条件编制、签署、发送、签收、更改、保管等内容。

# 1.2 适用范围

本制度适用于能源工程技术有限公司所属设计单位。

# 2 组织与职责

2.1 各设计项目组负责按规定执行。  
2.2 各设计项目专业负责人对条件进行管理,始终保证其有效性。

# 3 编制

3.1 设计条件（提出或返回）其文字、表格、图纸都应整洁、规范，图幅符合标准,且提条件专业须三级签署。  
3.2 填写《专业设计任务委托书》互提设计条件,其内容和深度应能满足接受专业正常开展设计工作的要求。设计经理应对各专业互提的设计条件进行综合审查。  
3.3 提出或返回条件的专业必须对条件的正确性和完整性负责。设计条件一经接受，即为相关专业的设计依据，不得随意更改。

# 4 签署

4.1 设计条件的签署参照《设计文件签署规定》的要求执行。  
4.2 签署中不能一人同时在设计、校核、审核的其中两栏内签字，不符合签署规定的条件，接收人有权拒收。

# 5 发送

提交条件专业将条件发送给接收条件项目专业负责人。

# 6 签收

接收条件项目专业负责人接受条件时，在《专业设计任务委托书》上签字确认。

# 7 更改

7.1 当发生以下情况时，应及时对原提供的设计条件进行变更：

（1）上级领导或建设单位对工程设计提出新的意图，原设计原则及设计条件有较大变动；  
（2）原提供的设计条件发现有明显问题或不妥之处；  
（3）对重大技术问题有新的处理方法，改进意见，能使设计更加完善合理；  
（4）在设计过程中，发现原提供设计条件，从技术上难以解决；  
（5）由于施工单位或分公司以外的专业配合使原提供的设计条件变更；  
（6）在设计过程中，发现原提供的设计条件不充分，需要补充。

7.2 当条件需要更改，而更改内容属于“涉及重大技术方案及国家政策性（安全、环保、节能等）的条件”或属于“落实方案及涉及多专业的重要条件”或在原条件上更改表达不清时，需重新填写《专业设计任务委托书》提交设计条件，并注明变更内容及理由，明确原条件作废。  
7.3 设计人协商解决，且在原条件上可更改清楚时，可以在原提条件上进行更改，但仍应在更改处签字确认。  
7.4 如果修改对设计影响范围较大,必须经过设计经理核准，决定是否需要修改和如何修改。若设计经理核准需要修改设计条件，则按  
7.2 执行，若设计经理核准不修改或暂缓修改，则以备忘录的形式备案。  
7.5 接收条件专业对接收的条件提出修改要求,且此修改必须进行时,其处理办法同上。

# 8 保管

8.1 各项目专业负责人对条件进行管理,始终保证其有效性。  
8.2 项目完成后,各设计条件作为项目文件存档。

# 9 附则

9.1 本制度由公司技术质量赋能部归口管理，设计分公司负责解释、修订。  
9.2 本制度自颁发之日起试行，试行期为 6 个月。6 个月内若无修订，则自动转入实施。
